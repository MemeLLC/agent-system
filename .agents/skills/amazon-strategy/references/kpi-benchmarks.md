## 重要指標定義表

| 指標 | 正式名称 | 計算式 | 意味 |
|------|----------|--------|------|
| **ACoS** | Advertising Cost of Sales | `広告費 ÷ 広告経由売上 x 100` | 広告売上に対する広告費の割合。低いほど効率的 |
| **TACoS** | Total Advertising Cost of Sales | `広告費 ÷ 総売上（広告+オーガニック） x 100` | 全売上に対する広告費の割合。事業全体の広告効率 |
| **ROAS** | Return on Ad Spend | `広告経由売上 ÷ 広告費` | 広告費1円あたりの売上。高いほど効率的 |
| **CPC** | Cost Per Click | `広告費 ÷ クリック数` | 1クリックあたりのコスト |
| **CTR** | Click Through Rate | `クリック数 ÷ インプレッション数 x 100` | 広告表示に対するクリックの割合。広告の訴求力を示す |
| **CVR** | Conversion Rate | `注文数 ÷ クリック数 x 100` | クリックに対する購入の割合。商品ページの説得力を示す |

### 指標間の関係

```
ACoS = CPC ÷ (商品価格 x CVR)
ROAS = 1 ÷ ACoS（小数表記の場合）
TACoS = ACoS x (広告売上比率)
```

**例:** 商品価格3,000円、CPC 50円、CVR 10%の場合
- ACoS = 50 ÷ (3,000 x 0.10) = 50 ÷ 300 = 16.7%
- ROAS = 1 ÷ 0.167 = 6.0x

| ACoS | ROAS | 評価 |
|------|------|------|
| **10%** | **10.0x** | 非常に効率的。ブランドKW等 |
| **15%** | **6.7x** | 優秀。目標として理想的 |
| **20%** | **5.0x** | 良好。多くのカテゴリで目標水準 |
| **25%** | **4.0x** | 標準的。カテゴリ平均レベル |
| **30%** | **3.3x** | やや高め。改善の余地あり |

| **40%** | **2.5x** | かなり高い。新商品ローンチ期のみ許容 |
| **50%+** | **2.0x未満** | 非常に高い。損益分岐点を確認すべき |

## 日本市場ベンチマーク

日本市場はグローバル平均と比較してCPCが低く、ACoSも異なる傾向がある。

| 指標 | 日本市場平均 | グローバル平均との比較 |
|------|------------|---------------------|
| **CPC** | **5〜10円**（SP） | グローバル平均の約1/10〜1/15 |
| **ACoS** | **15〜30%** | グローバルとほぼ同水準 |
| **TACoS** | **10〜15%** | 健全な水準 |
| **CTR** | 0.30-0.50% | グローバルとほぼ同水準 |
| **CVR** | 8-12% | グローバルとほぼ同水準 |

### 日本市場の特徴

| 特徴 | 詳細 |
|------|------|
| CPC水準 | グローバルと比較して非常に低い（5-10円が標準） |
| 競合性 | カテゴリにより大きく異なる。家電・美容は競争激化 |
| モバイル比率 | 購入の70%以上がモバイル経由 |
| レビュー重要度 | 日本の消費者はレビューを非常に重視（CVRに直結） |
| 季節性 | プライムデー、ブラックフライデーの影響が年々拡大 |

## 損益分岐ACoS計算方法

損益分岐ACoSは「広告で赤字にならない限界のACoS」を示す。

### 計算式

```
損益分岐ACoS = 利益率（%）
```

**詳細な計算:**

```
損益分岐ACoS = (商品価格 - 原価 - Amazon手数料 - FBA手数料 - その他経費) ÷ 商品価格 x 100
```

### 具体例

| 項目 | 金額 |
|------|------|
| 商品販売価格 | 3,000円 |
| 原価 | 800円 |
| Amazon販売手数料（15%） | 450円 |
| FBA配送手数料 | 400円 |
| その他経費（梱包等） | 100円 |
| **利益（広告費控除前）** | **1,250円** |
| **損益分岐ACoS** | **41.7%**（1,250 ÷ 3,000 x 100） |

### ACoS目標の設定

| 基準 | 計算 | 例（上記の場合） |
|------|------|-----------------|
| 損益分岐ACoS | 利益率 | 41.7% |
| **目標ACoS** | 損益分岐ACoS x 0.6-0.7 | **25-29%** |
| 攻めのACoS | 損益分岐ACoS x 0.8-0.9 | 33-37%（新商品ローンチ時） |
| 守りのACoS | 損益分岐ACoS x 0.4-0.5 | 17-21%（利益重視） |

> **ポイント:** 目標ACoSは損益分岐ACoSの60-70%に設定するのが一般的。これにより広告費を差し引いても十分な利益が確保できる。
