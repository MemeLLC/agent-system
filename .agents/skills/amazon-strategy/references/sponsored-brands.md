## 概要

スポンサーブランド広告はブランド認知とカテゴリシェア拡大を目的とした広告タイプ。検索結果ページの目立つ位置にブランドロゴ、カスタム見出し、複数商品を一括表示できる。

| 項目 | 内容 |
|------|------|
| 課金方式 | CPC（クリック課金） |
| 最低日予算 | 100円 |
| 最低入札額 | 2円 |
| 利用条件 | **Amazonブランド登録が必須** |
| 審査 | 通常72時間以内（クリエイティブ審査あり） |
| 表示ラベル | 「スポンサー」表示あり |

### 表示位置

| 表示位置 | 説明 |
|----------|------|
| **検索結果上部（Above the fold）** | 検索結果の最上段に横長バナー形式で表示。最も視認性が高い |
| **検索結果横** | デスクトップで検索結果の左側サイドバーに表示 |
| **検索結果下部** | 検索結果ページの下部に表示 |

## 3つのフォーマット

### 1. 商品コレクション

ブランドロゴ、カスタム見出し、最大3つの商品を表示するスタンダードフォーマット。

#### クリエイティブ仕様

| 要素 | 仕様 |
|------|------|
| ブランドロゴ | **400 x 400px以上**、1:1アスペクト比、PNG/JPG/GIF、1MB以下 |
| カスタム見出し | **全角35文字以内**（半角70文字） |
| カスタム画像 | **1,200 x 628px**推奨、PNG/JPG、5MB以下 |
| 掲載商品数 | 最低**3 ASIN**（最大100 ASIN） |
| 表示商品数 | 表示デバイスにより3〜5商品 |

#### ランディングページ選択肢

| タイプ | 説明 | 推奨場面 |
|--------|------|----------|
| **ストア** | 自社Amazonストアページ | ブランド認知・全商品訴求 |
| **カスタムランディングページ** | 選択した商品一覧ページ | 特定カテゴリ・キャンペーン訴求 |
| **商品詳細ページ** | 1つのASINの詳細ページ（商品1つの場合） | 主力商品への集中誘導 |

#### 見出しの禁止事項

- 商品やブランドと無関係な内容
- 「ベストセラー」「No.1」等の根拠なき最上級表現
- 絵文字の使用
- 全角大文字のみの英語表記
- 句読点の過剰使用（「!!!」「???」等）
- 圧力をかける表現（「今すぐ買わないと損」等）
- 顧客レビューの引用

### 2. ストアスポットライト

Amazonストアの複数サブページを直接訴求できるフォーマット。ブランドストーリーを伝えるのに最適。

#### クリエイティブ仕様

| 要素 | 仕様 |
|------|------|
| ブランドロゴ | 400 x 400px以上、1:1、PNG/JPG/GIF |
| カスタム見出し | 全角35文字以内 |
| サブページ数 | 最低**3サブページ**（最大3つ表示） |
| サブページ画像 | 各サブページのカスタム画像（400 x 400px推奨） |
| サブページ見出し | 各全角35文字以内 |

#### 要件

- Amazonストアが作成済みであること
- ストアに最低3つのサブページがあること
- 各サブページに最低1つの商品があること

#### AI見出し提案（2025年新機能）

- 商品情報とストアコンテンツを基にAIが見出しを自動提案
- 複数バリエーションを提示し、A/Bテストに活用可能
- カスタマイズ可能（AI提案をベースに手動編集）

### 3. SB動画

検索結果ページ内で自動再生される動画広告。静止画と比較して圧倒的なパフォーマンスを発揮。

#### 動画クリエイティブ仕様

| 項目 | 仕様 |
|------|------|
| 動画の長さ | 6〜45秒（**推奨: 15〜30秒**） |
| アスペクト比 | **16:9** |
| 解像度 | **1,920 x 1,080px**（最低1,280 x 720px） |
| ファイルサイズ | **500MB以下** |
| ファイル形式 | MP4、MOV |
| 映像コーデック | H.264 |
| 音声コーデック | AAC、**128kbps以上** |
| フレームレート | 24fps以上 |
| ビットレート | 1Mbps以上 |

#### 動画クリエイティブの注意点

**必須要素:**

- ミュート再生が前提（テキストオーバーレイで情報を伝える）
- 最初の2-3秒で視聴者の注意を引く
- 商品を早い段階で画面に出す
- CTAを動画内に含める（「詳しくはこちら」「今すぐチェック」等）

**禁止事項:**

- 黒帯（レターボックス/ピラーボックス）
- Amazon商標/ロゴの使用
- カスタマーレビューの表示
- 画面外にはみ出るテキスト
- ぼやけた/低解像度の映像
- 最初または最後の空白フレーム

**推奨構成（30秒動画の場合）:**

| セクション | 時間 | 内容 |
|------------|------|------|
| フック | 0-3秒 | 課題提示 or インパクトのあるビジュアル |
| 商品紹介 | 3-10秒 | 商品の主要機能・特徴 |
| ベネフィット | 10-20秒 | 使用シーン、Before/After |
| 社会的証明 | 20-25秒 | 受賞歴、認証、実績 |
| CTA | 25-30秒 | 行動喚起、ロゴ |

#### SB動画のパフォーマンス

| 指標 | SB動画 | SB静止画 | 比較 |
|------|--------|----------|------|
| CTR | **0.89%** | 0.34% | **2.6倍** |
| CVR | **11.2%** | 9.9% | **13%高い** |
| エンゲージメント | 高い | 標準 | 動画は目を引きやすい |

## ターゲティング

### キーワードターゲティング

SP広告と同様に3つのマッチタイプ（Broad/Phrase/Exact）を使用可能。

| マッチタイプ | 用途（SB特有の考慮点） |
|--------------|----------------------|
| **Broad** | ブランド認知拡大。幅広い検索クエリでブランドを露出 |
| **Phrase** | カテゴリ内での存在感確立 |
| **Exact** | 高意図KWでブランドを最上位に表示 |

### 商品ターゲティング

| タイプ | 説明 |
|--------|------|
| **カテゴリ** | 指定カテゴリの検索結果・商品ページに表示 |
| **個別商品** | 特定ASINの関連位置に表示 |

**SBターゲティングのポイント:**

- SBは認知〜検討フェーズ向け。SPより広めのKWを設定
- ブランドKWは必ず設定（自社ブランド検索時にSBで上部を確保）
- カテゴリKW（一般名詞）でカテゴリ内のブランド認知を獲得
- 競合ブランドKWで競合検索者にリーチ

## クリエイティブ設計のベストプラクティス

### 商品コレクション

1. **ロゴ:** 高解像度、背景透過PNG推奨、ブランドカラーを活用
2. **カスタム画像:** ライフスタイルイメージを使用（白背景の商品画像より効果的）
3. **商品選択:** ベストセラー商品を先頭に配置、価格帯のバランスを考慮
4. **季節性:** シーズンに合わせた画像・見出しの更新

### ストアスポットライト

1. **サブページ構成:** 「カテゴリ別」or「用途別」で分かりやすく
2. **画像の統一感:** 各サブページの画像のトーン&マナーを統一
3. **見出しの連動:** 全体テーマと各サブページ見出しの一貫性

### SB動画

1. **最初の3秒が勝負:** 製品のインパクトショットまたは問題提起
2. **テキストオーバーレイ:** 画面の下部20%には重要テキストを配置しない（UIと重なる）
3. **ブランドカラー:** 動画全体でブランドカラーを一貫して使用
4. **CTAの位置:** 動画の最後3-5秒にCTAを配置
5. **ループ:** 動画の最後が最初と自然につながる構成

## コピーライティングガイド

### 見出しの書き方

#### 効果的な見出しパターン

| パターン | 例 | 用途 |
|----------|-----|------|
| ベネフィット訴求 | 「毎朝のスキンケアを10分短縮」 | 時短・効率系商品 |
| カテゴリ訴求 | 「オーガニック認証スキンケア」 | ブランドポジション確立 |
| 社会的証明 | 「累計販売数100万個突破」 | 信頼性訴求 |
| 問題解決 | 「乾燥肌のお悩みに」 | 課題解決型商品 |
| 新商品告知 | 「新発売 春限定フレーバー」 | 新商品ローンチ |
| 特典訴求 | 「まとめ買いで15%OFF」 | セール・プロモーション |

#### 見出しの最適化ポイント

- **文字数:** 全角35文字以内。短いほど読まれやすい（20-25文字推奨）
- **KW含有:** ターゲットKWを自然に含める
- **具体性:** 数字を入れる（「100万人が選んだ」「3日で届く」）
- **ブランド名:** 見出しの先頭 or 末尾に配置
- **季節感:** シーズンイベントに合わせて更新

### A/Bテスト推奨項目

| テスト要素 | バリエーション例 |
|------------|----------------|
| 見出し | ベネフィット訴求 vs カテゴリ訴求 |
| カスタム画像 | ライフスタイル画像 vs 商品集合画像 |
| CTA（動画） | 「詳しくはこちら」 vs 「今すぐチェック」 |
| 商品順序 | ベストセラー先頭 vs 新商品先頭 |
| ランディングページ | ストア vs カスタムLP |

| 動画仕様（SB動画の場合） | 尺、構成、ナレーション有無 |
