## 1. 出品アカウントの種類と損益分岐点

### 大口出品 vs 小口出品 比較表

| 項目 | 大口出品（プロフェッショナル） | 小口出品（個人） |
|------|-------------------------------|-----------------|
| **月額登録料** | 4,900円（税別） | 無料 |
| **成約料** | 0円/個 | 100円/個 |
| **推奨条件** | 月間50個以上の販売 | 月間49個以下・テスト出品 |
| **カートボックス獲得** | 可能 | **不可** |
| **広告出稿** | 可能 | **不可** |
| **一括出品ツール** | 利用可（7商品同時登録） | **不可** |
| **ビジネスレポート** | 売上数・売上金額・アクセス数・購買率 | **不可** |
| **配送料設定** | カスタム設定可能 | Amazon指定の固定値 |
| **決済方法** | 8種類（代引き・コンビニ決済含む） | 5種類（カード系中心） |
| **新規商品ページ作成** | 可能 | **不可**（相乗りのみ） |
| **Amazon B2B** | 利用可能 | **不可** |
| **API接続** | 利用可能 | **不可** |

### 損益分岐点

**月間50個の販売が分岐点:**

- 小口: 50個 x 100円 = **5,000円**
- 大口: 月額 **4,900円**（固定）

月間50個以上の販売見込みがある場合は大口出品が経済的。ただし機能面（カートボックス獲得・広告利用・レポート）を考慮すると、**本格販売であれば月間販売数に関係なく大口出品を強く推奨**。

## 2. アカウント登録手順と必要書類

### 必要書類一覧

| 書類 | 詳細 | 注意事項 |
|------|------|---------|
| **顔写真付き身分証明書**（1つ） | パスポート / 運転免許証 / 写真付き住民基本台帳カード / 在留カード | マイナンバーカードは不可。運転免許証は両面必要 |
| **取引明細書**（180日以内） | クレジットカード明細 / ネットバンキング明細 / 預金通帳 / 残高証明書 | 発行から180日以内のもの |
| **メールアドレス** | Amazon個人アカウントとは別を推奨 | - |
| **電話番号** | SMS認証用 | - |
| **クレジットカード** | Visa / Mastercard / JCB / American Express / Diners Club | 月額登録料の支払い用 |
| **銀行口座** | 売上受取用 | 登録後に設定可能 |

### 法人の追加書類

- 登記簿謄本（履歴事項全部証明書）
- 法人の代表者または担当者の身分証明書
- 法人名義の銀行口座情報

### 登録4ステップ

**ステップ1: アカウント作成（約3営業日）**

1. Amazon出品サービス公式サイト（https://sell.amazon.co.jp）にアクセス
2. 「さっそく始める」をクリック
3. メールアドレスとパスワードを設定
4. 「大口出品」または「小口出品」を選択
5. 事業所の所在地と業種を入力
6. 身分証明書情報を入力・アップロード
7. 取引明細書をアップロード
8. SMS/電話による認証を完了

**ステップ2: 本人確認審査（約3営業日）**

- Seller Identity Verification（SIV）が実施される
- 書類の整合性チェック（氏名の一致確認など）
- 審査結果はメールで通知（追加書類を求められる場合あり）

**ステップ3: セラーセントラル設定（審査通過後）**

- 法人情報・連絡先の詳細設定
- 銀行口座の登録（売上受取用）
- 配送設定・返品先住所の設定
- 必要に応じた許認可書類の提出

**ステップ4: 商品登録と出品開始**

- 商品情報の入力
- 配送方法の選択（FBA or FBM）
- 価格設定と在庫数の入力
- 出品開始

> **所要期間の目安**: 手続き完了まで約2週間。書類不備がある場合はさらに時間がかかる。

## 3. アカウント健全性指標

セラーセントラルの「パフォーマンス」>「アカウント健全性」で確認可能。

### 主要指標と基準値

| 指標 | 基準値 | 計測期間 | 超過時のリスク |
|------|--------|----------|----------------|
| **注文不良率（ODR）** | **1%未満** | 60日間 | 出品停止・アカウント利用停止 |
| **出荷前キャンセル率** | **2.5%未満** | 7日間 | アカウント停止の可能性 |
| **出荷遅延率** | **4%未満** | 10日間/30日間 | カート獲得率低下 |
| **追跡可能率** | **95%以上** | 30日間 | カート獲得率低下 |
| **規約の順守** | 違反なし | 常時 | 即時アカウント停止の可能性 |

### 注文不良率（ODR）の構成要素

ODRは以下3つの合算で算出される:

| 構成要素 | 計算式 |
|---------|--------|
| **低評価率** | 1〜2つ星のレビュー数 / 総注文数 x 100 |
| **Amazonマーケットプレイス保証申請率** | 保証申請数 / 総注文数 x 100 |
| **クレジットカードチャージバック率** | チャージバック数 / 総注文数 x 100 |

> ODRが1%を超えると出品停止・アカウント利用停止の可能性がある。

### 健全性維持のポイント

- 注文は可能な限り迅速に処理・発送する
- 追跡番号を必ず入力する
- 顧客からの問い合わせには**24時間以内**に返信
- 商品説明と実際の商品に乖離がないようにする
- 返品・返金リクエストには誠実に対応する
- 在庫切れを防ぎ、キャンセルを最小限にする

## 4. ブランド登録（Brand Registry）

### 概要

商標権を持つブランドオーナー向けの**無料プログラム**。登録によりブランド保護ツールと販売促進ツールにアクセスできる。

### 登録要件

| 要件 | 詳細 |
|------|------|
| **商標登録** | 特許庁（JPO）で登録済み、または出願中の商標 |
| **商標の種類** | 文字商標または図形商標（標準文字・ロゴ） |
| **ブランド表示** | 商品またはパッケージにブランド名/ロゴが恒久的に表示されていること（取り外し可能なラベルは不可） |
| **出品アカウント** | 大口出品アカウントが必要 |

> 商標は出願中（出願番号が発行された段階）でも申請可能。ただし商標登録の完了には通常6〜12ヶ月、早期審査で約2ヶ月かかる。

### 登録手順

1. Amazon Brand Registry（https://brandregistry.amazon.co.jp）にアクセス
2. Amazonの出品アカウントでサインイン
3. 「新しいブランドを登録」をクリック
4. ブランド情報を入力（ブランド名、商標登録番号、商標の種類）
5. ブランドの商標が表示された商品画像をアップロード
6. 製造・販売情報を入力
7. 「出品者」にチェックを入れて「送信」

### ブランド登録の15のメリット

**ブランド保護（2項目）**

1. 知的財産侵害の報告・対処ツール（相乗り出品者への対策）
2. 商品カタログの編集優先権

**販売促進（4項目）**

3. スポンサーブランド広告・スポンサーディスプレイ広告の利用
4. A+コンテンツ（商品紹介コンテンツ）の作成
5. ストアページ（ブランドストア）の作成
6. Amazon Vine（先行レビュープログラム）の利用

**分析・運用効率化（4項目）**

7. Brand Analytics（キーワード分析・検索頻度ランキング）
8. A/Bテスト（商品ページの比較テスト「商品テスト」機能）
9. JAN/EAN免除申請
10. ブランドレジストリ専用サポートチーム

**その他の特典（5項目）**

11. 商品動画のアップロード
12. バーチャルバンドル（複数商品のセット販売）
13. マネージユアエクスペリメント（ページ要素のテスト）
14. Brand Tailored Promotions（セグメント別プロモーション）
15. Amazon Attribution（外部チャネルの効果測定）

### 却下される主な理由と対策

| 却下理由 | 対策 |
|---------|------|
| 商品画像が不鮮明 | 高解像度で、ブランド名がはっきり読み取れる画像を使用 |
| ブランド名と商標の不一致 | 申請情報と特許庁データベースの完全一致を確認 |
| エラーコード5665 | 未承認ブランドとして認識。商標登録の完了を待つ |
| 書類の信頼性問題 | 公式書類の原本スキャンを使用 |

## 5. カテゴリ別要件・出品制限

### 出品制限がかかりやすいカテゴリ

| カテゴリ | 制限内容 |
|---------|---------|
| **食品・飲料** | 食品衛生法に基づく届出・許可が必要な場合あり |
| **ビューティー・ヘルスケア** | ブランドによる制限、薬機法関連の書類 |
| **乳幼児・子供用品・玩具** | 安全基準適合証明が必要な場合あり |
| **CD/DVD/レコード** | 出品許可申請が必要 |
| **電気・バッテリー商品** | PSEマーク・技適マークの表示が必要 |
| **無線/WiFi/Bluetooth機器** | 技術基準適合証明が必要 |
| **アパレル系ブランド** | ブランドオーナーの許可・正規仕入れ証明 |
| **医薬品・医療機器** | 薬機法に基づく許可証 |
| **酒類** | 酒類販売免許の提示 |

### 出品許可申請方法

1. セラーセントラルの「在庫」>「商品登録」から対象商品を検索
2. 「出品許可が必要です」と表示される場合、「出品許可を申請」をクリック
3. 必要書類（請求書・許可証等）をアップロード
4. 審査結果を待つ（数日〜数週間）

### 請求書要件（2025年〜厳格化）

| 条件 | 要件 |
|------|------|
| **メーカーまたは卸業者からの請求書** | 10点以上の購入実績 |
| **Amazon.co.jpからの請求書** | **50点以上の購入実績**（2025年より厳格化） |
| **発行日** | 180日以内 |
| **名義** | 出品者名・住所が一致していること |

## 6. コンプライアンス

### 主要な法規制マーク

| マーク | 対象 | 詳細 |
|-------|------|------|
| **PSEマーク** | 電気用品安全法対象製品 | 製品名板にPSEマーク・届出番号・届出者名を表示 |
| **技適マーク** | 無線機能を持つ製品 | WiFi、Bluetooth、無線通信機能を持つ全製品に必須 |
| **食品衛生法** | 食品・飲料 | 届出・許可が必要 |
| **薬機法** | 医薬品・医療機器・化粧品 | 許可証が必要 |

### 2024〜2025年の重要変更

- **モバイルバッテリー・リチウム電池**: 2025年12月以降、より厳しい安全基準への適合が必要
- **ラベル情報の不備**: 2024年にAmazon出品者の37%がラベル情報の不備により商品がリスティングから削除された

> コンプライアンス書類は事前に万全に準備すること。不備があるとリスティング削除・アカウント警告のリスクがある。
