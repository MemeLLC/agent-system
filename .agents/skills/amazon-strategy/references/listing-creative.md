## 1. メイン画像

### 必須ルール

| 項目 | 要件 |
|------|------|
| **背景** | **純粋な白（RGB: 255, 255, 255）** |
| **最小サイズ** | 500 x 500ピクセル |
| **推奨サイズ** | **1,600 x 1,600ピクセル以上**（ズーム機能有効化のため） |
| **最大サイズ** | 10,000 x 10,000ピクセル |
| **商品の占有率** | 画像面積の**85%以上**を商品が占めること |
| **アングル** | 1つの視点のみ（複数アングル不可） |
| **パッケージ** | 商品をパッケージから出した状態で撮影 |
| **付属品・小道具** | 購入者の混乱を招く付属品・小道具は使用不可 |

### メイン画像の禁止事項

- テキスト、ロゴ、縁取り、カラーブロック、透かし、その他のグラフィック
- 複数アングルの合成画像
- プレースホルダー画像やイラスト（一部カテゴリを除く）
- パッケージのみの画像（パッケージが商品そのものである場合を除く）

## 2. サブ画像

### 基本ルール

| 項目 | 要件 |
|------|------|
| **枚数** | 最大**8枚**（メイン画像とは別に） |
| **背景** | 白以外の背景もOK |
| **テキスト・グラフィック** | 追加可能（商品情報のインフォグラフィック等） |
| **ファイル形式** | JPEG（推奨）、TIFF、PNG、GIF |
| **ファイルサイズ** | **10MB以下** |
| **カラーモード** | sRGB |
| **アスペクト比** | 1:1 または 5:6 推奨 |

### 推奨構成（7〜8枚）

| 枚数 | 画像タイプ | 目的 |
|------|-----------|------|
| 1枚目 | **メイン画像** | 白背景の商品単体（必須） |
| 2枚目 | **特徴ハイライト** | 主要機能・素材のクローズアップ |
| 3枚目 | **インフォグラフィック** | スペック・寸法・機能を図解 |
| 4枚目 | **使用シーン（ライフスタイル）** | 実際の使用場面を演出 |
| 5枚目 | **サイズ比較** | 手や日用品との大きさ比較 |
| 6枚目 | **セット内容** | 同梱物の一覧 |
| 7枚目 | **テキスト付き訴求** | 差別化ポイント・保証情報 |
| 8枚目 | **追加アングル/バリエーション** | 裏面・側面・カラバリ表示 |

## 3. 画像最適化ポイント

- **ズーム機能**: 長辺1,600ピクセル以上でズーム機能が有効化。CTR・CVRに大きく影響
- **モバイル対応**: スマートフォンでの見やすさを最優先に考える
- **テキストの可読性**: インフォグラフィック内のテキストはモバイルでも読めるサイズに
- **一貫性**: 全画像で色調・トーンを統一する
- **A/Bテスト**: ブランド登録済みなら「商品テスト」機能で画像パターンを比較テスト

## 4. ガイドライン違反ペナルティ

- 商品登録自体ができない
- 検索結果に商品が表示されなくなる（検索抑制）
- 最悪の場合、アカウントへの警告

## 5. 商品動画

### アップロード条件

- **ブランド登録済み**であること（Brand Registry必須）
- 大口出品アカウント

### 技術仕様

| 項目 | 要件 |
|------|------|
| **ファイル形式** | MP4 または MOV |
| **最大ファイルサイズ** | **5GB** |
| **推奨解像度** | **1080p（1920 x 1080）** |
| **最小解像度** | 480p |
| **推奨再生時間** | **1分以内** |
| **タイトル文字数** | 60文字以内 |
| **関連付け可能ASIN数** | 最大300 ASIN/動画 |

### サムネイル画像の仕様

| 項目 | 要件 |
|------|------|
| ファイル形式 | JPG / PNG / GIF / BMP |
| ファイルサイズ | 5MB以内 |
| 推奨サイズ | 1920 x 1080（16:9比率） |
| 最小幅 | 640px |

> サムネイルを設定しない場合、Amazonが動画から自動生成する。

### 表示位置

| デバイス | 表示位置 |
|---------|---------|
| **PC** | 6番目のサブ画像サムネイルとして表示 |
| **モバイル** | 1番目または6番目のサムネイル（カテゴリにより異なる） |

### 動画コンテンツの禁止事項

- 外部リンク・WebサイトURL
- SNSアカウントの宣伝
- 他社製品との比較（ネガティブな表現）
- 価格・プロモーション情報
- 時間限定のオファー

### 動画のベストプラクティス5項目

1. **最初の3秒で引き付ける**: 動きやエフェクトで視聴者の注意を即座にキャッチ
2. **テキストオーバーレイを活用**: 多くのユーザーがミュート状態で視聴するため、字幕・テキストを追加
3. **画面下部を空ける**: 再生コントロールが重なるため、重要な情報を画面下部に配置しない
4. **商品の使用シーンを見せる**: 実際の使い方を動画で示すことでCVRが向上
5. **高品質な映像**: プロフェッショナルな品質を保つ

### 審査プロセス

- 通常**1営業日以内**に審査完了
- 繁忙期は最大**7営業日**かかる場合あり
- 却下された場合は理由が管理画面に表示される

## 6. ストアフロント

### 概要

Amazon内にブランド独自のカスタムページを作成できる機能。商品の一覧表示、ブランドストーリーの発信、カテゴリ別の整理が可能。

### 利用条件

| 条件 | 詳細 |
|------|------|
| **出品プラン** | 大口出品（月額4,900円+販売手数料） |
| **ブランド登録** | Amazon Brand Registryの完了が必須 |
| **費用** | **無料**（出品プランの月額費用以外の追加料金なし） |

### 画像・動画の仕様

| 要素 | 推奨サイズ・仕様 |
|------|-----------------|
| **ヒーロー画像** | **3,000 x 600px以上** |
| **ブランドロゴ** | **400 x 400px以上** |
| **画像ファイルサイズ** | 5MB以内/枚 |
| **背景動画** | MP4（H.264コーデック）、最小1,280 x 640px |
| **背景動画の長さ** | 2〜20秒 |
| **最大セクション数** | 20セクション/ページ |

### ストアフロント作成手順

1. セラーセントラルの「ストア」>「ストアを管理」にアクセス
2. ブランド表示名とロゴ（400x400px以上）を設定
3. ページのメタ情報（説明文）を入力
4. デザインテンプレートを選択
5. テキスト・画像・動画セクションを追加
6. 「審査に提出」をクリック
7. Amazon審査（通常24時間以内、カテゴリにより最大1週間）

### デザインテンプレート4種

| テンプレート | 特徴 | 推奨用途 |
|------------|------|---------|
| **マーキー** | 大きなヒーロー画像 + 商品タイル | ブランドの世界観を強く訴求したい場合 |
| **商品ハイライト** | 特定商品にフォーカス | 主力商品を中心に見せたい場合 |
| **商品グリッド** | 一覧形式で多くの商品を表示 | 商品数が多い場合 |
| **空白テンプレート** | 完全カスタム | 独自のレイアウトで設計したい場合 |

### ストアフロントのメリット

- **ブランド構築**: ブランドの世界観・ストーリーを伝え、リピーター獲得につなげる
- **クロスセル**: 複数カテゴリの商品を同時に見せることで追加購入を促進
- **広告連携**: スポンサーブランド広告のランディングページとして使用可能
- **分析機能**: 「Store Insights」で訪問者数、トラフィックソース、CVRを確認可能

### ストアフロント設計のベストプラクティス

- **ブランドストーリー**: 創業者のビジョン、開発プロセス、ブランドの価値観を共有
- **ビジュアルの一貫性**: カラー、フォント、ロゴの配置を統一
- **ナビゲーション**: カテゴリ・テーマ別にページを整理して回遊性を高める
- **モバイル最適化**: 1カラム表示を前提に、テキスト量を抑え、画像を中心に構成
- **レスポンシブ画像**: モバイルでの読み込み速度を考慮

### ストアフロントの禁止事項

- 最上級表現（No.1、最高等）の根拠なき使用
- 割引・セール表現
- 誤解を招く情報
- 著作権侵害
- ストアフロント名の頻繁な変更（スパム判定リスク）
