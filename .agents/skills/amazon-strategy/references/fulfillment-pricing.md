## 1. FBA（Fulfillment by Amazon）

### 概要

Amazonの物流センター（FC）に商品を預け、保管・梱包・配送・カスタマーサービス・返品対応をAmazonが代行するサービス。

### メリット

- **プライム対象**: Amazonプライム対象商品として販売可能
- **検索上位表示**: 検索結果での上位表示優遇
- **カート獲得率向上**: カートボックス獲得率の大幅向上
- **24時間365日出荷**: 深夜注文も翌朝配送可能
- **CS代行**: カスタマーサービス・返品対応をAmazonが代行
- **業務負荷軽減**: 出品者の配送業務負荷が大幅に軽減

### デメリット

- 各種手数料が発生（配送代行手数料+在庫保管手数料）
- 在庫の直接管理ができない
- 長期在庫保管手数料のリスク
- Amazonの返品ルールが適用される（出品者独自のルール不可）

## 2. FBA手数料

### 配送代行手数料（商品サイズ別）

| サイズ区分 | 手数料（税込目安） |
|-----------|------------------|
| 小型 | **288円〜** |
| 標準サイズ（小） | **413円〜** |
| 標準サイズ（大） | **573円〜** |
| 大型 | **589円〜5,625円** |

> 1,000円以下の低価格商品は配送代行手数料が**66円割引**される。

### 在庫保管手数料

- **算出方法**: 基本料金 x 商品サイズ（立方cm）x 保管日数
- 季節・サイズカテゴリにより料率が変動

### 長期在庫保管手数料（2024年4月15日〜）

| 保管期間 | 手数料/個 |
|---------|----------|
| 271〜300日 | **16.662円** |
| 301〜330日 | **17.475円** |
| 331〜365日 | **18.085円** |
| 366日以上 | **27.431円** |

> 長期在庫は利益を大きく圧迫する。在庫回転率を意識し、271日を超える前に対策を講じること。

### その他の手数料

| 手数料 | 金額 | 説明 |
|-------|------|------|
| **返送・所有権放棄手数料** | 30〜350円/個 | FBA倉庫から商品を返送・廃棄する場合 |
| **在庫不足サーチャージ** | 変動 | 販売実績に対して在庫が不足している場合に発生する可能性 |
| **過剰在庫サーチャージ** | 変動 | 販売量に対して在庫が過大な場合 |

## 3. FBM（自己発送）

### 概要

出品者自身が商品の保管・梱包・配送・カスタマーサービスを行う方式。

### メリット

- **手数料不要**: FBA手数料が不要
- **在庫管理柔軟**: 自社倉庫・外部倉庫の活用が可能
- **独自梱包**: 独自の梱包・ブランディングが可能
- **返品ルール柔軟**: 返品ルールの柔軟な設定
- **大型商品対応**: 大型商品や特殊商品に適している

### デメリット

- プライムマーク非対応（マケプレプライム除く）
- カートボックス獲得で不利
- 配送スピードでFBAに劣りやすい
- カスタマーサービスの自社対応が必要
- 出荷遅延のリスクを自社で管理

## 4. ハイブリッド運用の推奨

FBAとFBMを商品特性に応じて使い分けるハイブリッド運用が最も効果的。

| 商品タイプ | 推奨配送方法 | 理由 |
|-----------|------------|------|
| **売れ筋の定番品** | FBA | スピード・カート獲得を重視 |
| **大型・重量商品** | FBM | コスト管理 |
| **ギフト対応・特殊梱包が必要な商品** | FBM | 丁寧に対応 |
| **新商品テスト** | FBM | 少量から開始し、需要確認後にFBA移行 |
| **季節商品** | 使い分け | FBAの長期保管手数料を考慮 |

## 5. カートボックス獲得戦略

### カートボックスの重要性

商品詳細ページの「カートに入れる」ボタンの横に表示される出品者枠。同一商品に複数の出品者がいる場合、カートボックスを獲得した出品者が優先的に表示される。

> **売上の約80%以上**がカートボックス経由で発生。獲得は売上に直結する。

### 獲得条件6項目

| 条件 | 詳細 |
|------|------|
| 1. **大口出品** | 大口出品のみ（小口出品は獲得不可） |
| 2. **競争力のある価格** | 他出品者と同等以下の「販売価格 + 送料」 |
| 3. **アカウント健全性** | ODR 1%未満、出荷遅延率 4%未満等 |
| 4. **在庫の確保** | 在庫切れは即座にカート喪失 |
| 5. **FBAの利用** | FBA利用者はカート獲得で大幅に優遇される |
| 6. **出品者評価** | 高い出品者フィードバック評価 |

## 6. 価格設定戦略

### 基本原則

- 「販売価格 + 送料」の総額が最も重要
- Amazonのアルゴリズムはコストパフォーマンスの高い出品者を優先
- 極端な値下げは品質への疑念を招くため注意

### 価格競争力の維持方法

| 方法 | 詳細 |
|------|------|
| **自動価格調整** | セラーセントラルの「自動価格設定」機能を活用 |
| **FBA活用** | 送料込みの総額で競争力を確保 |
| **セット販売** | 単品価格競争を避け、バンドル商品で差別化 |
| **モニタリング** | 競合価格の変動を週次で確認 |

## 7. 利益計算

### 利益計算式

```
利益 = 売上 − (仕入原価 + Amazon販売手数料 + FBA手数料 + 広告費 + 配送費 + その他経費)
```

### 損益分岐ACoSの計算

```
損益分岐ACoS = 利益率（%）

例:
  商品価格: 3,000円
  原価 + FBA手数料 + Amazon手数料: 1,800円
  利益: 1,200円
  利益率: 40%

  → 損益分岐ACoS = 40%
  → ACoSが40%以下であれば広告でも利益が出る
  → 目標ACoS: 25%（利益確保 + 再投資余地）
```
