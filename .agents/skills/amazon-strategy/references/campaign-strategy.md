## フルファネル広告アプローチ

Amazon広告は購買ファネルの各段階に対応する広告タイプを組み合わせることで最大効果を発揮する。

### ファネル構造と広告タイプの対応

```
┌─────────────────────────────────────────┐
│           認知（Awareness）              │
│   DSP OTT / DSP OLV / DSP ディスプレイ  │
│   → ブランドを知らない潜在顧客にリーチ    │
├─────────────────────────────────────────┤
│           検討（Consideration）          │
│   SB動画 / SB商品コレクション / SD       │
│   → ブランド・商品を認知した層に訴求      │
├─────────────────────────────────────────┤
│           購入（Purchase）               │
│   SP / SB / SD（コンバージョン最適化）    │
│   → 購買意欲の高いユーザーを獲得         │
├─────────────────────────────────────────┤
│        ロイヤルティ（Loyalty）            │
│   SD リマーケティング / DSP リターゲ     │
│   Brand Tailored Promotions             │
│   → 既存顧客のリピート・LTV向上          │
└─────────────────────────────────────────┘
```

### ファネル別KPI

| ファネル段階 | 主要KPI | 目標値目安 |
|-------------|---------|-----------|
| 認知 | インプレッション、リーチ、動画視聴完了率 | 視聴完了率 25%以上 |
| 検討 | CTR、DPV（商品詳細ページビュー）、ブランド新規率 | CTR 0.5%以上、新規率 60%以上 |
| 購入 | CVR、ACoS、ROAS | ACoS 目標以下、ROAS 目標以上 |
| ロイヤルティ | リピート率、LTV、TACoS | リピート率前月比+5% |

## 予算配分

### 初期予算配分（Amazon広告開始〜3ヶ月目）

| 広告タイプ | 配分比率 | 理由 |
|-----------|---------|------|
| **SP（スポンサープロダクト）** | **70%** | 最もROASが高く、データ収集の起点 |
| **SB（スポンサーブランド）** | **20%** | ブランド認知とカテゴリ内プレゼンス確立 |
| **SD（スポンサーディスプレイ）** | **10%** | リターゲティングの基盤構築 |

### 成長期予算配分（4ヶ月目〜）

SP/SB/SDが安定稼働し、ファネル全体をカバーする段階。

| ファネル段階 | 配分比率 | 含む広告タイプ |
|-------------|---------|---------------|
| **TOF（Top of Funnel）** | **40%** | DSP、SB動画、SD（リーチ目標） |
| **MOF（Middle of Funnel）** | **30%** | SB商品コレクション、SD（ページ訪問目標） |
| **BOF（Bottom of Funnel）** | **30%** | SP、SD（コンバージョン目標） |

### 新ブランドローンチ時の特別配分

| ファネル段階 | 配分比率 | 理由 |
|-------------|---------|------|
| **TOF** | **60%以上** | まずブランド認知を確立する必要がある |
| **MOF** | **25%** | 認知したユーザーの検討促進 |
| **BOF** | **15%** | 初期は売上よりも認知・検討を優先 |

### 月間予算の目安

| 事業規模 | 月間予算目安 | 含む広告タイプ |
|----------|------------|---------------|
| 小規模（テスト） | 3〜10万円 | SP のみ |
| 中規模（成長期） | 10〜50万円 | SP + SB + SD |
| 大規模（本格運用） | 50〜200万円 | SP + SB + SD |
| エンタープライズ | 200万円以上 | SP + SB + SD + DSP |

## キャンペーン構造4種

### 1. ブランド防衛キャンペーン

| 項目 | 設定 |
|------|------|
| 目的 | 自社ブランド名検索時のシェア確保 |
| ターゲティング | 自社ブランド名の完全一致キーワード |
| 目標ACoS | **10%以下** |
| 入札戦略 | ダイナミックダウン or 固定 |
| 予算 | 全体の10-15% |
| 優先度 | ★★★ 最優先 |

**ブランド防衛が重要な理由:**

- 自社ブランド名で検索するユーザーは最もCVRが高い（30-50%）
- 競合がブランドKWに入札している場合、自社商品が表示されないリスク
- 最もCPCが低く、ROASが高いキャンペーン
- ブランド検索結果の「面」を自社で確保

**設定例:**

```
キャンペーン: SP-ブランド防衛
キーワード: [ブランド名] 完全一致
入札: 50-100円（競合状況により調整）
Top of Search調整: +50%
ACoS目標: 10%以下
```

### 2. カテゴリキャンペーン

| 項目 | 設定 |
|------|------|
| 目的 | 一般的なカテゴリKWでの露出・獲得 |
| ターゲティング | カテゴリ関連の一般KW |
| 目標ACoS | **25-30%** |
| 入札戦略 | ダイナミックアップ&ダウン |
| 予算 | 全体の40-50% |
| 優先度 | ★★★ 最優先 |

**マッチタイプの段階的移行:**

| フェーズ | マッチタイプ | 目的 |
|----------|------------|------|
| 1（1-2週） | 部分一致（Broad） | 幅広いKW発掘 |
| 2（3-4週） | フレーズ一致（Phrase） | 有効なフレーズの特定 |
| 3（5週〜） | 完全一致（Exact） | 高効率KWへの集中投下 |

**注意:** 各フェーズは並行運用。Broadで発見したKWをPhrase/Exactに移行し、元のBroadではネガティブKWとして除外。

### 3. 競合ターゲティングキャンペーン

| 項目 | 設定 |
|------|------|
| 目的 | 競合ブランド/商品の検索・閲覧ユーザーを獲得 |
| ターゲティング | 競合ブランド名KW + 競合ASIN |
| 目標ACoS | **30-40%** |
| 入札戦略 | ダイナミックダウン |
| 予算 | 全体の15-20% |
| 優先度 | ★★☆ 中優先 |

**競合ターゲティングの注意点:**

- 競合ブランド名のKWは直接的な比較広告にならないよう注意
- 自社の差別化ポイント（価格、品質、レビュー）で優位性がある場合に有効
- CVRはブランドKWより低いため、ACoS目標は緩めに設定
- 商品ターゲティング（ASIN指定）はSD広告との併用が効果的

### 4. キャッチオール（オート発掘）キャンペーン

| 項目 | 設定 |
|------|------|
| 目的 | 新しいKW・ASINの発掘、データ収集 |
| ターゲティング | オートターゲティング（低入札） |
| 目標ACoS | **制限なし**（データ収集が目的） |
| 入札戦略 | 固定入札（低め） |
| 予算 | 全体の10-15% |
| 優先度 | ★★☆ 常時稼働 |

**運用ポイント:**

- 入札額はカテゴリ平均の50-70%に設定
- 週次でサーチタームレポートを確認
- 高パフォーマンスKW → マニュアルキャンペーンに移行
- 低パフォーマンスKW → ネガティブKWに追加
- 常時稼働で継続的にデータを収集

## キーワードリサーチ

### Amazon内ツール

| ツール | 説明 | 費用 |
|--------|------|------|
| **Brand Analytics** | ブランド登録者向け。検索頻度ランク、クリックシェア、CVシェア | 無料（ブランド登録必須） |
| **Product Opportunity Explorer** | 市場ニッチ、検索ボリューム、トレンドを分析 | 無料（ブランド登録必須） |
| **サーチタームレポート** | SP/SB広告の実際の検索クエリデータ | 無料（広告出稿必須） |
| **Amazonサジェスト** | 検索バーの自動補完キーワード | 無料 |

### 外部ツール

| ツール | 主要機能 | 費用目安 |
|--------|---------|---------|
| **Helium 10** | Cerebro（逆ASIN検索）、Magnet（KW検索）、Xray（市場分析） | $39-399/月 |
| **Jungle Scout** | Keyword Scout、競合分析、セールス推定 | $49-399/月 |
| **SellerSprite** | KW分析、市場分析、日本市場に強い | ¥2,980-9,980/月 |

### KWリサーチ6ステップ手順

#### ステップ1: シードKWの収集

- 自社商品のメイン商品名・カテゴリ名をリストアップ
- Amazonサジェストで関連KWを収集
- 競合商品のタイトル・バレットポイントからKWを抽出

#### ステップ2: KWの拡張

- Helium 10 Cerebro で競合ASINの流入KWを分析
- Brand Analytics で検索頻度ランク上位のKWを確認
- Product Opportunity Explorer でニッチKWを発見

#### ステップ3: 検索ボリュームの確認

- 各KWの月間検索ボリュームを確認
- 検索ボリュームの大きさでKWを分類（ヘッド/ミドル/ロングテール）

#### ステップ4: 競合性の評価

- 各KWの競合数・広告出稿数を確認
- CPC見込み額を確認
- 競合の強さ（レビュー数、BSRランク）を評価

#### ステップ5: 関連性のスコアリング

- 自社商品との関連性を1-5で評価
- CVRの見込みを評価（高関連性 = 高CVR）
- 不適切なKWを除外

#### ステップ6: KWリストの構造化

| 分類 | KW例（プロテインの場合） | 優先度 |
|------|--------------------------|--------|
| ブランドKW | [自社ブランド名] プロテイン | 最優先 |
| コアカテゴリKW | プロテイン、ホエイプロテイン | 高 |
| 特徴KW | プロテイン チョコ味、プロテイン 1kg | 高 |
| 用途KW | 筋トレ プロテイン、ダイエット プロテイン | 中 |
| 競合KW | [競合ブランド名] プロテイン | 中 |
| ロングテールKW | プロテイン 人工甘味料不使用 国産 | 中 |

## ネガティブキーワード

### 2つのマッチタイプ

| タイプ | 説明 | 用途 |
|--------|------|------|
| **完全一致（Exact）** | 指定した語句と完全に一致するクエリのみ除外 | 特定のクエリを精密に除外 |
| **フレーズ一致（Phrase）** | 指定したフレーズを含むすべてのクエリを除外 | 幅広く不適切なクエリを除外 |

### 除外すべきキーワードの種類

| 種類 | 例 | 理由 |
|------|-----|------|
| **無関係KW** | 全く関係のない商品名 | CVR 0%、無駄クリック |
| **低CVR KW** | 20クリック以上0コンバージョン | 費用対効果が悪い |
| **他マッチタイプへ移行済みKW** | オート→マニュアルへ移行したKW | 重複入札の防止 |
| **低利益KW** | CPC > 利益の30% となるKW | 採算が合わない |
| **情報検索KW** | 「〇〇 とは」「〇〇 比較」 | 購買意欲が低い |
| **返品関連KW** | 「〇〇 返品」「〇〇 解約」 | ネガティブ意図 |

### 週次レビュー運用

| ステップ | アクション | 判断基準 |
|----------|-----------|---------|
| 1 | サーチタームレポートをダウンロード | 週次（毎週月曜日推奨） |
| 2 | クリック数でソート | 上位100クエリを確認 |
| 3 | 0コンバージョンKWを抽出 | 10クリック以上かつ0 CV |
| 4 | 高ACoS KWを抽出 | ACoS目標の2倍以上 |
| 5 | ネガティブKWとして追加 | 適切なマッチタイプで追加 |
| 6 | 高CVR KWをマニュアルへ移行 | CVR 10%以上 or ACoS目標以下 |

## デイパーティング

### 日本市場のピーク時間帯

| 時間帯 | 特徴 | 入札調整推奨 |
|--------|------|------------|
| **6:00-8:00** | 通勤時間。モバイル閲覧増加 | +10-20% |
| **12:00-13:00** | **昼休みピーク**。モバイル購買増加 | **+20-30%** |
| **14:00-18:00** | 日中。安定的なトラフィック | 基準（±0%） |
| **19:00-23:00** | **夜間ゴールデンタイム**。最大の購買ピーク | **+30-50%** |
| **23:00-6:00** | 深夜。トラフィック減少 | -20-50% |

### 曜日別の傾向

| 曜日 | 特徴 | 入札調整 |
|------|------|---------|
| 月〜金（平日） | 通勤時間+昼休み+夜間がピーク | 基準 |
| **土曜** | **日中全体がピーク**。家でゆっくり閲覧 | **+10-20%** |
| **日曜** | **最もCVRが高い**。購買決定が多い | **+15-25%** |

### デイパーティングの実装

> **注意:** Amazon広告にはネイティブのデイパーティング機能がない。以下の方法で疑似的に実装。

1. **日予算の時間帯別配分:** 朝〜昼は低予算、夜間に予算を残す
2. **ルールベース入札（SP）:** 時間帯別のROAS目標を設定
3. **DSP:** DSPでは時間帯別の入札調整が可能
4. **手動調整:** ピーク時間前に入札を上げ、オフピーク時に下げる

## シーズナル戦略

### 4段階のシーズナル運用

| 段階 | 期間 | アクション |
|------|------|-----------|
| **1. 事前準備** | イベント2週間前 | 予算増額（1.5-2倍）、入札引き上げ（+20-30%）、クリエイティブ更新、在庫確認 |
| **2. 期間中** | イベント期間 | 予算最大化（2-3倍）、入札積極化（+30-50%）、リアルタイムモニタリング、予算切れ防止 |
| **3. 直後** | イベント終了〜1週間 | 予算を徐々に戻す、リターゲティング強化（SD/DSP）、データ分析 |
| **4. オフシーズン** | 通常期 | 通常予算に戻す、次回に向けたデータ分析、クリエイティブ制作開始 |

### 日本の主要イベント

| イベント | 時期 | 特徴 | 注力カテゴリ |
|----------|------|------|-------------|
| **初売り** | 1月1日〜 | 年始の購買意欲が高い | 全カテゴリ |
| **バレンタインデー** | 2月14日前2週間 | ギフト需要 | 食品、ファッション、美容 |
| **新生活シーズン** | 3月〜4月 | 引っ越し、入学、入社 | 家電、家具、日用品 |
| **プライムデー** | 7月中旬（予定） | **Amazon最大セール**。全カテゴリ爆発 | 全カテゴリ（特に家電） |
| **ブラックフライデー** | 11月第4金曜〜 | 大幅値引きセール | 全カテゴリ |
| **サイバーマンデー** | BF翌週月曜 | デジタル・家電中心 | 家電、ガジェット |
| **年末商戦** | 12月 | クリスマス+年末 | ギフト、おもちゃ、食品 |

### プライムデー対策（最重要）

| フェーズ | 期間 | 施策 |
|----------|------|------|
| 準備期 | 1ヶ月前 | 在庫確保（通常の3倍）、クリエイティブ制作、KWリスト拡充 |
| ウォームアップ | 2週間前 | 予算+50%、SB/SD開始、セール告知 |
| 当日 | プライムデー中 | 予算3倍、入札+50-100%、1時間ごとに監視 |
| アフター | 終了後1週間 | リターゲティング強化、レビュー促進 |

## 新商品ローンチ4フェーズ戦略

### フェーズ1: プレローンチ（ローンチ2-4週間前）

| 施策 | 詳細 |
|------|------|
| **Vine** | Amazon Vineプログラムで最大30件のレビューを獲得 |
| **リスティング最適化** | タイトル、バレットポイント、A+コンテンツ、画像7枚の完全準備 |
| **KWリサーチ** | 競合分析、メインKW/サブKW/ロングテールKWの特定 |
| **在庫確保** | FBA納品完了、十分な在庫量の確保 |
| **価格設定** | 導入価格の決定（通常より10-15%低めで開始する場合も） |

### フェーズ2: 初期（ローンチ後1-30日）

| 項目 | 設定 |
|------|------|
| **予算** | 通常想定の**2-3倍** |
| **SP-オート** | 4マッチグループすべてON、標準入札の1.2倍 |
| **SP-マニュアル部分一致** | メインKW 20-30個、標準入札の1.5倍 |
| **SP-マニュアルフレーズ一致** | メインKW 10-20個、標準入札の1.3倍 |
| **目標** | データ収集優先。ACoSは緩め（40-50%許容） |
| **モニタリング** | 日次でサーチターム確認、3日ごとに入札調整 |

### フェーズ3: 最適化期（31-60日）

| 項目 | 設定 |
|------|------|
| **SP-完全一致追加** | フェーズ2で発見した高CVR KWを完全一致で個別キャンペーン化 |
| **SB追加** | 商品コレクション + SB動画（動画がある場合）の開始 |
| **競合ASIN** | 商品ターゲティングで競合上位5 ASINを指定 |
| **ネガティブKW** | 低パフォーマンスKWの除外 |
| **目標** | ACoS 25-35%に改善 |
| **モニタリング** | 週次でKW/ASIN調整 |

### フェーズ4: スケール期（61-90日+）

| 項目 | 設定 |
|------|------|
| **SD追加** | 閲覧リマーケ + 競合商品ターゲティング |
| **DSP検討** | 月間予算150万円以上の場合はDSP導入を検討 |
| **Brand Tailored Promotions** | リピート顧客向け割引プログラム開始 |
| **マッチタイプ移行** | Broad → Phrase → Exact の段階的移行完了 |
| **目標** | ACoS 15-25%（カテゴリ平均以下） |
| **モニタリング** | 週次KPI確認、月次で戦略レビュー |

### ローンチ90日間の予算推移

| 期間 | 日予算目安 | 月予算目安 | 広告タイプ |
|------|-----------|-----------|-----------|
| 1-30日 | 5,000-10,000円 | 15-30万円 | SP（オート+マニュアル） |
| 31-60日 | 8,000-15,000円 | 24-45万円 | SP + SB |
| 61-90日 | 10,000-20,000円 | 30-60万円 | SP + SB + SD |
| 90日以降 | 最適化に基づき調整 | - | SP + SB + SD（+ DSP） |

### Amazon Attribution

| 項目 | 詳細 |
|------|------|
| 概要 | Amazon外の広告（Google、Meta、TikTok等）のAmazon内売上貢献を計測 |
| 費用 | 無料 |
| 対応チャネル | Google Ads、Meta Ads、メール、SNS、インフルエンサー等 |
| 計測項目 | クリック、商品詳細ページビュー、カート追加、購入、売上 |
| 活用法 | Amazon外広告のROI測定、予算配分の最適化 |

### Brand Tailored Promotions

既存顧客セグメントに対して、パーソナライズされた割引クーポンを配信するプログラム。

#### 8つのオーディエンスセグメント

| # | セグメント | 説明 | 推奨割引率 |
|---|-----------|------|-----------|
| 1 | **ブランドカート放棄者** | カートに入れたが未購入 | 10-15% |
| 2 | **ブランド離脱リスク** | 過去購入者で最近購入なし | 15-20% |
| 3 | **ブランドリピーター** | 複数回購入の優良顧客 | 10% |
| 4 | **ブランドフォロワー** | ブランドをフォロー中 | 10-15% |
| 5 | **高額購入者** | 購入金額上位の顧客 | 10% |
| 6 | **最近の購入者** | 直近で購入した顧客 | 10%（クロスセル向け） |
| 7 | **潜在的な新規顧客** | 類似商品を閲覧/検討中 | 15-20% |
| 8 | **トップティア顧客** | 最も価値の高い顧客セグメント | 10-15%（特別感演出） |

#### 割引設定

| 項目 | 詳細 |
|------|------|
| 割引率 | **10-50%**（10%刻み） |
| 最低割引率 | 10% |
| 最大割引率 | 50% |
| 費用 | 割引額はブランド負担、配信は無料 |
| 表示場所 | 商品詳細ページ、セール・クーポンページ |

### Amazon Vine

| 項目 | 詳細 |
|------|------|
| 概要 | Amazonが招待した信頼性の高いレビュアーに無料で商品を提供し、レビューを獲得 |
| 最大レビュー数 | **30件** |
| 費用 | **0〜$200**（ASINあたり、登録時にプランを選択） |
| 対象 | レビュー30件未満の商品 |
| レビュー品質 | 詳細で信頼性の高いレビュー。星評価の操作は不可 |
| 期間 | 登録から30日以内にレビュアーへ発送 |

**Vine活用のポイント:**

- 新商品ローンチ時に最優先で登録
- FBA在庫として30個（最大）を準備
- レビューが集まるまで2-4週間かかる
- Vine開始後にSP広告を強化するとレビュー付き商品での広告効果が向上
