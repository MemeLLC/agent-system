## 概要

スポンサープロダクト広告はAmazon広告の中核であり、最も利用率が高い広告タイプ。検索連動型のCPC（クリック課金）広告で、ショッピング結果ページや商品詳細ページに表示される。

| 項目 | 内容 |
|------|------|
| 課金方式 | CPC（クリック課金） |
| 最低日予算 | 100円（推奨: 1,000円以上） |
| 最低入札額 | 2円 |
| 利用条件 | 大口出品プラン + ショッピングカートボックス獲得が必要 |
| ブランド登録 | 不要（ただしSP動画はブランド登録必須） |
| 対象商品 | 新品のみ（中古品・再生品は不可） |
| 広告表示 | 「スポンサー」ラベル付きで表示 |
| 審査 | 通常72時間以内 |

**SP広告が最重要である理由:**

- Amazon広告費全体の約75-80%を占める主力広告
- 購買意欲の高いユーザーに直接リーチ
- 最もROASが高い広告タイプ（平均ROAS 5-7x）
- データ収集の起点として他広告タイプの最適化に貢献

## 表示場所と入札調整

### 3つの表示場所

| 表示場所 | 説明 | 特徴 |
|----------|------|------|
| **Top of Search（検索結果最上部）** | 検索結果1ページ目の最上段 | CTR最高、CVR最高、CPC最高。最も競争が激しい |
| **Rest of Search（検索結果その他）** | 検索結果1ページ目の中段・下段・2ページ目以降 | CPCが比較的低い。露出量が多い |
| **Product Pages（商品詳細ページ）** | 競合商品の詳細ページ、カート下、レビュー横など | 競合からの奪取に有効。インプレッション最多 |

### プレースメント別入札調整

| プレースメント | 調整幅 | 推奨設定 |
|----------------|--------|----------|
| Top of Search | 最大 **+900%**（実質10倍） | ブランドKW: +50〜100%、一般KW: +20〜50% |
| Rest of Search | 最大 **+900%** | 通常は調整なし（0%） |
| Product Pages | 最大 **+900%** | 競合ターゲティング時: +20〜40% |

> **注意:** プレースメント調整はベース入札額に対して加算される。ベース入札100円 + Top of Search +50% = Top of Searchでの入札は150円。

## ターゲティング

### オートターゲティング

Amazonのアルゴリズムが商品情報（タイトル、説明、バレットポイント、バックエンドキーワード）を解析し、関連する検索クエリや商品に自動で広告を表示する。

#### 4つのマッチグループ

| マッチグループ | 説明 | 用途 | 推奨入札 |
|----------------|------|------|----------|
| **Close Match（類似一致）** | 商品に密接に関連するキーワードに一致 | コアKW発掘。最もCVRが高い | 基準入札の100% |
| **Loose Match（大まかな一致）** | 商品に間接的に関連するキーワードに一致 | ロングテールKW発掘。リーチ拡大 | 基準入札の70-80% |
| **Substitutes（代替品）** | 自社商品の代替となる類似商品ページに表示 | 競合からの奪取 | 基準入札の80-90% |
| **Complements（補完品）** | 自社商品の補完となる商品ページに表示 | クロスセル機会の発見 | 基準入札の50-70% |

**オートターゲティングの運用ポイント:**

- 各マッチグループの入札額は個別に設定可能
- 成果の低いマッチグループは入札を下げるか、ネガティブターゲティングで除外
- 週次でサーチタームレポートを確認し、高パフォーマンスKWをマニュアルキャンペーンに移行
- 新商品ローンチ時はまずオートで2-4週間データ収集

### マニュアルターゲティング

#### キーワードターゲティング（3マッチタイプ）

| マッチタイプ | 説明 | 例（KW:「プロテイン」） | 用途 | CPC傾向 |
|--------------|------|--------------------------|------|----------|
| **Broad（部分一致）** | KWを含む検索クエリ全般（語順不問、類義語含む） | 「ホエイ プロテイン おすすめ」「プロテイン バー チョコ」 | KW発掘・リーチ拡大 | 低〜中 |
| **Phrase（フレーズ一致）** | KWのフレーズが語順通り含まれる検索クエリ | 「プロテイン おすすめ」「プロテイン 安い」 | 中程度の精度とリーチのバランス | 中 |
| **Exact（完全一致）** | KWと完全に一致する検索クエリのみ | 「プロテイン」のみ | 高CVR・高効率のKWに集中投下 | 高 |

**マッチタイプ運用の段階的移行:**

1. **発掘フェーズ（1-2週目）:** Broad中心 → サーチタームレポートで有効KW特定
2. **絞り込みフェーズ（3-4週目）:** 有効KWをPhrase/Exactに移行
3. **最適化フェーズ（5週目〜）:** Exact中心 + 継続的なBroadでの新KW発掘

#### 商品ターゲティング

| タイプ | 説明 | 用途 |
|--------|------|------|
| **個別ASIN** | 特定の商品（ASIN）の詳細ページに広告表示 | 競合の特定商品からの奪取、補完商品へのクロスセル |
| **カテゴリ** | 指定カテゴリ全体の商品に広告表示 | リーチ拡大、新しいターゲット商品の発見 |

**カテゴリターゲティングの絞り込みオプション:**

- ブランド名
- 価格帯（自社より高い商品を指定 → 割安感を演出）
- レビュー評価（自社より低い商品を指定 → 品質優位を訴求）
- プライム配送有無

## 入札戦略

### 4つの入札戦略

| 戦略 | 説明 | 適用場面 |
|------|------|----------|
| **固定入札** | 設定した入札額を常に使用 | 予算管理を厳密にしたい場合。テスト初期 |
| **ダイナミックダウンのみ** | CVの見込みが低い場合に入札を自動引き下げ | 安定運用期。無駄な支出を抑制 |
| **ダイナミックアップ&ダウン** | CVの見込みに応じて入札を自動引き上げ/引き下げ | 積極的な売上拡大期。データが十分にある場合 |
| **ルールベース入札** | 指定したROAS目標に基づき入札を自動調整 | 明確なROAS目標がある場合。大規模運用 |

### ダイナミック入札の調整幅

| プレースメント | アップ幅上限 | ダウン幅 |
|----------------|-------------|----------|
| Top of Search | 最大 **+100%** | 最大 -100%（0円まで） |
| その他のプレースメント | 最大 **+50%** | 最大 -100%（0円まで） |

**入札戦略の選択フロー:**

```
新規キャンペーン → 固定入札 or ダイナミックダウン（2-4週間）
    ↓ データ蓄積後
中期運用 → ダイナミックアップ&ダウン
    ↓ ROAS目標が明確
長期安定運用 → ルールベース入札
```

## SP動画（2025年新機能）

ブランド登録済みの出品者が利用可能。検索結果ページで自動再生される動画形式のSP広告。

### 動画仕様

| 項目 | 仕様 |
|------|------|
| 動画の長さ | 6〜45秒（推奨: 15〜30秒） |
| アスペクト比 | 16:9 |
| 解像度 | 1920x1080px（最低1280x720px） |
| ファイルサイズ | 500MB以下 |
| ファイル形式 | MP4、MOV |
| 映像コーデック | H.264 |
| 音声コーデック | AAC、128kbps以上 |
| 再生方式 | ミュート自動再生（タップで音声ON） |

### SP動画のベストプラクティス

- 最初の3秒で商品の最大の特徴を視覚的に伝える
- テキストオーバーレイを使用（ミュート視聴前提）
- 商品の使用シーンを見せる
- ループ再生を意識した構成（開始と終了をシームレスに）
- 黒帯（レターボックス/ピラーボックス）は不可

## キャンペーン構成ベストプラクティス

### 推奨6キャンペーン構造

| # | キャンペーン名 | ターゲティング | マッチタイプ | 目標ACoS | 目的 |
|---|----------------|----------------|-------------|----------|------|
| 1 | **SP-オート発掘** | オートターゲティング | 全4グループ | 30-40% | 新KW・ASIN発掘 |
| 2 | **SP-ブランド防衛** | マニュアルKW | 完全一致 | 5-10% | 自社ブランド名の防衛 |
| 3 | **SP-カテゴリ部分** | マニュアルKW | 部分一致+フレーズ一致 | 25-35% | カテゴリKWでのリーチ拡大 |
| 4 | **SP-カテゴリ完全** | マニュアルKW | 完全一致 | 15-25% | 高CVRキーワードへの集中投下 |
| 5 | **SP-競合** | マニュアルKW | フレーズ一致+完全一致 | 30-40% | 競合ブランドKWでの奪取 |
| 6 | **SP-商品ターゲ** | 商品ターゲティング | ASIN+カテゴリ | 20-30% | 競合・補完商品からの獲得 |

### キャンペーン間のKW移行フロー

```
SP-オート発掘 → 高CVR KW抽出 → SP-カテゴリ部分（部分一致）
    ↓                                    ↓
ネガティブKW追加              高CVR KW抽出 → SP-カテゴリ完全（完全一致）
                                              ↓
                                    入札最適化・スケール
```

## 予算の目安

### フェーズ別推奨予算

| フェーズ | 日予算 | 月予算目安 | 説明 |
|----------|--------|------------|------|
| **テスト期** | 500〜1,000円/日 | 15,000〜30,000円 | データ収集・KW発掘が目的 |
| **本格運用期** | 3,000〜10,000円/日 | 90,000〜300,000円 | 最適化しながら売上拡大 |
| **積極拡大期** | 10,000円+/日 | 300,000円+ | シェア拡大・カテゴリ制覇 |

### 予算配分の考え方

- **商品数が少ない場合（1-5 ASIN）:** 1キャンペーンあたり500-1,000円/日
- **商品数が多い場合（10+ ASIN）:** カテゴリ/ブランドラインごとにグルーピング
- **オートキャンペーン:** 全体予算の20-30%を割り当て
- **ブランド防衛:** 全体予算の10-15%（低CPC高CVRのため少額で十分）

## 2.5ルール

SP広告のCPC入札額を決める簡易指標。

**計算式:**

```
目標CPC = 商品販売価格 x 0.025（2.5%）
```

**具体例:**

| 商品価格 | 目標CPC | 説明 |
|----------|---------|------|
| 1,000円 | 25円 | 低価格帯商品。ACoSに注意 |
| 2,000円 | 50円 | 標準的な入札レンジ |
| 3,000円 | 75円 | 中価格帯。余裕のある入札 |
| 5,000円 | 125円 | 高価格帯。積極的な入札可能 |
| 10,000円 | 250円 | 高単価商品。Top of Search狙い |

> **補足:** 2.5ルールはあくまで出発点。実際の入札額はACoS目標、利益率、競合状況に応じて調整する。利益率が高い商品は3-4%、低い商品は1.5-2%に調整。

## クリエイティブ仕様

SP広告のクリエイティブは商品リスティング（カタログ）の情報がそのまま使用される。広告専用のクリエイティブは作成不要だが、リスティング最適化が広告パフォーマンスに直結する。

### メイン画像

| 項目 | 仕様 |
|------|------|
| 最小サイズ | 1,000 x 1,000px |
| 推奨サイズ | 2,000 x 2,000px（ズーム機能有効化） |
| 最大サイズ | 10,000 x 10,000px |
| アスペクト比 | 1:1（正方形推奨） |
| 背景 | 純白（RGB 255,255,255） |
| 商品占有率 | フレームの85%以上 |
| ファイル形式 | JPEG、PNG、TIFF、GIF（非アニメーション） |
| ファイルサイズ | 10MB以下 |

**メイン画像の禁止事項:**

- テキスト、ロゴ、透かしの追加
- 複数商品の表示（バリエーションパックを除く）
- プロップ・背景の使用
- ボーダー、色付き背景
- 「ベストセラー」等のバッジ

### 商品タイトル

| 項目 | 仕様 |
|------|------|
| 最大文字数 | 200文字（全角） |
| 検索結果表示 | 約80文字（それ以降は切り捨て） |
| モバイル表示 | 約60文字 |

**タイトル構成の推奨フォーマット:**

```
[ブランド名] [商品名] [重要な特徴/成分] [サイズ/容量] [色/フレーバー] [数量]
```

**例:** `ブランドX プロテイン ホエイ100% チョコレート味 1kg 国内製造`

## 成果最大化のベストプラクティス

### リスティング最適化（広告前の必須準備）

1. **タイトル:** 主要KWを前半に配置、80文字以内で要点を伝える
2. **バレットポイント:** 5つすべてを活用、各100-150文字、KWを自然に含める
3. **商品説明:** A+コンテンツ（ブランド登録者）を活用、ビジュアルリッチに
4. **画像:** メイン+6枚のサブ画像（使用シーン、サイズ感、成分表など）
5. **バックエンドKW:** 250バイト以内、タイトルと重複しないKWを設定
6. **価格:** 競合と比較して適正価格を設定
7. **レビュー:** 最低15件、星4.0以上を目指す（Vineプログラム活用）
8. **在庫:** FBA利用でプライムバッジ獲得

### 運用最適化

- **週次:** サーチタームレポート確認、ネガティブKW追加、入札調整
- **隔週:** 新KW追加、マッチタイプ移行判断
- **月次:** キャンペーン構造見直し、予算再配分、ACoS/ROAS分析
- **四半期:** 全体戦略レビュー、新キャンペーンテスト

### パフォーマンス判断基準

| 指標 | 良好 | 要改善 | 対処法 |
|------|------|--------|--------|
| ACoS | 目標以下 | 目標の1.5倍以上 | 入札下げ、KW精査、リスティング改善 |
| CTR | 0.4%以上 | 0.2%以下 | 画像改善、タイトル改善、価格調整 |
| CVR | 10%以上 | 5%以下 | リスティング改善、レビュー増加、価格調整 |
| インプレッション | 十分 | 極端に少ない | 入札上げ、KW追加、予算増加 |
