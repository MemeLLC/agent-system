## 概要

スポンサーディスプレイ広告はAmazon内外でリターゲティングやオーディエンスベースの配信が可能なディスプレイ広告。購買ファネルの中間〜上部をカバーし、SPやSBでリーチできないユーザーにアプローチする。

| 項目 | 内容 |
|------|------|
| 課金方式 | **CPC（クリック課金）** または **vCPM（ビューアブルインプレッション課金）** |
| 最低日予算 | 100円 |
| 最低CPC | 2円 |
| 最低vCPM | 100円 |
| 利用条件 | **Amazonブランド登録が必須** |
| 審査 | 通常72時間以内 |

**SD広告の位置づけ:**

- SP/SBがカバーしない「検討」「リターゲティング」領域を補完
- Amazon外（Twitch、IMDb等）への配信で購買ファネル上部をカバー
- リマーケティングによる離脱ユーザーの回収
- 競合商品閲覧者への自社商品訴求

## 配信面一覧

SD広告はAmazon内外の幅広い配信面に表示される。

| 配信面 | 場所 | 説明 |
|--------|------|------|
| **Amazon商品詳細ページ** | Amazon内 | 競合商品の詳細ページに表示。最もCVRが高い |
| **Amazon検索結果ページ** | Amazon内 | 検索結果のサイドバーやインライン |
| **Amazonトップページ** | Amazon内 | Amazon.co.jpのトップページバナー |
| **Twitch** | Amazon外 | ゲーム・エンタメ配信プラットフォーム |
| **IMDb** | Amazon外 | 映画・TV情報サイト |
| **Amazon Freevee** | Amazon外 | 無料ストリーミングサービス |
| **サードパーティサイト** | Amazon外 | Amazon Publisher Services経由の外部サイト |

**配信面のコントロール:**

- Amazon内外の配信面を個別にON/OFF可能
- Amazon外配信は認知拡大目的で使用
- パフォーマンス重視の場合はAmazon内のみに限定推奨

## ターゲティング

### コンテキストターゲティング（商品ターゲティング）

特定の商品やカテゴリに関連する場所に広告を表示する。

| タイプ | 説明 | 用途 |
|--------|------|------|
| **個別ASIN** | 指定したASINの商品詳細ページに表示 | 競合の特定商品からの奪取 |
| **カテゴリ** | 指定カテゴリの商品群に表示 | カテゴリ全体でのリーチ拡大 |

**カテゴリターゲティングの絞り込み:**

- 価格帯（例: 自社商品より高い価格帯の商品を指定）
- ブランド（例: 特定の競合ブランドのみ）
- レビュー評価（例: 星3以下の低評価商品）
- プライム配送有無

### オーディエンスターゲティング

ユーザーの行動データに基づいて広告を表示する。

#### 閲覧リマーケティング

| 項目 | 詳細 |
|------|------|
| 対象 | 自社商品または類似商品を閲覧したユーザー |
| ルックバック期間 | 最大**30日間** |
| 用途 | 閲覧したが購入に至らなかったユーザーの再獲得 |
| 推奨設定 | 7日間/14日間/30日間で分けてテスト |

#### 購入リマーケティング

| 項目 | 詳細 |
|------|------|
| 対象 | 自社商品または類似商品を購入したユーザー |
| ルックバック期間 | 最大**365日間** |
| 用途 | リピート購入促進、クロスセル/アップセル |
| 推奨設定 | 商品の平均購入サイクルに合わせて期間設定 |

#### Amazonオーディエンス

Amazonの購買・閲覧データに基づく詳細なオーディエンスセグメント。

| セグメント | 説明 | 例 |
|------------|------|-----|
| **インマーケット** | 特定カテゴリで直近購買検討中のユーザー | 「プロテイン購入を検討中」 |
| **ライフスタイル** | 購買パターンから推定されるライフスタイル | 「健康志向」「アウトドア愛好家」 |
| **興味・関心** | 閲覧・検索パターンから推定される関心 | 「フィットネスに関心あり」 |
| **ライフイベント** | 人生の転機にあるユーザー | 「引っ越し予定」「出産準備中」 |

### 2025年新機能

| 機能 | 説明 |
|------|------|
| **最適化ターゲティング** | AIが自動で最適なオーディエンスを発見・拡張。手動設定に比べ平均20-30%のリーチ拡大 |
| **カスタムオーディエンス** | 複数のオーディエンスシグナルを組み合わせた独自セグメント作成 |
| **B2Bターゲティング** | ビジネス購買者（Amazon Business利用者）を特定してターゲティング |

## クリエイティブ仕様

### 基本要素

| 要素 | 仕様 |
|------|------|
| ブランドロゴ | **600 x 100px**、PNG/JPG、1MB以下 |
| カスタム画像（横長） | **1,200 x 628px**、PNG/JPG、5MB以下 |
| カスタム画像（縦長） | **628 x 1,200px**、PNG/JPG、5MB以下 |
| 見出し | **50文字以内**（全角） |
| カスタムテキスト | 商品特徴を伝える短文 |

### バナーサイズ一覧

SD広告は配信面に応じて自動リサイズされるが、以下のサイズで表示される。

| サイズ（px） | 名称 | 配信面 |
|-------------|------|--------|
| **300 x 250** | ミディアムレクタングル | 商品詳細ページ、サードパーティサイト |
| **728 x 90** | リーダーボード | 検索結果上部、サードパーティサイト |
| **160 x 600** | ワイドスカイスクレーパー | サイドバー |
| **300 x 600** | ハーフページ | サードパーティサイト |
| **320 x 50** | モバイルリーダーボード | モバイル |
| **970 x 250** | ビルボード | デスクトップ上部 |

### 画像のガイドライン

**推奨:**

- ライフスタイル画像（商品を使用しているシーン）
- 明るくクリアな画像
- ブランドカラーの活用
- シンプルな構図（要素を詰め込みすぎない）

**禁止事項:**

- テキスト過多の画像（画像面積の20%以下に抑える）
- Amazon商標/ロゴの使用
- 「ベストセラー」等の未承認バッジ
- 不鮮明・低解像度の画像
- 他ブランドの商標

## 自動生成 vs カスタムクリエイティブ

SD広告では自動生成クリエイティブとカスタムクリエイティブの2種類から選択可能。

### 比較表

| 項目 | 自動生成 | カスタム |
|------|----------|---------|
| 設定の手軽さ | ◎ 商品選択のみで自動作成 | △ 画像・見出し等の準備が必要 |
| CTR | 基準 | **15-25%高い** |
| ブランド表現 | 限定的 | ◎ ブランドの世界観を表現可能 |
| A/Bテスト | 不可 | ◎ 複数バリエーションでテスト可能 |
| 更新頻度 | 商品情報変更時に自動反映 | 手動で更新が必要 |
| 推奨場面 | テスト初期、リソース不足時 | 本格運用、ブランド訴求時 |

### カスタムクリエイティブの推奨構成

| 要素 | 推奨内容 |
|------|----------|
| 画像 | 使用シーンのライフスタイル写真 |
| 見出し | ベネフィット訴求（「毎日のスキンケアを時短」等） |
| ロゴ | 高解像度、ブランドカラー統一 |
| CTA | Amazon標準CTA（「今すぐチェック」） |

## 入札・最適化目標

SD広告では3つの最適化目標から選択する。目標によって課金方式と最適化対象が異なる。

### 3つの最適化目標

| 最適化目標 | 課金方式 | 説明 | 推奨場面 |
|------------|----------|------|----------|
| **リーチ** | vCPM | インプレッション最大化。最も多くのユーザーに表示 | ブランド認知拡大、新商品ローンチ |
| **ページ訪問** | CPC | クリック最大化。商品詳細ページへの誘導 | 検討促進、商品ページへのトラフィック増加 |
| **コンバージョン** | CPC | 購入に最も近いユーザーに最適化 | 売上最大化、リターゲティング |

### 最適化目標の選択基準

```
目的: ブランド認知拡大 → リーチ（vCPM）
目的: 商品ページ誘導 → ページ訪問（CPC）
目的: 売上・ROAS重視 → コンバージョン（CPC）
```

### 入札の考え方

| 最適化目標 | 入札の目安 | 注意点 |
|------------|-----------|--------|
| リーチ | vCPM 100-500円 | インプレッション単価。広範囲配信向け |
| ページ訪問 | CPC 5-30円 | SPより低めのCPCで運用可能 |
| コンバージョン | CPC 10-50円 | CVR高いセグメントに集中 |

## キャンペーン構成例

### 推奨5キャンペーン構成

| # | キャンペーン名 | ターゲティング | 最適化目標 | 目標ACoS | 目的 |
|---|----------------|----------------|-----------|----------|------|
| 1 | **SD-閲覧リマーケ** | 閲覧リマーケティング（7日） | コンバージョン | 15-20% | 閲覧者の再獲得 |
| 2 | **SD-購入リマーケ** | 購入リマーケティング（90日） | コンバージョン | 10-15% | リピート購入促進 |
| 3 | **SD-競合商品** | コンテキスト（競合ASIN） | ページ訪問 | 25-35% | 競合からの奪取 |
| 4 | **SD-カテゴリ** | コンテキスト（カテゴリ） | ページ訪問 | 30-40% | カテゴリ内リーチ拡大 |
| 5 | **SD-オーディエンス** | Amazonオーディエンス（インマーケット） | リーチ | 35-45% | 認知拡大・新規獲得 |

### キャンペーン別詳細設定

#### SD-閲覧リマーケ

```
ターゲティング: 閲覧リマーケティング
ルックバック: 7日間（高意図）+ 14日間（中意図）+ 30日間（広範囲）
最適化目標: コンバージョン
日予算: 1,000-3,000円
クリエイティブ: カスタム（商品画像+ベネフィット見出し）
配信面: Amazon内のみ推奨
```

#### SD-購入リマーケ

```
ターゲティング: 購入リマーケティング
ルックバック: 商品の購入サイクルに合わせる（例: 消耗品90日、耐久財365日）
最適化目標: コンバージョン
日予算: 500-2,000円
クリエイティブ: カスタム（新商品/関連商品のクロスセル訴求）
配信面: Amazon内のみ推奨
```

#### SD-競合商品

```
ターゲティング: コンテキスト（個別ASIN）
対象: 主要競合の上位5-10 ASIN
最適化目標: ページ訪問
日予算: 1,000-3,000円
クリエイティブ: カスタム（差別化ポイントを強調）
配信面: Amazon内のみ
```

#### SD-カテゴリ

```
ターゲティング: コンテキスト（カテゴリ）
絞り込み: 価格帯（自社より高い）、レビュー評価（自社より低い）
最適化目標: ページ訪問
日予算: 1,000-5,000円
クリエイティブ: カスタム（カテゴリ内での優位性訴求）
配信面: Amazon内 + 外
```

#### SD-オーディエンス

```
ターゲティング: Amazonオーディエンス（インマーケット + ライフスタイル）
最適化目標: リーチ
日予算: 2,000-5,000円
クリエイティブ: カスタム（ライフスタイル画像 + ブランド訴求）
配信面: Amazon内 + 外（Twitch、IMDb等）
```

### 予算配分の目安

| キャンペーン | 予算比率 | 理由 |
|-------------|----------|------|
| SD-閲覧リマーケ | 30% | 最もCVRが高い |
| SD-購入リマーケ | 15% | リピート率が高い場合に有効 |
| SD-競合商品 | 25% | 競合からの直接的な奪取 |
| SD-カテゴリ | 15% | リーチ拡大とデータ収集 |
| SD-オーディエンス | 15% | ファネル上部のカバー |
