# Amazon 戦略・設計スキル

Amazon出品の設計と広告運用の戦略立案から仕様書作成までを支援する。出品設計（リスティング最適化・クリエイティブ・フルフィルメント）と広告設計（SP/SB/SD/DSP）の両方をカバーする。

## ワークフロー

```
Step 1: 情報収集        → ビジネス概要・商品情報・目的・予算をヒアリング
Step 2: 戦略選択        → 出品設計 / 広告設計 / 統合設計を判定し、最適な手法を選定
Step 3: 設計・構成      → リスティング構成・キャンペーン構造・クリエイティブを設計
Step 4: 仕様書作成      → 出品仕様書・広告運用指示書・クリエイティブ仕様書を出力
```

---

## Step 1: 情報収集

`contexts/` ディレクトリに既存の情報があれば最初に確認する。

### 必須情報

| 項目 | 内容 |
|------|------|
| ビジネス概要 | 業種・事業モデル・自社の強み |
| 商品情報 | 商品カテゴリ・価格帯・差別化ポイント・JAN/EAN有無 |
| 目的 | 新規出品 / リスティング改善 / 広告開始 / 広告改善 / 統合設計 |
| 予算 | 月間広告予算・初期投資可能額 |
| ターゲット | 想定顧客層・購買動機 |
| KPI | 目標ACoS / ROAS / TACoS / 売上目標 |

### 状況把握（あれば確認）

| 項目 | 内容 |
|------|------|
| アカウント状態 | 大口/小口・ブランド登録有無・アカウント健全性 |
| 既存実績 | 現在の売上・ACoS・CVR・レビュー数 |
| クリエイティブ素材 | 既存の商品画像・動画・A+コンテンツ |
| FBA/FBM | 現在の配送方法 |
| 計測環境 | Amazon Attribution・Brand Analytics利用有無 |

## Step 2: 戦略選択

ユーザーの目的に応じて、以下の3つの戦略パスから選択する。

### A. 出品設計（リスティング最適化）

商品ページの構成・最適化・クリエイティブ設計を行うパス。

| 設計領域 | 詳細リファレンス | 読むタイミング |
|---------|----------------|--------------|
| アカウント・出品設定 | [account-setup.md](references/account-setup.md) | アカウント開設・ブランド登録・カテゴリ要件を確認する時 |
| 商品ページ設計 | [product-page-design.md](references/product-page-design.md) | タイトル・箇条書き・説明文・A+・バリエーションを設計する時 |
| リスティングクリエイティブ | [listing-creative.md](references/listing-creative.md) | 画像・動画・ストアフロントの仕様を確認する時 |
| フルフィルメント・価格 | [fulfillment-pricing.md](references/fulfillment-pricing.md) | FBA/FBM選択・価格戦略・手数料を設計する時 |

**出品設計の活用シーン：**
- 新規商品のAmazon出品を設計する
- 既存リスティングのCVR改善
- A+コンテンツ・ストアフロントの新規作成
- Amazon SEO（A10アルゴリズム）対策

### B. 広告設計（Amazon Advertising）

広告メニューの選定・キャンペーン構造・運用設計を行うパス。

| 広告タイプ | 特徴 | 前提条件 | 詳細リファレンス |
|-----------|------|---------|----------------|
| **スポンサープロダクト（SP）** | 商品単体を検索結果・商品ページに表示。売上直結型 | 大口出品 + カート獲得 | [sponsored-products.md](references/sponsored-products.md) |
| **スポンサーブランド（SB）** | ロゴ+見出し+複数商品。ブランド認知向上 | 大口出品 + ブランド登録 | [sponsored-brands.md](references/sponsored-brands.md) |
| **スポンサーディスプレイ（SD）** | Amazon内外にリターゲティング配信 | 大口出品 + ブランド登録 | [sponsored-display.md](references/sponsored-display.md) |
| **Amazon DSP** | プログラマティック広告。認知〜リターゲティング | Amazon直接 or 代理店 | [amazon-dsp.md](references/amazon-dsp.md) |

**広告戦略・KPI：**

| 設計領域 | 詳細リファレンス | 読むタイミング |
|---------|----------------|--------------|
| キャンペーン戦略 | [campaign-strategy.md](references/campaign-strategy.md) | フルファネル設計・予算配分・キーワード・シーズナル戦略を立てる時 |
| KPI・ベンチマーク | [kpi-benchmarks.md](references/kpi-benchmarks.md) | ACoS/ROAS/TACoS目標設定・損益分岐計算をする時 |

**ビジネス目標 → 推奨広告タイプ：**

| 目標 | 第1選択 | 第2選択 | 第3選択 |
|------|---------|---------|---------|
| 売上直結（既存商品） | SP（完全一致） | SD（リマーケ） | SB動画 |
| 新規顧客獲得 | SP（カテゴリKW） | SB（商品コレクション） | SD（オーディエンス） |
| ブランド認知向上 | SB（動画/ストア） | DSP（OLV/OTT） | SD（vCPM） |
| 競合からのシェア奪取 | SP（競合ASIN） | SD（商品ターゲ） | SB（カテゴリKW） |
| 新商品ローンチ | SP（オート+マニュアル） | SB動画 | Vine+A+ |
| リピート促進 | SD（購入リマーケ） | Brand Tailored | SP（ブランド防衛） |

**予算別推奨構成：**

| 月予算 | 推奨構成 | 配分 |
|--------|---------|------|
| 〜3万円 | SP のみ | SP 100% |
| 3〜10万円 | SP + SB | SP 70% / SB 30% |
| 10〜30万円 | SP + SB + SD | SP 60% / SB 25% / SD 15% |
| 30万円〜 | SP + SB + SD + (DSP検討) | SP 50% / SB 20% / SD 15% / DSP 15% |

### C. 統合設計

出品設計と広告設計を同時に行う。新規Amazon進出や大幅リニューアル時に推奨。
Step 2A → Step 2B の順で設計し、出品基盤を整えてから広告を構築する。

## Step 3: 設計・構成

### 出品設計の場合

1. **アカウント・基盤設計**: 出品プラン・ブランド登録・FBA/FBM の方針決定
2. **商品ページ設計**: タイトル構成式・箇条書き5項目・説明文・バックエンドKW の設計
3. **クリエイティブ設計**: メイン画像+サブ画像7枚の構成・A+コンテンツモジュール選定・動画仕様
4. **ストアフロント設計**: ページ構成・ナビゲーション・ヒーロー画像

### 広告設計の場合

1. **キャンペーン構造設計**: 目的別キャンペーン体系（ブランド防衛/カテゴリ/競合/発掘）
2. **ターゲティング設計**: KWマッチタイプ・商品ターゲティング・オーディエンスの設計
3. **入札・予算設計**: 入札戦略・日予算・プレースメント調整の設計
4. **クリエイティブ設計**: 広告フォーマット別のクリエイティブ・コピー仕様

### 命名規則

キャンペーン・広告グループの命名規則を統一する。

```
[商品/ブランド]_[広告タイプ]_[ターゲティング種別]_[マッチタイプ]_[日付]

例:
ProWidget_SP_Manual_Exact_BrandDefense_202602
ProWidget_SP_Auto_Discovery_202602
ProWidget_SB_Video_Category_202602
ProWidget_SD_Retarget_Views30d_202602
```

## Step 4: 仕様書作成

以下の成果物をMarkdown形式で出力する。対象パスに応じて必要なものを選択する。

### 出品仕様書（出品設計の場合）

1. **商品ページ設計書**
   - タイトル（キーワード配置の根拠付き）
   - 箇条書き5項目（構成: 見出しKW + 特徴 → メリット）
   - 商品説明文
   - バックエンドキーワード一覧
   - A+コンテンツ モジュール構成とコピー
   - バリエーション設計（該当する場合）

2. **クリエイティブ仕様書**
   - 画像構成表（メイン+サブ8枚の構図・内容・目的）
   - 動画仕様（構成・秒数・テキストオーバーレイ）
   - ストアフロント設計図

3. **運用設計書**
   - FBA/FBM方針
   - 価格戦略
   - レビュー獲得計画（Vine活用含む）
   - A10 SEO最適化チェックリスト

### 広告運用仕様書（広告設計の場合）

1. **戦略概要**
   - 目的・KPI・予算
   - ファネル設計と広告タイプの役割分担
   - 予算配分

2. **キャンペーン設計書**
   - キャンペーン一覧（名称・目的・ターゲティング・入札戦略・日予算）
   - 広告グループ構成
   - キーワードリスト（マッチタイプ別）
   - ネガティブキーワードリスト

3. **クリエイティブ仕様書**
   - 広告タイプ別のクリエイティブ仕様（画像サイズ・動画要件・コピー文字数）
   - 見出し・コピーの案

4. **運用・計測設計**
   - 週次運用チェックリスト
   - KPIモニタリング項目
   - 最適化ルール（入札調整・KW追加除外の基準）
   - シーズナル対応カレンダー

## プラットフォーム基本情報

| 項目 | 値 |
|------|-----|
| 対象マーケット | Amazon.co.jp（日本） |
| 出品プラン | 大口: 月額4,900円 / 小口: 100円/個 |
| ブランド登録要件 | 商標登録（出願中でも申請可能、完了まで6〜12ヶ月） |
| 広告課金方式 | CPC（SP/SB/SD）/ CPM（DSP/SD vCPM） |
| 日本市場 平均CPC | 5〜10円 |
| 日本市場 平均ACoS | 20〜30% |
| Amazon全体 平均CVR | 9.96%（他プラットフォームの約7〜8倍） |
| 主要セールイベント | 初売り(1月)・Prime Day(7月)・ブラックフライデー(11月)・年末(12月) |
