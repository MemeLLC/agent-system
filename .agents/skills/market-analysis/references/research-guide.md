## 1. 市場規模・成長率の把握

**調べること:**
- 市場規模（金額ベース：売上高合計）
- 市場規模（数量ベース：出荷数、利用者数など）
- 過去5年の成長率の推移
- 今後3〜5年の市場規模予測
- 地域別・セグメント別の内訳

**クエリ例:**
- `"[業種] 市場規模 2025"`
- `"[業種] 市場規模 予測 推移"`
- `"[業種] market size forecast"`
- `site:meti.go.jp [業種]` / `site:stat.go.jp [業種]`
- `filetype:pdf [業種] 市場調査 2025`

**情報源の優先順:**
- 官公庁の統計（経済産業省、総務省統計局）
- 調査会社のレポートサマリー（矢野経済研究所、富士経済、Statista、Grand View Research）
- 業界団体の公表データ
- 上場企業のIR資料（有価証券報告書、決算説明資料）

**注意:**
- 最低3つの異なるソースで市場規模のデータを突き合わせること
- ソース間で数値が異なる場合は、定義の違い（対象範囲、含まれるセグメント）を確認する
- 「市場規模」の定義（出荷額 vs 小売額 vs 消費者支出）に注意する

## 2. 市場構造の理解

**調べること:**
- 主要プレイヤー（上位3〜5社）とそのシェア
- 流通チャネル（直販、代理店、オンライン、小売など）
- 参入障壁の有無と程度（資金、技術、許認可、ネットワーク効果）
- 市場の成熟度（黎明期 / 成長期 / 成熟期 / 衰退期）
- 市場の集中度（寡占 vs 分散）

**クエリ例:**
- `"[業種] シェア ランキング 2025"`
- `"[業種] 業界地図"`
- `"[業種] 参入障壁"`
- `"[業種] 流通チャネル 販売経路"`
- `"[業種] カオスマップ 2025"`
