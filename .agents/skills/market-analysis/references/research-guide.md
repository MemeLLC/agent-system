## 1. 市場規模・成長率の把握

**調べること:**
- 市場規模（金額ベース：売上高合計）
- 市場規模（数量ベース：出荷数、利用者数など）
- 過去5年の成長率の推移
- 今後3〜5年の市場規模予測
- 地域別・セグメント別の内訳

**クエリ例:**
- `"[業種] 市場規模 2025"`
- `"[業種] 市場規模 予測 推移"`
- `"[業種] market size forecast"`
- `site:meti.go.jp [業種]` / `site:stat.go.jp [業種]`
- `filetype:pdf [業種] 市場調査 2025`

**情報源の優先順:**
- 官公庁の統計（経済産業省、総務省統計局）
- 調査会社のレポートサマリー（矢野経済研究所、富士経済、Statista、Grand View Research）
- 業界団体の公表データ
- 上場企業のIR資料（有価証券報告書、決算説明資料）

**注意:**
- 最低3つの異なるソースで市場規模のデータを突き合わせること
- ソース間で数値が異なる場合は、定義の違い（対象範囲、含まれるセグメント）を確認する
- 「市場規模」の定義（出荷額 vs 小売額 vs 消費者支出）に注意する

## 2. 市場構造の理解

**調べること:**
- 主要プレイヤー（上位3〜5社）とそのシェア
- 流通チャネル（直販、代理店、オンライン、小売など）
- 参入障壁の有無と程度（資金、技術、許認可、ネットワーク効果）
- 市場の成熟度（黎明期 / 成長期 / 成熟期 / 衰退期）
- 市場の集中度（寡占 vs 分散）

**クエリ例:**
- `"[業種] シェア ランキング 2025"`
- `"[業種] 業界地図"`
- `"[業種] 参入障壁"`
- `"[業種] 流通チャネル 販売経路"`
- `"[業種] カオスマップ 2025"`

## 3. 価格調査

**調べること:**
- 市場全体の価格帯（ローエンド〜ハイエンド）
- 採用されている料金モデル（月額/年額、従量課金、フリーミアム、買い切りなど）
- 価格弾力性・顧客の支払い意思
- 市場で成功している価格戦略のパターン

**クエリ例:**
- `"[商品/サービス] 価格 相場"`
- `"[商品/サービス] 料金プラン 一覧"` / `"[商品/サービス] 料金 比較"`
- `"[業種] 料金体系 種類"`
- `"[業種] 顧客 予算"`
- `"[業種] 価格戦略 成功事例"`

**注意:**
- 最低3社以上の料金ページをWebFetchで確認すること
- 料金モデルの種類だけでなく、具体的な金額も記録する

## 4. 顧客調査

### 4-1. 属性データ（定量）

**調べること:**
- 年齢層（主要な年齢帯と構成比）
- 性別構成比
- 職種・職業（会社員、自営業、フリーランス、学生、主婦など）
- 年収・世帯年収
- 居住地域（都市部/地方、地域別の偏り）
- 学歴
- 世帯構成（単身、夫婦、子育て世帯など）
- 利用者数・ユーザー数の推移

**クエリ例:**
- `"[業種/サービス] ユーザー 属性 調査"`
- `"[業種/サービス] 利用者 年齢 性別 年収"`
- `"[業種/サービス] 利用者数 推移"`
- `"[業種/サービス] 消費者 アンケート 調査"`
- `site:stat.go.jp [業種] 利用動向`

### 4-2. 行動・心理（定性）

**調べること:**
- 購買動機（なぜ買うのか、何がきっかけか）
- 抱えている課題・ペイン
- 情報収集行動（どこで知るか：SNS、口コミ、検索、広告、店頭など）
- 重視する選定基準（価格、品質、ブランド、口コミ、利便性など）
- 購買頻度・タイミング（季節性、ライフイベント連動）

**クエリ例:**
- `"[業種/サービス] 購入 きっかけ 理由"`
- `"[業種/サービス] 選び方 比較 ポイント"`
- `"[業種/サービス] 口コミ 評判 不満"`
- `"[業種/サービス] 購買行動 調査"`

**情報源の優先順:**
- 官公庁の統計・白書（総務省統計局、消費者庁、各省庁の実態調査）
- 調査会社のアンケート結果（マクロミル、MMD研究所、ICT総研など）
- 業界団体の利用動向レポート
- 既存サービスの公式発表（ユーザー数、利用者層に関するプレスリリース）
- レビューサイト・口コミサイト（実際のユーザーの声）

**注意:**
- 定量データ（数値・割合）を優先して収集する。定性的な情報はそれを補足する形で記載する
- 調査の母集団・サンプルサイズ・調査時期を可能な限り記録する
