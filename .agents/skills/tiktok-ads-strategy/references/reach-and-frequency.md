# Reach & Frequency（リーチ＆フリークエンシー）

## 概要

Reach & Frequency（R&F）は、TikTok Ads Managerで利用可能な予約購入型キャンペーン方式。リーチ数とフリークエンシー（接触頻度）を事前にコントロールし、**固定CPMで予測可能な広告配信**を実現する。

- **購入方式**: 予約型。認定顧客のみ利用可能（TikTok営業担当への問い合わせが必要）
- **費用感**: 固定CPM（予算規模に依存）。早期予約でCPMを抑制可能。コスト効率重視のパターンでは$30,000〜
- **主な差別化ポイント**: 配信前にリーチ・フリークエンシーのシミュレーションが可能。±5%の精度で予測通りの配信を実現
- **対応キャンペーン目的**: リーチ（Reach）、動画視聴（Video Views）、トラフィック（Traffic）
- **対応クリエイティブ**: ビデオ広告（In-Feed仕様）、Spark Ads、Display Card（Interactive Add-on）
- **フィードタイプ**: Standard Feed または Top Feed を選択可能

**主要パフォーマンスデータ**:
- 最低10本のユニーククリエイティブで**1.3倍アド想起**、**3.0倍購買意向**、**1.5倍認知度**
- インフィード + プレミアムフォーマットの組み合わせで**+7.5%アド想起**、**1.4倍認知度**、**3.0倍購買意向**
- CPGブランドのSpark Adsは他チャネルと比較して**96%高い有料メディアROAS**
- 3パート動画シーケンスで**89%のキーメッセージ記憶向上**

## 仕様

### 動画仕様

| 項目 | 仕様 |
|------|------|
| アスペクト比 | 9:16（推奨）、16:9 |
| 解像度（縦型） | 540 x 960 px以上 |
| 解像度（横型） | 960 x 540 px以上 |
| 動画尺 | 制限なし（推奨15〜60秒） |
| ファイル形式 | .mp4, .mov |
| ファイルサイズ | 最大500 MB |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| キャプション文字数 | 100文字推奨（Spark Adsは制限なし） |
| 空欄可否 | Reach / Video View / R&F目標では空欄可 |
| 絵文字 | emoji v.11を使用 |

### 対応広告フォーマット

| フォーマット | 説明 |
|-------------|------|
| ビデオ広告 | 標準のインフィード動画広告 |
| Spark Ads | オーガニック投稿を広告として配信 |
| Display Card | 動画下部にカード型CTAを表示 |

### 購入ルール

| 項目 | 詳細 |
|------|------|
| 購入タイプ | 予約購入（認定顧客のみ） |
| CPM | 固定（事前確定） |
| 最低リーチ | 100,000以上 |
| 事前予約 | 最大225日前から可能 |
| 配信精度 | ±5%以内 |
| 最大クリエイティブ数 | 1広告グループあたり20本 |

### Standard Feed vs Top Feed

| 項目 | Standard Feed | Top Feed |
|------|--------------|----------|
| 広告枠位置 | セッション中の任意の広告枠 | セッション中の最初の広告枠 |
| ユーザー注目度 | 標準 | プレミアム（高い） |
| 用途 | リーチ拡大・コスト効率 | ブランド認知・インパクト重視 |
| CPM | 相対的に低い | 相対的に高い |

## クリエイティブ制作ガイド

### クリエイティブ数と多様性

- **最低10本のユニーククリエイティブ推奨**（北米データ）:
  - **1.3倍** アド想起向上
  - **3.0倍** 購買意向向上
  - **1.5倍** 認知度向上
- **クリエイティブ疲れ防止**: 高いクリエイティブ数でリフレッシュ頻度を高め、疲れを軽減

### 広告フォーマットミックス

- **インフィード動画 + プレミアムフォーマットの組み合わせ**が最強:
  - **+7.5%** アド想起
  - **1.4倍** 認知度
  - **3.0倍** 購買意向
- ビデオ広告、Spark Ads、Display Cardを組み合わせて配信

### Spark Ads活用

- **オーガニック風の生動画**が最もパフォーマンスが高い
- **38秒以上の動画**が短い動画を上回る傾向
- CPGブランドのSpark Adsは他チャネルと比較して**96%高い有料メディアROAS**
- トレンドエフェクト・編集スタイル・ストーリーラインの活用がパフォーマンスドライバー

### アドシーケンシング（広告連続配信）

R&Fの最大の強みの1つが、**Advanced Ads Delivery機能**による広告の表示順序制御。

**3つの配信モード**:

| モード | 説明 | 用途 |
|--------|------|------|
| Standard delivery | 均等配信。すべての広告が同程度のトラフィックを獲得 | 通常のリーチキャンペーン |
| Scheduled delivery | 特定の時間帯に特定の広告を配信 | 時間帯別の最適化 |
| Sequenced delivery | 指定した順序で広告を配信 | ストーリーテリングに最適 |

- **3パート動画シーケンスで89%のキーメッセージ記憶向上**（Ipsos調査）
- 最大20本のクリエイティブを1つの広告グループに設定可能

**ストーリーテリング活用例**（スキンケアブランド）:
- パート1: クレンザー紹介
- パート2: セラム紹介
- パート3: モイスチャライザー紹介
- ユーザーをフルスキンケアジャーニーにガイド

### やるべきこと

- 最低10本のユニーククリエイティブを用意
- インフィード動画とプレミアムフォーマットを組み合わせる
- Spark Adsでオーガニック風コンテンツを活用
- アドシーケンシングでブランドストーリーを順序立てて配信
- 定期的にクリエイティブをリフレッシュして疲れを防止
- トレンドエフェクト・編集スタイルを積極的に取り入れる

### やってはいけないこと

- クリエイティブ数が少ない状態で長期運用（疲れが発生）
- シーケンス配信でストーリーの一貫性がないクリエイティブの使用
- 単一フォーマットのみでの配信

## 広告設定ガイド

### 推奨キャンペーン目的

- **Reach（リーチ）**: 最大数のユーザーに広告を表示
- **Video Views（動画視聴）**: 動画の再生回数・エンゲージメントを最大化
- **Traffic（トラフィック）**: ウェブサイト/アプリへの流入を促進

### ターゲティングアプローチ

- **ブロードオーディエンス推奨**: 配信フリークエンシーが設定値に近づきやすい
- **最低リーチ閾値**: 100,000リーチ以上
- **避けるべき設定**:
  - 非常に狭いオーディエンス
  - 短期間（特に単日）キャンペーン
  - リーチ率80%以上

### フリークエンシー最適化

- **最適頻度**: 週2〜3回
- **Target Frequency機能**: 特定のフリークエンシーをターゲットとして設定可能
- **3週間 x 週3回 = 合計9回のタッチポイント**が理想

### 予約方式（4つから選択）

| 方式 | 説明 | 最適な場面 |
|------|------|----------|
| Fixed Budget（固定予算） | 特定予算を全額使い切る | 予算確定済みのキャンペーン |
| Fixed Impressions（固定インプレッション） | 特定インプレッション数を達成 | インプレッション目標がある場合 |
| Fixed Reach（固定リーチ） | 特定リーチ数を達成 | リーチ数がKPIの場合 |
| Fixed Reach Percentage（固定リーチ率） | 特定リーチ率を達成 | リーチ率がKPIの場合 |

### シーケンシャル配信の設定

- **Sequenced delivery**を選択して広告の表示順序を制御
- ブランドストーリーを段階的に展開するクリエイティブを用意
- 最大20本のクリエイティブを1広告グループに設定

### キャンペーン計画ツール

- **Campaign Forecaster**: 予算・CPM・推定リーチ・1人あたりフリークエンシー・日別支出を比較
- **早期予約**: 最大225日前から予約可能。早期予約でCPMを低く抑える

### キャンペーン期間

- **最低3週間推奨**: ブランド認知向上に必要な最低期間
- 単日キャンペーンは避ける

## コピーライティングガイド

### キャプション

- **R&Fキャンペーンではキャプション空欄可能**: Reach、Video View、R&F（Beta）目標ではキャプションを空白にできる
- **Spark Adsの場合**: キャプションはオーガニック動画のキャプションがそのまま使用される（文字数制限なし）
- **通常の広告の場合**: 100文字以内を推奨

### テキストオーバーレイ戦略

- **音声あり・なし両方に対応**: USPをオンスクリーンテキストオーバーレイで表示し、音声なしユーザーにも訴求
- **表示速度**: 1秒あたり5〜10語
- **セーフゾーン内配置**: テキストはCTA・ボディコピー・エンゲージメントボタンと重複させない

### CTA設定

- **CTA制限事項**: スポンサードハッシュタグを宣伝する場合、「このハッシュタグに参加」CTAは利用不可
- **効果的なCTA例**:
  - 「今すぐチェック」
  - 「詳しくはプロフィールから」
  - 「無料で体験」
  - 「限定オファーを見る」

### シーケンシャル配信のコピー戦略

シーケンス配信では、各パートで異なる角度のバリュープロポジションをテスト:

| パート | コピーの役割 |
|--------|------------|
| パート1 | 問題提起・好奇心喚起のコピー |
| パート2 | 解決策・ベネフィット提示のコピー |
| パート3 | CTA・限定感・行動促進のコピー |

### コピーバリエーション

- Spark Adsではオーガニックキャプションの品質を事前確認
- 複数のCTAバリエーションをテスト
