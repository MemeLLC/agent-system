# Messaging Ads（メッセージ広告）

## 概要

広告から直接メッセージングアプリ（TikTokダイレクトメッセージなど）での会話を開始できるフォーマット。ユーザーとの1対1のコミュニケーションを促進し、リード獲得やトラフィック目的で使用する。

- **購入方式**: オークション型（運用型）
- **課金方式**: oCPM, CPC
- **対応キャンペーン目的**: リード獲得（Lead Generation）、トラフィック（Traffic）
- **主な差別化ポイント**: DM内での1対1コミュニケーション、ウェルカムメッセージテンプレート、ポップアップフォームカード対応
- **最適な業種**: 美容、EC、サービス業
- **技術的難易度**: 低〜中
- **地域制限**: **Direct Messaging Adsは東南アジア中心**（インドネシア、シンガポール、マレーシア、フィリピン、タイ、ベトナム）。Instant Messaging Adsはより広い地域で利用可能

### パフォーマンス実績

- 美容ブランドの事例: Direct Messaging Adsで**コンバージョン率22.5%**を達成
- 1週間で**900以上の会話**を生成
- クリック最適化と比較して**会話単価が67%低下**
- 3分以内の返信で**会話エンゲージメント率が最大2.2倍向上**

## クリエイティブ制作ガイド

### 動画クリエイティブの原則

- **会話への誘導を意識**: 動画の目的は「メッセージを送りたい」と思わせること
- **パーソナルなトーン**: 1対1のコミュニケーションに繋がる広告なので、親密感のあるクリエイティブが効果的
- **実在の人物を起用**: スタッフや代表者が直接語りかける形式が高いエンゲージメントを獲得
- **商品・サービスの魅力を短く伝える**: 詳細は会話で伝えられるため、興味喚起に集中

### 効果の高いクリエイティブパターン

| パターン | 説明 | 効果 |
|----------|------|------|
| **商品ベネフィット紹介** | 商品の主要メリットを簡潔に紹介 | 興味喚起 |
| **テスティモニアル** | 実際の顧客の声・レビュー | 信頼構築 |
| **プロモーション告知** | 限定オファーや割引情報 | 緊急性・行動促進 |
| **デモ・How-to** | 使い方や効果を実演 | 理解促進 |
| **質問形式** | 「あなたに合う○○を知りたくないですか？」 | 会話開始の動機付け |

### クリエイティブと会話の一貫性（ポリシー要件）

- 動画で紹介した商品・サービス・オファーと、DM内の会話内容が一致していること
- 動画内のブランドとDMで対応するブランドが同一であること
- 動画で言及した価格・セール情報はDM内でも同じ内容を提供

### やるべきこと

- パーソナルで親密感のあるトーンで制作する
- 実在のスタッフや代表者が語りかける形式を採用する
- 会話でさらに詳しく聞きたくなるような興味喚起に集中する
- 動画とDM会話の内容を一致させる

### やってはいけないこと

- 動画内の情報とDM会話の内容に矛盾がある
- 過度に営業的・押し売り的なトーン
- ブランドのなりすましやTikTokブランド要素の無断使用
- DM内でセンシティブな個人情報（クレジットカード、社会保障番号など）を求める

## 広告設定ガイド

### Direct Messaging Ads設定

**前提条件:**
- TikTokビジネスアカウントが必要
- DM権限を「全員」に設定
- ビジネスプライバシーポリシーの生成（会社名、住所、メールアドレスが必要）

**キャンペーン設定:**

| 設定項目 | 推奨値 |
|----------|--------|
| キャンペーン目的 | Lead Generation（リード獲得） |
| 最適化目標 | Conversations（会話） |
| 課金方式 | oCPM または CPC |
| 最低日予算（キャンペーン） | $50/日 |
| 最低日予算（広告グループ） | $20/日 |

### ウェルカムメッセージテンプレート設定

- **挨拶テキスト**: 最大200文字。自社の紹介や商品説明を記載
- **提案質問**: 最大3つ（各70文字以内）。ユーザーがワンタップで会話を開始できる質問を設計
- **ポップアップフォームカード**: DM内でリード情報を収集するフォーム（チャット外に遷移不要）

### 自動チャット機能

- スクリプト化された会話フローでリード選別を自動化
- 営業時間内の広告配信と自動応答ツールの組み合わせが効果的
- 3分以内の返信で**会話エンゲージメント率が最大2.2倍向上**

### Instant Messaging Ads設定

**WhatsApp連携:**
- WhatsApp Business API番号を設定
- 事前入力メッセージ: 「こんにちは！TikTokの広告を見て、詳しく知りたくなりました」
- チャットが自動的に開く

**対応プラットフォーム:**
- Facebook Messenger
- WhatsApp
- Zalo
- カスタムURL

### ターゲティング戦略

- **年齢層**: 商品に応じて設定（例: 美容ブランドは25〜55歳以上）
- **興味関心**: 自社商品カテゴリに関連するセグメント
- **営業時間に合わせた配信**: 迅速な返信が可能な時間帯に集中配信
- **リターゲティング**: 過去に動画を視聴・エンゲージしたユーザー

### メッセージングパートナーの活用

- TikTok DM、WhatsApp、Messengerからのメッセージを一元管理
- 会話ラベリング、フィルタリング、自動返信、注文管理などの高度な機能
- 推奨パートナー: respond.io、SleekFlow、Omnichatなど

## コピーライティングガイド

### 動画キャプション

- **ソフトなCTA**: 「気になったらメッセージしてね」「DMでお気軽にご相談ください」
- **具体的な行動指示**: 「メッセージを送る」「今すぐ問い合わせ」をキャプションやクリエイティブに一貫して使用
- **ベネフィットの明示**: 「DMで限定クーポンをお送りします」「個別に最適なプランをご提案」
- キャプションは最大100文字（最初の80〜100文字が表示）

### ウェルカムメッセージのコピー

**挨拶文（最大200文字）の例:**
```
こんにちは！○○へようこそ。
お気軽にご質問ください。あなたにぴったりの商品をご提案いたします。
```

**提案質問の例（最大3つ、各70文字以内）:**

| 業種 | 質問例 |
|------|--------|
| **美容** | 「おすすめのスキンケアを教えて」「今のキャンペーンは？」「肌悩みを相談したい」 |
| **教育** | 「コース内容を知りたい」「無料体験はある？」「料金プランを教えて」 |
| **不動産** | 「物件の詳細を教えて」「見学予約をしたい」「おすすめエリアは？」 |
| **EC** | 「在庫状況を確認したい」「サイズの相談がしたい」「配送について教えて」 |

### DM会話のコピー原則

- **迅速な応答**: 3分以内の返信を目標に（エンゲージメント率2.2倍向上）
- **パーソナライズ**: ユーザーの質問に合わせたカスタム応答
- **次のステップを明示**: 会話を次のアクション（購入、予約、申込）に導く
- **センシティブ情報の回避**: DM内で個人の機密情報を求めない（TikTokポリシー）

### スパム対策

- スパムメッセージや不審なユーザーはブロック・スパム報告
- TikTokのアルゴリズムが低意図・有害なインタラクションをフィルタリング
- オーディエンス品質の向上と配信最適化に寄与
