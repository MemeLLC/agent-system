# Dynamic Showcase Ads / DSA（ダイナミックショーケース広告）

## 概要

商品カタログに基づいて自動生成されるパーソナライズド動画広告。閲覧履歴や興味関心に基づいて、ユーザーごとに最適な商品を動的に表示する。カタログベースのリターゲティングに特化したフォーマット。

- **購入方式**: オークション型（運用型）
- **課金方式**: oCPM
- **対応キャンペーン目的**: 販売（Product Sales）> カタログ販売（Catalog Sales）
- **主な差別化ポイント**: カタログベースの自動パーソナライズ配信、リターゲティング特化
- **パフォーマンス実績**:
  - 商品レベルの動画活用でエンゲージメントが3〜5倍向上
  - セール・割引情報を含むDSAはROASが48%向上

> **重要: DSAの新規作成は不可**
> DSAの機能はVideo Shopping Ads（VSA）に統合済み。新規作成はできない。Product Sales目的のCatalog Ads内でカタログベースのリターゲティング機能として引き続き利用可能。
>
> **移行先の流れ**: DSA → Video Shopping Ads（VSA）→ GMV Max（2025年7月〜）
>
> 新規キャンペーンはVSAまたはGMV Maxで作成することを推奨。

## 仕様

### 動画テンプレート仕様

| 項目 | 仕様 |
|------|------|
| 動画の長さ | 5〜16秒 |
| アスペクト比 | 9:16（縦型）、1:1（正方形）、16:9（横型） |
| 推奨解像度 | 720 x 1280 px（縦型）/ 640 x 640 px（正方形）/ 1280 x 720 px（横型） |
| ファイル形式 | .mp4、.mov、.mpeg、.avi |
| 最大ファイルサイズ | 500MB |
| テンプレート機能 | エフェクト・BGM・スティッカーの自動生成 |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| 広告キャプション | 英語: 最大100文字 / 日本語等: 最大50文字 |
| 商品タイトル（フィード） | 最大150文字 |
| 商品説明（フィード） | 最大20,000文字 |

### カタログ（商品フィード）仕様

| 項目 | 仕様 |
|------|------|
| 必須フィールド | 9項目（商品ID、タイトル、説明、URL、画像URL、価格、在庫状況など） |
| オプションフィールド | 27項目 |
| フィード形式 | CSV、XML |
| 更新頻度 | 毎日更新を推奨 |
| 画像リンク | 1商品あたり最大10画像 |
| 動画リンク | 対応 |
| 最小画像サイズ | 500 x 500 px |

### ピクセル要件

| イベント | 必須/推奨 | 用途 |
|---------|----------|------|
| ViewContent | 必須 | 商品ページ閲覧のトラッキング |
| AddToCart | 必須 | カート追加のトラッキング |
| Purchase | 必須 | 購入完了のトラッキング |
| CompletePayment | 推奨 | 支払い完了のトラッキング |

### 配信ルール

| 項目 | ルール |
|------|--------|
| 配信面 | TikTok、Pangle |
| キャンペーン目的 | Product Sales > Catalog Sales |
| ターゲティング | リターゲティング、プロスペクティング |
| 最低キャンペーン費用 | $500/キャンペーン |

### 現在のステータス

| 項目 | 状況 |
|------|------|
| 新規作成 | 不可（VSAに統合済み） |
| 既存キャンペーン | Product Sales目的のCatalog Ads内で引き続き利用可能 |
| カタログベースリターゲティング | 引き続き利用可能 |
| 統合先 | Video Shopping Ads（VSA）→ GMV Max（2025年7月〜） |
| 推奨アクション | 新規キャンペーンはVSAまたはGMV Maxで作成 |

## クリエイティブ制作ガイド

### 基本原則

- **テンプレートベースの自動生成**: 商品情報（画像・名前・価格）をテンプレートに流し込み、パーソナライズされた動画を自動生成
- **商品レベルの動画活用**: 商品フィードに動画リンクを含めると、エンゲージメントが3〜5倍向上
- **セール情報の組み込み**: セール・割引情報を含むDSAはROASが48%向上

### テンプレートデザインのコツ

| ポイント | 説明 |
|---------|------|
| シンプルなレイアウト | 商品画像が映えるクリーンなデザイン |
| ブランド要素 | ロゴ・ブランドカラーを統一的に配置 |
| 価格表示 | 価格・割引率を目立つ位置に配置 |
| 動的テキスト | 商品名・価格などフィードから自動挿入 |
| BGM/効果音 | テンプレート内でBGM・スティッカー・エフェクトを自動生成 |

### カタログ画像の最適化

- **高品質な商品画像**: 最低500 x 500 px（推奨720px以上）
- **白背景推奨**: 商品が際立つクリーンな背景
- **複数アングル**: 1商品あたり最大10枚の画像を登録可能
- **動画の追加**: 可能であれば商品レベルの動画も登録（エンゲージメント3〜5倍向上）

### クリエイティブチェックリスト

- テンプレートが5〜16秒の長さに収まっているか
- 商品画像が高解像度で鮮明か
- 価格・割引情報が含まれているか
- ブランドの統一感があるか
- 動的テキスト（商品名・価格）が正しく表示されるか

### やるべきこと

- 商品フィードに動画リンクを含める（エンゲージメント大幅向上）
- セール・割引情報を積極的に組み込む（ROAS 48%向上）
- シンプルで商品が映えるテンプレートデザインにする
- 商品画像は720px以上の高品質なものを使用する

### やってはいけないこと

- 低解像度の商品画像を使用しない（500px未満）
- 動的テキストの表示確認を怠らない
- カタログの更新を怠らない（毎日更新推奨）
- ピクセルの設定不備を放置しない

## 広告設定ガイド

### キャンペーン目的

- **Product Sales（商品販売）** > **Catalog Sales（カタログ販売）** で設定

### 前提条件（5ステップ）

1. **商品カタログのアップロード**: TikTok Ads Managerにカタログを作成・商品を登録
2. **TikTokピクセルの設置**: ViewContent、AddToCart、Purchaseの3つのイベントが必須
3. **イベントソースの接続**: カタログとピクセルイベントを紐付け
4. **動画テンプレートの作成**: 商品情報を流し込むテンプレートを設計
5. **Catalog Salesキャンペーンの作成**: 上記を組み合わせてキャンペーンを設定

### ターゲティングオプション

| ターゲティング | 説明 | 推奨ケース |
|--------------|------|-----------|
| リターゲティング | 商品閲覧・カート追加済みだが未購入のユーザーに再表示 | CVR向上、カート放棄対策 |
| プロスペクティング（DABA） | 新規ユーザーに広くリーチ | 新規顧客獲得、認知拡大 |

### リターゲティングイベント

| イベント | 用途 |
|---------|------|
| ViewContent | 商品ページを閲覧したユーザー |
| AddToCart | カートに追加したが購入していないユーザー |
| Purchase | 過去の購入者（クロスセル・アップセル） |

### ピクセル設定の重要性

- ピクセルの精度がDSAのパフォーマンスを左右する
- 不完全・破損したイベント設定は広告配信とROASに悪影響
- ピクセルが正しくインストール・検証されていることを必ず確認
- イベントデータに基づき、TikTokがコンバージョンしやすいユーザーに最適な商品を配信

### 予算

| 項目 | 仕様 |
|------|------|
| 最低キャンペーン費用 | $500/キャンペーン |
| コスト変動要因 | ターゲティング精度、広告品質、競合状況 |

### オーディエンス設定

- 興味関心、位置情報、行動データ、年齢、性別でターゲティング
- ウェブサイトのトラッキングデータに基づくパーソナライズ配信
- アルゴリズムがユーザーのニーズに合った商品を自動選択・配信

## コピーライティングガイド

### コピーライティングの原則

- **汎用メッセージから商品固有のストーリーへ**: DSAでは幅広い汎用コピーではなく、商品カテゴリや個別商品に合わせた具体的なメッセージが効果的
- **動的テキストの活用**: テンプレート内で商品名・価格・割引率を動的に挿入
- **リターゲティング用コピー**: 「お探しの商品がまだありますよ」「カートに入れた商品が値下げ中」など、再訪を促すメッセージ
- **プロスペクティング用コピー**: 「あなたにぴったりの〇〇」「人気商品をチェック」など、新規ユーザー向けの興味喚起

### 商品フィードのコピー最適化

- **商品タイトル**: 簡潔で検索性の高いタイトルを設定。重要な情報を先頭に配置（最大150文字）
- **商品説明**: 特徴・素材・サイズ・使用シーンなど詳細情報を記載（最大20,000文字）
- **価格情報**: 正確な価格・セール価格を常に最新の状態に維持
- **在庫状況**: 在庫切れ商品は自動的に広告から除外されるよう、正確な在庫情報を維持

### ターゲティング別コピー戦略

| ターゲティング | コピーの方向性 | 例 |
|--------------|--------------|-----|
| リターゲティング | 再訪・購入完了を促す | 「お探しの商品がまだありますよ」「カートに入れた商品が値下げ中」 |
| プロスペクティング | 興味喚起・新規発見 | 「あなたにぴったりの〇〇」「人気商品をチェック」 |
| クロスセル | 関連商品の提案 | 「〇〇を購入された方におすすめ」 |
