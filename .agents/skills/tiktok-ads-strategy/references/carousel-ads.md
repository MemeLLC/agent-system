# Carousel Ads（カルーセル広告）

## 概要

複数の画像をスワイプで表示するインフィード広告フォーマット。ユーザーが手動でコンテンツを閲覧し、CTAボタンからコンバージョンへ誘導する。ストーリーテリング、商品カタログ表示、プロモーション訴求に最適。

- **購入方式**: オークション型（運用型）。日予算2万〜10万円程度からスモールスタート可能
- **課金方式**: CPC, oCPM
- **対応キャンペーン目的**: トラフィック、ウェブコンバージョン、アプリプロモーション、リーチ
- **主な差別化ポイント**: 静止画ベースで動画制作不要、複数商品/特徴を1広告で訴求可能
- **パフォーマンス実績**:
  - 単一画像広告比でCTRが20〜30%向上
  - ストーリー構成のカルーセルで視聴時間が最大18%延長
  - ECキャンペーンでCPCが15〜25%低下

## 仕様

### 画像仕様

| 項目 | 仕様 |
|------|------|
| 画像枚数 | 2〜35枚（広告用は2〜10枚推奨、最低3枚以上を推奨） |
| 画像形式 | JPG / JPEG / PNG |
| ファイルサイズ | 1画像あたり100KB推奨（最大500KB）、全画像合計500KB以下推奨 |
| アスペクト比 | 9:16（推奨）、1:1、16:9 |
| 推奨解像度（縦型） | 720 x 1280 px以上（9:16） |
| 推奨解像度（正方形） | 640 x 640 px以上（1:1） |
| 推奨解像度（横型） | 1200 x 628 px以上 |
| 最小解像度 | 720px以上 |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| 広告キャプション | 最大20文字（1行のみ）。半角1文字、全角2文字としてカウント |
| 広告タイトル | 12〜40文字（日本語は約20文字） |
| ブランド/アプリ名 | 2〜20文字 |
| 絵文字 | 使用不可（アプリ名・ブランド名にも使用不可） |
| 特殊文字 | 「{}」「#」は説明文に表示されない |
| プロフィール画像 | 98 x 98 px、50KB以下 |

### 音楽（BGM）仕様 ※必須

| 項目 | 仕様 |
|------|------|
| 音楽設定 | 必須（BGMなしでは出稿不可） |
| 音楽ソース | CML（Commercial Music Library）またはアップロード |
| 対応フォーマット | .mp3、.wav、.m4a、.flac |
| 最小再生時間 | 2秒 |
| 最大ファイルサイズ | 約10MB |
| ループ再生 | カルーセル全体で連続ループ |

### 配信面

- TikTok For Youページ（インフィード）のみ。ストーリーズや検索結果には非対応
- TikTok Pangle: CML音楽を選択した場合、Pangleでは無音になる。Pangle配信のみの場合はアップロード音楽を使用

## クリエイティブ制作ガイド

### 基本原則

- **最初の1枚で勝負する**: 最初の画像がフック（引き止め）の役割を果たす。視覚的にインパクトのある画像を配置し、スワイプを促す
- **ストーリーテリング構成**: 各画像が連続したストーリーを伝えるようにデザインする。ストーリー順に画像を並べると視聴時間が最大18%向上
- **画像の統一感**: すべての画像で同じアスペクト比・色調・デザインテイストを統一する
- **高品質な画像**: ぼやけた画像、暗い画像、低解像度の画像は即離脱につながる。鮮明でクリアな画像を使用

### 効果的なクリエイティブパターン

| パターン | 説明 | 適用場面 |
|---------|------|---------|
| ビフォー/アフター | 変化の過程を複数枚で見せる | 美容、フィットネス、リフォーム |
| ステップバイステップ | 手順やプロセスを段階的に紹介 | チュートリアル、レシピ、使い方 |
| 商品カタログ | 複数商品を一覧で紹介 | EC、アパレル、食品 |
| 特徴紹介 | 1枚1特徴で商品の魅力を深掘り | SaaS、テック製品、サービス |
| ランキング/リスト | 人気商品TOP5など | メディア、比較サイト |

### クリエイティブチェックリスト

- 最初の画像が「スワイプしたい」と思わせるフックになっているか
- 各画像が単体でも意味が通るか
- 画像間に視覚的な連続性があるか
- テキストの使用は最小限に抑えているか（画像上のテキスト過多を避ける）
- 最後の画像にCTAを含めているか
- 画像の解像度は720px以上か
- BGM/音楽を設定しているか（必須）

### Smart Creative AIの活用

TikTokのSmart Creative AI機能を有効にすると、画像の順番・キャプション・CTAの組み合わせを自動テストし、最もコンバージョンの高い組み合わせを優先配信する。手動最適化の手間を削減しつつ、パフォーマンスを向上できる。

### やるべきこと

- 3枚以上の高品質画像を用意（9:16推奨）
- 各画像にCTAを組み込む（最終画像は特に重要）
- ストーリー性のある画像構成にする
- テキストオーバーレイでキャプション制限（20文字）を補完する

### やってはいけないこと

- 画像上にテキストを過剰に配置しない
- セーフゾーン（下部20%、右側10%）に重要な情報を配置しない（UIエレメントと被る）
- BGM設定を忘れない（必須のため出稿不可になる）
- 絵文字を使用しない（カルーセルでは使用不可）

## 広告設定ガイド

### 対応キャンペーン目的

| 目的 | 説明 |
|------|------|
| トラフィック（Traffic） | ウェブサイトやアプリへの訪問促進 |
| ウェブコンバージョン（Web Conversion） | 購入・登録などのコンバージョン獲得 |
| アプリプロモーション（App Promotion） | アプリインストール促進 |
| リーチ（Reach） | 幅広いリーチの獲得 |

### カルーセルの種類

| 種類 | 用途 | 画像枚数 |
|------|------|---------|
| スタンダードカルーセル | トラフィック・コンバージョン・アプリプロモーション向け。全業種対応 | 2〜35枚（手動アップロード） |
| VSAカルーセル | Product Sales目的。カタログ画像を使ったインフィード広告 | カタログから自動生成 |

### 設定のポイント

- **画像枚数**: 最低3枚以上を推奨（ストーリーテリング向上のため）
- **各画像に固有のランディングページURL設定可能**: 画像ごとに異なる遷移先を設定し、コンバージョンポイントを増やす
- **ダイナミックフォーマット**: OFFにしてから「カルーセル」を選択する
- **A/Bテスト**: 異なるクリエイティブ・キャプション・CTAの組み合わせを継続的にテスト

### 予算・入札

- コンバージョン目的の場合、日予算は目標CPAの20倍以上を推奨
- 入札戦略は「最大配信（Maximum Delivery）」から開始し、データ蓄積後に「コストキャップ（Cost Cap）」へ移行
- 最低予算要件: キャンペーン $50/日、広告グループ $20/日

### CTAボタンの推奨

| CTA | 推奨用途 |
|-----|---------|
| 今すぐ購入（Shop Now） | 商品ページ遷移、EC |
| 詳しく見る（Learn More） | ウェブサイト・LP遷移 |
| 登録する（Sign Up） | メール登録、無料トライアル |
| ダウンロード（Download） | アプリインストール |

## コピーライティングガイド

### テキスト制限の重要性

カルーセル広告のキャプションは最大20文字と非常に制限が厳しい。この制限を前提にコピーを設計する必要がある。

### コピーライティングのコツ

- **簡潔・明確**: 20文字制限を最大限活用し、核心を一言で伝える
- **行動喚起**: 「詳しく見る」「今すぐ購入」「スワイプして確認」などのCTAを画像内に挿入
- **数字を活用**: 「3つのステップ」「売上50%UP」など、具体的な数字で興味を引く
- **質問形式**: 「〇〇で悩んでいませんか?」などの問いかけでスワイプを促す
- **各画像で訴求を変える**: 1枚目=フック、2枚目=詳細、3枚目=証拠、最終枚=CTA

### テキストオーバーレイのルール

キャプション20文字の制限を補うために、画像内にテキストオーバーレイを入れることが推奨される。

- 画像上のテキストは最小限に
- フォントサイズは読みやすいサイズで
- セーフゾーン（下部20%、右側10%）にテキストを配置しない（UIエレメントと被る）

### 注意事項

- 絵文字は使用不可
- 「{}」「#」は説明文に表示されない
- 1つのキャプションとCTAボタンが全カードに共通で適用される
