# Branded Mission（ブランドミッション）

## 概要

Branded Missionは、TikTokクリエイター（フォロワー1,000人以上）からUGCをクラウドソーシングし、トップパフォーマンスのクリエイター動画を広告として配信する**業界初の3-in-1広告ソリューション**。広告主はミッションを提示し、クリエイターが動画を投稿、優秀な投稿を選定して広告配信する。

- **購入方式**: 予約型（パッケージ料金）。TikTok営業担当経由。Hashtag ChallengeやTopViewとバンドルされることが多い
- **費用感**: 中〜上位5桁ドル（$50,000〜）。月間広告予算$10,000以上で多様なUGCクリエイティブが必要な場合に推奨
- **主な差別化ポイント**: クリエイターからの本格的UGCを大量収集し、広告素材として活用。1キャンペーンで半年分のクリエイティブライブラリを構築可能
- **対応キャンペーン目的**: 認知（Reach）、動画視聴（Video Views）
- **利用可能市場**: 20ヶ国（日本を含む）

**クリエイターの参加条件**:
- 18歳以上
- フォロワー1,000人以上
- 1人あたり最大3動画まで投稿可能

## クリエイティブ制作ガイド

### クリエイティブブリーフの設計

- **1つの明確なクリエイティブプロンプト**: 解釈しやすく、アダプトしやすい単一のビビッドなプロンプトを定義
- **ガイドライン + 自由度のバランス**: ブランドを守りつつ、ユーモアやトレンドを取り入れる余地を残す
- **クリエイティブインスピレーション**: 明確なガイドラインを提供しつつ、クリエイターのユニークなタッチを許容

### 例示動画（Example Videos）

- **最低1本、推奨3〜5本、最大10本**を用意
- クリエイターに期待するコンテンツのタイプを明確に示す
- TikTokアカウントまたはインフルエンサーアカウントに、**ローンチ12時間〜7日前**にアップロード
- TikTok広告審査チームによるレビューが必要

### ミッション要件の設定

**一般要件（最低1つ必須）**:
- **公式音楽の使用**: ブランド提供のオリジナル音楽を使用
- **公式ハッシュタグの使用**: ブランド指定のハッシュタグを含める
- **Branded Effectの使用/トリガー**: ブランドエフェクトの使用を要件に含める（強く推奨）

**ブランド要件（オプション、強く推奨）**:
- **ブランドロゴを1秒以上表示**: ブランド認知向上に効果的
- **ブランド名・ロゴの提供**

### キー情報の配置

- **重要な要素（コピー、CTA）は動画の中央に配置**
- UI要素による遮蔽を避ける

### UGC品質向上のポイント

- クリエイターに明確な方向性を示しつつ、TikTokネイティブなスタイルを維持させる
- 単一のビビッドなプロンプトで統一感のあるUGCを集める
- 集まった動画からトップパフォーマンスのものを選定・ブースト

### やるべきこと

- 1つの明確なクリエイティブプロンプトを設定
- 最低1本（推奨3〜5本）の例示動画を制作
- 例示動画をローンチ12時間〜7日前にアップロード
- 一般要件を最低1つ設定（音楽/ハッシュタグ/エフェクト）
- Branded Effectの使用要件を追加（強く推奨）
- ブランドロゴ1秒以上表示の要件を追加（強く推奨）
- キー情報を動画中央に配置する指示を含める

### やってはいけないこと

- ガイドラインが曖昧なブリーフ
- クリエイターの自由度を過度に制限するルール
- 例示動画なしでのローンチ
- ブランドイメージに合わないコンテンツへの禁止事項の不記載

## 広告設定ガイド

### キャンペーンワークフロー

| ステップ | 内容 |
|---------|------|
| 1. ミッション作成 | TikTok Ads Managerの「Mission Manager」から設定 |
| 2. 要件設定 | クリエイター向けの要件（音楽/ハッシュタグ/エフェクト/ロゴ）を定義 |
| 3. クリエイター参加 | 1,000+フォロワーの18歳以上クリエイターが動画を投稿（最大3本/人） |
| 4. 動画選定 | トップパフォーマンス動画がショートリストとして浮上 |
| 5. ブースト決定 | 広告主がブランドに最も合致する動画を選定してブースト |
| 6. 広告配信 | 選定動画がIn-Feed Adとして配信 |

### 動画管理（Branded Mission Video Management）

- TikTok Ads Managerのスタンドアロンページ
- **Candidate Videos**: 選定候補として共有・レビュー
- **ブースト判断**: 即座にブースト / 6〜12時間以内にブースト / ミッションページから除外
- **チームレビュー**: 候補動画をチームに共有してレビュー

### 予算・コスト

| 項目 | 詳細 |
|------|------|
| 最低予算 | 中〜上位5桁（$50,000〜） |
| バンドル推奨 | Hashtag ChallengeやTopViewとのバンドルが一般的 |
| 推奨条件 | 月間広告予算$10,000以上で多様なUGCクリエイティブが必要な場合 |

### ローンチタイミング

- **月曜日ローンチ推奨**: クリエイターに1週間のフル制作期間を提供し、週末のトラフィックピークに合わせる

### クリエイター報酬

- キャンペーン終了時に、選定クリエイターはBranded Missionの条件に基づいて報酬を受け取る
- 有料動画ビューに応じた支払い

### アセット活用

- 選定動画の**使用権を取得**
- **1キャンペーンで半年分のクリエイティブライブラリを構築可能**
- 各広告にクリエイティブ属性をタグ付けし、パフォーマンス分析

### 推奨バンドル

| パターン | 組み合わせ | 予算目安 |
|---------|----------|---------|
| UGC収集型 | Branded Mission + Branded Effects + In-Feed Ads | $100,000〜$300,000 |
| フル装備参加型 | Branded HTC + Branded Effects + TopView + Branded Mission | $400,000+ |

## コピーライティングガイド

### ミッション名

- **文字数制限**: 最大70英字
- **推奨**: 18英字以内（主要モバイルデバイスで1行表示）
- **命名のポイント**:
  - ミッションの内容が一目でわかる名前
  - クリエイターの参加意欲を刺激する表現
  - ブランド名を含める

### ミッション説明文

- **文字数**: 最大300英字（150英字以内推奨）
- **内容**:
  - ミッションの目的・テーマ
  - 参加方法の明確な説明
  - 求めるコンテンツのスタイル・トーン

### ハッシュタグ・テキスト要件

- **ハッシュタグ**: スペース、句読点を含む全文字がカウント対象
- **絵文字**: emoji v.11を使用
- **キャプション**: クリエイター動画のオーガニックキャプションがそのまま使用される

### クリエイティブブリーフのコピー

明確で具体的な方向性を示す:
- 「この商品を使って、日常の○○シーンを撮影してください」
- 「公式音楽に合わせて○○を表現してください」
- 「Branded Effectを使って○○を体験してください」

**禁止事項も明記**: ブランドイメージに合わないコンテンツの例示

### クリエイター向けインセンティブコピー

- 報酬条件を明確に提示
- 「トップ動画は広告として配信され、より多くの視聴者にリーチ」
- クリエイターの動機付けになる表現を使用
