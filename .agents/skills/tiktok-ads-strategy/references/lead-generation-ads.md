# Lead Generation Ads（リード獲得広告）

## 概要

TikTokアプリ内でInstant Form（インスタントフォーム）を表示し、ユーザーの連絡先情報を収集する広告フォーマット。外部ランディングページへ遷移させずにアプリ内で完結するため、フォーム読み込みが高速でコンバージョン率が高い。

- **購入方式**: オークション型（運用型）
- **課金方式**: oCPM, CPC
- **対応キャンペーン目的**: リード獲得（Lead Generation）
- **主な差別化ポイント**: アプリ内完結のフォーム体験、TikTokアカウント情報からの自動入力、CRM連携対応
- **最適な業種**: B2B、教育、不動産、美容、コンサルティング
- **技術的難易度**: 低

### Instant Form仕様

| 項目 | 仕様 |

| フォームタイプ | More Volume / Higher Intent |
| テンプレート | Simple Signup / Rich Content / Lead Qualification |
| 自動入力 | 名前、メール、電話番号（TikTokアカウント情報から） |

| テンプレート | 用途 |
|------------|------|

| リッチコンテンツ（Rich Content） | カスタムコンテンツブロックを追加可能。認知・信頼醸成向け |
| リード認定（Lead Qualification） | カード型レイアウト。回答に基づくプリフィル質問でリード質を向上 |

## クリエイティブ制作ガイド

### 動画制作の基本原則

- **最初の3秒で勝負を決める**: 冒頭でユーザーの痛みやニーズに直接語りかける。「今すぐ登録」から始めるのではなく、共感できるシーンや課題提起から入る
- **縦型フルスクリーン（9:16）で撮影**: TikTokネイティブなフォーマットが最もパフォーマンスが高い
- **モバイル撮影を活用**: スマホ撮影のクリエイティブは、スタジオ撮影と比較して購入・チェックアウト・アプリインストールで**63%高い**パフォーマンスを記録
- **2〜3秒ごとにカメラアングルやシーンを切り替え**: 静止画のような単調なショットは視聴者を失う
- **音声を活用**: TikTokは音声プラットフォーム。BGM、ナレーション、トレンドサウンドを積極的に使用
- **クリエイティブの品質が売上リフトの最大47%を占める**ことを意識する

### 効果の高いクリエイティブパターン

| パターン | 説明 | 適用場面 |
|----------|------|----------|
| **UGC風テスティモニアル** | 一般ユーザーや小規模クリエイターが商品・サービスを紹介 | 信頼構築、認知拡大 |
| **Before/After** | 使用前後の変化を視覚的に見せる | 美容、フィットネス、教育 |
| **問題解決型** | 課題提示 → 解決策（自社サービス）→ CTA | B2B、SaaS、コンサルティング |
| **クイズ・診断型** | 「あなたはどのタイプ？」など参加型コンテンツ | リード獲得の質向上 |
| **限定オファー** | 期間限定割引や無料トライアルの告知 | 緊急性の訴求 |
| **開封・デモ動画** | 商品のアンボクシングやサービスデモ | EC、アプリ、SaaS |

### クリエイティブ最適化チェックリスト

- 冒頭3秒にフックを入れているか
- 縦型9:16で制作しているか
- 字幕・テキストオーバーレイを追加しているか（音声オフ対応）
- 実在の人物が登場しているか
- ブランドロゴを控えめに配置しているか（広告感の軽減）
- CTAが明確に表示されているか
- 動画尺は9〜15秒の範囲内か

### やるべきこと

- 共感できるシーンや課題提起から動画を開始する
- UGC風の自然なクリエイティブを制作する
- テキストオーバーレイで音声オフでも伝わるようにする
- 具体的な数字（「30日で○○達成」「98%の満足度」）を活用する

### やってはいけないこと

- 「今すぐ登録」から始める（ユーザーが離脱する）
- 単調なカメラアングルで撮影する
- スタジオ撮影風の過度に作り込んだクリエイティブ
- 音声なしの無音動画

## 広告設定ガイド

### キャンペーン構成

| 設定項目 | 推奨値 |
|----------|--------|
| キャンペーン目的 | Lead Generation（リード獲得） |
| 最適化目標 | リード送信（Submit） |
| 課金方式 | oCPM（最適化インプレッション単価）またはCPC |
| 推奨広告グループ数 | 3〜5グループ（異なるターゲティング・クリエイティブで並走テスト） |
| 最低日予算（キャンペーン） | $50/日 |
| 最低日予算（広告グループ） | $20/日 |

### フォームタイプの選択

| フォームタイプ | 特徴 | 推奨場面 |
|---------------|------|----------|
| **More Volume（ボリューム重視）** | 確認画面なし。送信ハードルが低くリード数が最大化 | リード数を最優先する場合、初期テスト時 |
| **Higher Intent（高意図）** | 送信前に確認画面+CAPTCHA表示。リード品質が向上 | リード品質を重視する場合、営業リソースが限られる場合 |

### ターゲティング戦略

- **興味関心ターゲティング**: 自社商品に関連する興味カテゴリを選択
- **類似オーディエンス**: 既存顧客データからLookalike Audienceを作成
- **リターゲティング**: 過去の動画を50%以上視聴したユーザーをリターゲット
- **幅広すぎるターゲティングを避ける**: ハイパーターゲットされたサブオーディエンスに絞り、各セグメントの言語で語りかける

### 入札・予算設定

- **oCPM入札**: 計算式 = （総コスト / 総インプレッション数）x 1,000
- **初期予算**: 十分な学習データを蓄積できる金額を設定
- **A/Bテスト**: More Volume vs Higher Intentを並行テスト

### フォーム設計のベストプラクティス

- **質問数は最小限に**: 質問が増えるほど完了率が低下する。必要最低限の情報のみ収集
- **自動入力を活用**: 名前・メール・電話番号はTikTokアカウント情報から自動入力されるため、手入力の負担を軽減
- **条件ロジックを活用**: 前の回答に基づいて次の質問を動的に変更し、不適格リードにはフォームを早期終了
- **選別質問（Qualifying Question）**: 多肢選択・画像選択の質問でpreferred/non-preferredにリードを自動分類
- **Form Card（簡易フォーム）**: 質問数は最大3問に制限。時間に敏感なオファー（イベント申込、フラッシュセール）に最適

### CRM連携

- **自動化**: TikTok Instant FormをCRMに直接連携してリードナーチャリングを自動化
- **迅速なフォローアップ**: リード獲得後の即時対応がコンバージョン率を大幅向上
- **トラッキングパラメータ**: UTMパラメータでリード元チャネルを識別

## コピーライティングガイド

### 動画内テキストオーバーレイ

- **フック文（冒頭）**: 「知らなかった○○の方法」「○○で悩んでいませんか？」など疑問・好奇心を刺激
- **ベネフィット提示**: 機能ではなく、ユーザーが得られる価値を具体的に
- **数字を活用**: 「30日で○○達成」「98%の満足度」など具体的な数字で説得力を高める
- **緊急性**: 「今だけ」「先着○名」「期間限定」

### キャプション（広告テキスト）

- **短く簡潔に**: TikTokユーザーはキャプションを長時間読まない（最大100文字、最初の80〜100文字が表示）
- **CTAを明確に**: 「詳細はプロフィールから」「今すぐ無料登録」
- **ハッシュタグ**: 関連ハッシュタグを2〜3個（多すぎない）
- **絵文字の適度な使用**: TikTokのカジュアルなトーンに合わせる

### フォーム内コピー

| 要素 | ベストプラクティス | 例 |
|------|-------------------|-----|
| **ヘッドライン** | 価値提案を一文で | 「無料で○○診断を受け取る」 |
| **説明文** | フォーム入力の理由を明確に | 「30秒で完了。あなたに最適なプランをご提案します」 |
| **CTAボタン** | 具体的なアクション | 「無料で始める」「今すぐ予約」「見積もりを取得」 |
| **プライバシーポリシー** | 安心感の提供（必須） | 「個人情報は厳重に管理されます」 |
| **サンクスページ** | 次のステップを明示 | 「担当者から24時間以内にご連絡いたします」 |

### フォーム質問のベストプラクティス

- **標準フィールド**（名前、メール、電話番号）は自動入力対応なので優先的に使用
- **カスタム質問**: 最大10問まで追加可能だが、必要最小限にとどめる
- **質問形式の選択**: 短文回答、多肢選択、画像選択（最大8画像）、予約スケジューリングから目的に応じて選択
- **条件付き回答**: 前の質問への回答に基づいて表示する選択肢を動的に変更

### Form Card（簡易フォーム）のコピー

- 質問数は最大3問に制限（名前・メール・電話番号のみ）
- 時間に敏感なオファー（イベント申込、フラッシュセール）に最適
- 動画の直下にフォームが表示されるため、CTAとの一貫性が重要
