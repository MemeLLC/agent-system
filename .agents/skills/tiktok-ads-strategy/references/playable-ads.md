# Playable Ads（プレイアブル広告）

## 概要

HTML5で構築されたインタラクティブなフルスクリーン広告。ゲームやアプリのデモ体験をインストール前に提供する。ユーザーがタップ・スワイプ・プレイできるため、ゲーム・アプリインストールキャンペーンに特に効果的。

- **購入方式**: オークション型（運用型）
- **課金方式**: oCPM, CPC
- **対応キャンペーン目的**: アプリプロモーション（App Promotion / App Install）
- **主な差別化ポイント**: インストール前にゲーム/アプリのデモ体験を提供、HTML5ベースのインタラクティブ広告
- **配信プレースメント**: **Pangleが必須**（TikTok以外のサードパーティモバイルアプリに配信するネットワーク）。TikTokフィード内でも配信可能だがPangleが主力
- **技術的難易度**: 高（HTML5開発が必要）
- **パフォーマンス実績**: 標準的なモバイル広告と比較してコンバージョン率が**123%向上**、CPI（インストール単価）が**11%低下**

## 仕様

### プレイアブル広告仕様

| 項目 | 仕様 |
|------|------|
| ファイル形式 | .zip（HTML5パッケージ） |
| ファイルサイズ | 5MB以下（圧縮後） |
| 必須ファイル | index.html（ルートディレクトリ）、config.json |
| メインHTMLファイル名 | index.html（第一階層に配置） |
| config.json | 広告の画面向き（orientation）を定義 |
| 外部リクエスト | 禁止（HTTPリクエスト不可） |
| JSリダイレクト | 禁止 |
| SDK | Pangle js-sdkへのアクセスを許可（アプリストア遷移用） |

### リードイン動画仕様（併用する場合）

| 項目 | 仕様 |
|------|------|
| 解像度 | 1080 x 1920 px（推奨） |
| アスペクト比 | 9:16 |
| 動画形式 | .MP4, .MOV |
| ファイルサイズ | 最大500MB |

### 適用可能なアプリカテゴリ

| カテゴリ | 適合度 | 説明 |
|----------|--------|------|
| **モバイルゲーム** | 最適 | ゲームプレイのデモ体験が最も効果的 |
| **漫画・コミックアプリ** | 高 | インタラクティブなストーリープレビュー |
| **ストリーミングアプリ** | 中 | 主要機能のハイライト |
| **ユーティリティアプリ** | 中 | キー機能のインタラクティブデモ |

## クリエイティブ制作ガイド

### デザインの基本原則

- **コアの楽しさを数秒で伝える**: パズルゲームなら1つのシンプルなパズル、ランナーゲームなら10秒間の障害物ジャンプなど、ゲームの核心的な魅力を凝縮
- **シンプルで直感的な操作**: タップ・スワイプなどTikTokユーザーが馴染みのある操作で完結させる
- **サイレントファースト設計**: TikTokユーザーの**60%は音声オフ**で閲覧。音声がなくても伝わる視覚的フィードバック・キューを設計
- **ジャイロスコープ・加速度センサーの使用は避ける**: 特定の姿勢・状況でしか機能せず、ユーザー体験を損なう

### リード動画の活用

- **リードイン動画を組み合わせる**: プレイアブル体験の前に短い動画を流し、スムーズな導入を提供
- **動画でゲームの魅力を予告**: 「このゲームをプレイしてみよう」的な導入がエンゲージメントを向上

### CTAの設計

- **ダウンロードボタンを常に表示**: ユーザーがいつでもアプリストアに遷移できるようにする
- **ゲーム名・ロゴ・アイコンを表示**: プレイアブル体験後のファイナル画面にゲームのアイデンティティを明確に
- **CTAボタンは大きく目立つデザイン**: 「今すぐプレイ」「フルバージョンをダウンロード」
- **複数のクリック可能要素**: ゲーム名、ロゴ、アップグレードボタンなどもクリック可能にし、コンバージョン機会を増やす

### クリエイティブ疲れへの対策

- **2〜4週間でパフォーマンスが低下**: 同じ広告を見続けるユーザーの反応が鈍化
- **バリエーション制作**: 異なるレベル、キャラクター、チャレンジで複数バージョンを用意
- **定期的な更新**: 新しいプレイアブル体験を継続的に投入

### 技術的な制作tips

- **軽量エンジンの使用**: PixiJS、CreateJSなど軽量なHTML5ゲームエンジンを活用
- **ベクターグラフィックス**: ファイルサイズ削減と高品質の両立
- **アセットの圧縮**: 画像・音声を徹底的に圧縮して5MB以内に収める
- **分割ダウンロード**: プレイアブル素材を小分けにし、並列ダウンロードを有効化

### やるべきこと

- ゲームの核心的な魅力を数秒で凝縮して伝える
- タップ・スワイプなど馴染みのある直感的な操作にする
- 音声オフでも伝わる視覚的フィードバックを設計する
- ダウンロードボタンを常に画面上に表示する
- 2〜4週間ごとにバリエーションを更新する

### やってはいけないこと

- ジャイロスコープ・加速度センサーを使用する
- 複雑な操作を要求する
- 外部HTTPリクエストを含める
- JSリダイレクトを含める
- 実際のアプリと異なる体験を提供する（ポリシー違反）
- 5MBを超えるファイルサイズにする

## 広告設定ガイド

### キャンペーン設定

| 設定項目 | 推奨値 |
|----------|--------|
| キャンペーン目的 | App Promotion（アプリプロモーション） |
| キャンペーンタイプ | App Install（アプリインストール） |
| 配置 | **TikTok + Pangle（Pangleは必須で含める）** |
| 課金方式 | oCPM または CPC |
| 最適化目標 | アプリインストール |
| 最低日予算（キャンペーン） | $50/日 |
| 最低日予算（広告グループ） | $20/日 |

### 配置（プレースメント）の注意点

- **Pangle配置が必須**: プレイアブル広告は主にPangle（TikTokのオーディエンスネットワーク）で配信される
- **Pangleとは**: TikTok以外のサードパーティモバイルアプリ（ゲーム、ユーティリティなど）に広告を配信するネットワーク。38万以上のグローバルアプリ、DAU 29億のアプリトラフィック
- TikTokフィード内でも配信可能だが、Pangleでの配信が主力
- **Pangleの広告フォーマット**: インタースティシャル広告、リワード動画広告、ネイティブ広告、バナー広告、アプリオープン広告

### ターゲティング戦略

- **ゲームカテゴリ興味関心**: 類似ジャンルのゲームに興味を持つユーザーをターゲット
- **類似オーディエンス**: 既存インストールユーザーからLookalike Audienceを作成
- **デバイス別**: ゲームの動作要件に合ったデバイスをターゲット
- **地域別最適化**: ゲームの対応言語・リリース地域に合わせてターゲティング

### 入札・予算の最適化

- **初期は幅広く**: 学習フェーズでは広めのターゲティングと十分な予算で配信
- **CPI目標を明確に**: 過去のインストール単価データを基に現実的な目標を設定
- **クリエイティブごとの予算配分**: パフォーマンスの高いプレイアブルに予算をシフト

## コピーライティングガイド

### リードイン動画のテキスト

- **チャレンジ系**: 「このレベル、クリアできる？」「99%が失敗するパズル」
- **好奇心喚起**: 「次に何が起こるか予想できない」「意外な結末」
- **数字活用**: 「レベル100まで何分で到達できる？」
- **感情喚起**: 「中毒注意」「止められなくなる」

### CTAテキスト

| CTA | 適用場面 |
|-----|----------|
| **「今すぐプレイ」** | ゲーム全般 |
| **「フルバージョンをダウンロード」** | デモ後のインストール誘導 |
| **「もっとプレイする」** | 続きへの興味喚起 |
| **「無料でダウンロード」** | 無料ゲームの訴求 |
| **「チャレンジに挑戦」** | 競争心の刺激 |

### インタラクション中のテキスト

- **チュートリアル指示**: 「タップして跳ぶ」「スワイプして移動」など操作説明
- **進捗フィードバック**: 「すごい！」「あと少し！」などポジティブな反応
- **スコア・結果表示**: プレイ結果を明示して達成感を演出
- **テキストは最小限**: プレイ体験を邪魔しないシンプルな表現

### キャプション

- ゲームの最大の魅力を一文で表現
- ハッシュタグは関連性の高いもの2〜3個
- 例: 「#モバイルゲーム」「#無料ゲーム」「#パズル」などジャンル系タグ
- 広告説明文は1〜100文字（最初の80〜100文字が表示）
- 絵文字使用可

| 指標 | ベンチマーク |
|------|------------|
| コンバージョン率（vs 標準モバイル広告） | **123%向上** |

### 主要KPIの優先度

| 指標 | 説明 | 重要度 |
|------|------|--------|
| CTR | クリック率。広告のエンゲージメント度合い | 高 |
| CVR | コンバージョン率。ユーザー品質とクリエイティブの関連性 | 高 |
| CPI | インストール単価。費用対効果 | 高 |
| リテンション率 | インストール後の継続率。ユーザー品質 | 中〜高 |
| プレイ完了率 | プレイアブル体験の完了率 | 中 |
