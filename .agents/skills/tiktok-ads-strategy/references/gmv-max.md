# GMV Max（TikTok Shop専用）

## 概要

TikTok ShopのAI搭載型広告自動化ソリューション。オーディエンスターゲティング・プレースメント・スケジューリングをリアルタイムの行動データとアルゴリズム最適化で自動化する。2025年7月以降、TikTok Shop広告の唯一のキャンペーンタイプとなる。

- **購入方式**: オークション型（自動最適化）
- **課金方式**: oCPM（自動入札）
- **対応キャンペーン目的**: 販売（Sales）- TikTok Shopが販売先
- **主な差別化ポイント**: AI駆動の完全自動化、オーガニック配信と有料広告の統合最適化、ROI保護機能
- **最適な業種**: EC（TikTok Shop出店者）
- **技術的難易度**: 中（AI自動化のため設定はシンプルだが、大量のクリエイティブ供給が必要）
- **パフォーマンス**: 初期テストでGMVが**30%向上**（TikTok内部データ）
- **利用可能地域**: アメリカ、インドネシア、マレーシア、フィリピン、タイ、ベトナム、シンガポール

### 2つのキャンペーンタイプ

| タイプ | 概要 |
|--------|------|
| **Product GMV Max** | TikTok Shopの全チャネルROIを最適化。利用可能な全クリエイティブアセットを使用して広告を自動生成。オーガニック配信と有料広告の両方を最適化 |
| **LIVE GMV Max** | ライブ配信のルームトラフィックを最適化し、総収益を最大化。クリエイティブタイプ: Video-to-live、Live-to-live |

## クリエイティブ制作ガイド

### 最重要原則: クリエイティブのボリュームが全て

GMV Maxの成功において最大の要因は**クリエイティブのボリューム**。システムがAIで最適なクリエイティブを自動選択するため、選択肢が多いほどパフォーマンスが向上する。

- **最低50〜70本の動画**を用意してクリエイティブ探索に投入
- **毎日5〜10本の新しい動画クリエイティブ**を追加
- クリエイティブの供給が途絶えるとパフォーマンスが急激に低下

### クリエイティブの種類と推奨パターン

| 種類 | 説明 | 効果 |
|------|------|------|
| **アフィリエイトUGC** | クリエイターが日常生活で商品を紹介 | 自然な訴求、高い信頼性 |
| **セラーオーガニック投稿** | ブランドアカウントからのTikTokネイティブ投稿 | ブランド認知、一貫性 |
| **開封・チュートリアル** | 商品の開封や使い方を短く紹介 | 商品理解の促進 |
| **変身・Before/After** | 使用前後の変化を視覚的に表現 | 効果の実証 |
| **トレンド活用** | トレンドサウンド・スキット・編集スタイルに合わせた商品紹介 | リーチの最大化 |

### 動画制作のコツ

- **最初の3秒以内**にフックを入れて視聴者を引き留める
- **商品の使用シーン**を必ず映す
- **画面上テキスト**でベネフィット・価格を表示
- **明確なCTA**で締めくくる
- TikTokネイティブなフックを使用（トレンドに沿った導入）

### クリエイティブの管理

- **GMV Maxに直接クリエイティブのアップロードは不可**: すべてのクリエイティブはオーガニック投稿（セラーコンテンツ、アフィリエイトコンテンツ、Spark Ads認可投稿）として事前に存在している必要がある
- **特定動画の除外機能**: 古いコンテンツやブランドイメージに合わない動画をキャンペーンから除外可能
- **優先動画の設定**: LIVE GMV Maxでは最大10本の優先動画を選択可能（各ライブ配信ごとに更新推奨）

### LIVE GMV Max固有のクリエイティブ戦略

- ライブ配信に最も関連性の高い動画を優先設定
- 各ライブ配信ごとに優先動画の選択を更新
- ライブ配信中のエンゲージメントを高めるコンテンツを事前に用意

### やるべきこと

- 最低50〜70本のクリエイティブを用意する
- 毎日5〜10本の新規クリエイティブを追加する
- オーガニック投稿として事前にコンテンツを公開する
- 古い・ブランド不適合なクリエイティブを除外する
- ベストセラー商品をGMV Maxに優先投入する

### やってはいけないこと

- クリエイティブの供給を途絶えさせる（パフォーマンスが急激に低下）
- 直接アップロードしようとする（オーガニック投稿経由のみ対応）
- AIの学習を阻害する頻繁な手動調整
- ROIターゲットの頻繁な変更

## 広告設定ガイド

### Product GMV Max設定

**キャンペーン構成:**

| 設定項目 | 推奨値 |
|----------|--------|
| キャンペーンタイプ | Product GMV Max |
| 最適化目標 | ROIターゲット |
| 課金方式 | oCPM（自動入札） |
| 商品選択 | 全商品 or 選択商品 |

**ROIターゲット設定:**
- システム推奨値に従うか、自社の目標に合わせてカスタム設定
- 初期テスト時はROI 1.3〜1.5からスタート
- 3日間の観察期間で消化が30%未満の場合、ROIを1.5→1.3に調整（各調整は50%以内）

**予算設定:**
- **初期予算**: 約$500（システムの探索に十分な余裕を持たせる）
- **日次予算**: 最低でもAOV（平均注文額）の10倍以上
- **システム推奨に従う or 過去のLIVE広告の2〜3倍**

**Max Delivery最適化（Max配信最適化）:**
- 初期はMax Deliveryを有効にして早期トラフィックを最大化
- 5〜7日間の日次予算で運用し、システムの最適化を待つ
- ROIターゲットが不確実な場合に特に有効
- テスト後、安定したROIを求める場合はTarget ROIに切り替え

### LIVE GMV Max設定

**キャンペーン構成:**

| 設定項目 | 推奨値 |
|----------|--------|
| キャンペーンタイプ | LIVE GMV Max |
| 最低キャンペーン期間 | 3日以上 |
| 推奨ライブ時間 | 3時間以上（理想は8時間以上/日） |
| ROIターゲット | 過去のLIVEパフォーマンスに基づく |
| 予算 | 過去のLIVE広告の2〜3倍 |

**ライブ配信の最適化:**
- ライブ配信のタイミングと時間をキャンペーン日数全体で一貫させる
- 3時間以上のセッションでGMV成長が加速
- 8時間以上のライブ配信で単セッションGMVが顕著に増加

### ROI保護機能

**仕組み:**
- キャンペーンのROIが設定ターゲットの90%を下回った場合、自動的に広告クレジットを発行
- **計算式**: 広告クレジット = 実際のコスト - （実際の総収益 / (0.9 x 設定ROIターゲット)）
- 例: $100支出、$300売上目標で$270達成 → $30のクレジット返還

**適格条件:**
- 日次注文数が20件以上
- キャンペーンROIが設定ターゲットの90%を下回っている
- キャンペーンライフタイム全体で適用

**非適格条件（ROI保護が適用されない場合）:**
- Target ROIを手動で変更した日
- キャンペーンを一時停止・削除した日
- Max Delivery・Creative Boost予算を使用している商品・ライブルーム

**確認方法:**
- キャンペーンリストでアクティブステータス横の緑のシールドアイコンで適格性を確認

### 商品選択戦略

| 商品タイプ | GMV Max適合度 | 推奨アプローチ |
|-----------|-------------|--------------|
| **ベストセラー（過去7日$1,500+GMV）** | 最適 | GMV Maxで直接プロモーション |
| **クリエイティブ豊富な商品** | 高 | 十分な動画アセットでGMV Maxに投入 |
| **コールドスタート商品** | 低 | まずVideo Shopping AdsやProduct Listing Adsで実績作り |

## コピーライティングガイド

### 商品リスティングのコピー

GMV Maxはクリエイティブを自動選択するため、各オーガニック投稿の品質が直接パフォーマンスに影響する。

- **商品タイトル**: 検索キーワードを含む明確で簡潔なタイトル
- **商品説明**: ベネフィット重視、スペック情報も含める
- **価格表示**: 割引率や元値との比較を明示

### 動画内テキスト

- **フック（最初の3秒）**: 「○○で悩んでいる人必見」「これを知らないと損」
- **ベネフィット表示**: 画面上テキストで商品の主要メリットを表示
- **価格・割引情報**: 「今だけ○○%OFF」「送料無料」
- **CTA**: 「今すぐショップで購入」「カートに追加」「詳細はプロフィールから」

### ライブ配信のスクリプト

| フェーズ | コピー要素 | 例 |
|----------|-----------|-----|
| **オープニング** | 視聴者への挨拶、今日の内容紹介 | 「今日は○○の新商品を特別価格でご紹介！」 |
| **商品紹介** | 特徴、ベネフィット、使い方 | 「この○○、実は○○にも使えるんです」 |
| **デモ** | 実演、Before/After | 「見てください、たった○分でこの仕上がり」 |
| **オファー** | 限定価格、特典 | 「ライブ限定で○○%OFF！在庫残りわずか」 |
| **CTA** | 購入への誘導 | 「今すぐカートに追加してください！」 |

### オーガニック投稿のコンテンツ戦略

GMV Maxでは直接クリエイティブをアップロードできないため、オーガニック投稿の品質と量が成功の鍵となる。

- **セラーコンテンツ**: ブランドアカウントからのネイティブ投稿を定期的に公開
- **アフィリエイトコンテンツ**: クリエイターとの連携でUGCを大量生産
- **Spark Ads認可投稿**: パフォーマンスの高い投稿をSpark Adsとして認可

### パフォーマンス測定のコピー観点

- **Gross Revenue（総収益）** を最重要指標として動画・コピーの品質を評価
- ROI単体ではなく、総収益が高い動画のコピーパターンをベンチマークにする
- 高パフォーマンス動画のコピー構成を分析し、他の動画に横展開
