# Search Ads（検索広告）

## 概要

TikTokの検索結果ページにキーワードターゲティングで表示される広告。ユーザーが特定のキーワードで検索した際に、検索結果の「Sponsored」枠に広告を表示する。Google検索広告と同様のキーワードオークション方式だが、Google検索より低CPCの場合が多い。動画クリエイティブとカルーセル画像の両方に対応。

- **購入方式**: オークション型（運用型）。日予算2万〜10万円程度からスモールスタート可能
- **課金方式**: CPC, oCPM
- **対応キャンペーン目的**: ウェブコンバージョン（Sales）、トラフィック
- **主な差別化ポイント**: 検索意図の高いユーザーにリーチ可能、キーワードベースのターゲティング
- **重要データ**: 70%の広告グループで自動検索配置をONにした方がOFFより効率的なパフォーマンス（TikTok社内調査）

## クリエイティブ制作ガイド

### 基本原則

- **検索意図に合致するクリエイティブ**: キーワードとクリエイティブの関連性がクオリティスコアに直結する。靴のキーワードグループには靴関連のクリエイティブを使用
- **最初の3秒にキーワード/見出しを配置**: ユーザーの検索意図に即座に応える情報を冒頭に表示
- **問題解決型コンテンツ**: 検索ユーザーは「解決策」を求めている。製品がどう問題を解決するかを明確に見せる
- **オーセンティックな動画**: 磨き上げたプロ映像よりも、リアルで共感できるコンテンツが高パフォーマンス

### クリエイティブの量と更新

- 1広告グループあたり5〜20個のクリエイティブバリエーションを用意
- クリエイティブごとにパフォーマンスに大きな差が出るため、複数バリエーションのテストが必須
- クリエイティブ疲れ（CTR低下・エンゲージメント減少）が見られたら週1回程度で更新

### クリエイティブ制限事項

- インタラクティブステッカー、高度なエフェクトなどのアドオンは検索配信面では非対応
- 1広告グループに複数クリエイティブを登録可能だが、表示されるのは1つのみ（デフォルトで選択）

### クリエイティブチェックリスト

- クリエイティブがターゲットキーワードと直接関連しているか
- 最初の3秒以内に検索意図に応える情報があるか
- 明確なCTAが含まれているか（「今すぐ購入」「詳しく見る」「注文する」）
- 最も関連性の高い商品/コレクションページに遷移するか
- 複数バリエーションを用意しているか（最低5個）

### やるべきこと

- キーワードに直接関連するクリエイティブを制作する
- 冒頭3秒で検索意図に応える
- Spark Adsを活用して既存の高パフォーマンスオーガニック投稿を広告化する
- 複数バリエーションで継続的にA/Bテストする

### やってはいけないこと

- キーワードと無関係なクリエイティブを使用しない（クオリティスコアが低下する）
- クリエイティブの更新を怠らない（疲れが発生する）
- ブロードマッチを幅広いカタログなしで使用しない（初期学習期を除く）

## 広告設定ガイド

### 配信方式の種類

| 方式 | 説明 | 推奨ケース |
|------|------|-----------|
| Search Ads Campaign（検索広告キャンペーン） | 専用キャンペーンとしてキーワード・入札・クリエイティブを個別管理 | 検索広告を本格的に運用する場合 |
| Automatic Search Placement（自動検索配置） | インフィードキャンペーンのトグルをONにするだけ | 手軽に検索面にも配信したい場合 |

### Automatic Search Placement（自動検索配置）

- 新規キャンペーン作成時、デフォルトでON
- 既存のインフィード動画クリエイティブとターゲティングをそのまま検索結果に配信
- 70%の広告グループでONにした方がOFFより効率的なパフォーマンス（TikTok社内調査）
- ネガティブキーワードで不要な検索語句をブロック可能
- 「設定して放置」ではなく、定期的に配信検索語句をレビューし最適化が必要

### キーワード戦略

#### マッチタイプ

| マッチタイプ | 説明 | 推奨フェーズ |
|-------------|------|------------|
| ブロードマッチ | 類義語・関連検索にも広く配信 | ローンチ期（学習データ蓄積） |
| フレーズマッチ | 指定フレーズが含まれる検索に配信 | 最適化期 |
| 完全一致 | 完全に一致する検索のみ | 効率最大化期 |

#### 3フェーズ戦略

1. **ローンチ期**: ブロードマッチでインテントバリエーションを広くキャプチャし、学習を加速
2. **最適化期**: 実績のあるキーワードをフレーズ/完全一致に昇格させ、効率とリーチのバランスをとる
3. **除外期**: 検索語句レポートからネガティブキーワードを追加し、無関係なクエリを除外

#### ネガティブキーワード

| マッチタイプ | 動作 |
|-------------|------|
| ブロードマッチ | ブロック語をすべて含む検索をブロック（語順不問） |
| フレーズマッチ | 指定順でフレーズを含む検索をブロック |
| 完全一致 | 完全一致の検索のみブロック |

- 検索語句レポートを定期的にレビューし、無関係なクエリをネガティブキーワードに追加
- ブランド毀損リスクのあるキーワードも除外

### クオリティスコア

- キーワード・クリエイティブ・アカウントのクエリの3要素の関連性で評価
- 各クリエイティブとキーワードの組み合わせごとに個別スコアが付与（平均値ではない）
- スコアが高いほどオークションでの優先度が上がり、効率的なトラフィックとコンバージョンを獲得しやすい
- 改善方法: キーワードと直接関連するクリエイティブを使用し、広告タイトル・説明文にターゲットキーワードを含める

### 予算・入札

| 項目 | 推奨 |
|------|------|
| テスト初期予算 | 約$30/日から開始 |
| コンバージョン最適化時の日予算 | 目標CPA x 20倍 |
| 入札戦略 | 初期は「最大配信」→ データ蓄積後に「コストキャップ」 |
| 目標CPA設定 | ローンチ時は空欄推奨（配信制限を避けるため） |
| テスト期間 | 最低5〜7日間 |

### 入札方式

| 方式 | 説明 |
|------|------|
| oCPM | 最適化されたCPM。コンバージョンしやすいユーザーに配信 |
| CPM | 1,000インプレッションあたりのコスト |

### 最適化のフロー

1. 5〜7日間テスト配信
2. 配信されない場合 → 予算増額 or キーワード拡張
3. コンバージョンがない場合 → キーワード/クリエイティブの見直し
4. 勝ちパターン発見後 → 段階的にスケール

## コピーライティングガイド

### コピーライティングの原則

- **キーワードをタイトル・説明文に含める**: 検索意図との関連性を高め、クオリティスコアを向上
- **複数のタイトルバリエーション**: 異なるオーディエンスセグメント・検索意図に対応する複数のタイトルを用意
- **明確なベネフィット提示**: 「〇〇が解決できる」「△△が手に入る」など具体的な価値を記載
- **強力なCTA**: 「今すぐ購入」「詳しくはこちら」「注文する」など明確な行動喚起

### 効果的なコピーパターン

| パターン | 例 |
|---------|-----|
| 問題提起 → 解決 | 「乾燥肌にお悩み?この1本で解決」 |
| 数字訴求 | 「3日で届く!送料無料キャンペーン中」 |
| 限定感 | 「本日限定50%OFF」 |
| 社会的証明 | 「10万人が選んだ〇〇」 |
| 質問形式 | 「まだ高い化粧水使ってる?」 |

### テキストオーバーレイ（動画内テキスト）

- 検索クエリに関連するキーワード/見出しを動画の最初の3秒以内に表示
- フォントは大きく読みやすいものを使用
- セーフゾーンを遵守（上部120px、下部120px、右端を避ける）

### CTAボタンの推奨

| CTA | 推奨用途 |
|-----|---------|
| 今すぐ購入（Shop Now） | 商品ページ遷移、EC |
| 詳しく見る（Learn More） | ウェブサイト・LP遷移 |
| 注文する（Order Now） | EC直接購入 |
