## 1. クリエイティブの基本原則

### 「Don't Make Ads, Make TikToks」の哲学

TikTok広告の根本的な設計思想は「広告を作るな、TikTokを作れ」。プラットフォームのコンテンツ体験と広告体験の境界を意図的に曖昧にする戦略。

- **従来型広告**: ブランドが一方的にメッセージを発信する「割り込み型」
- **TikTok広告**: ユーザーが自発的に楽しむコンテンツの中に自然に溶け込む「ネイティブ型」

### UGC vs プロフェッショナルコンテンツ

| 指標 | データ |
|------|--------|
| UGCの信頼性 | 従来のブランド制作動画と比較して**2.4倍** |
| UGC広告のクリック率 | インフルエンサーキャンペーンの**4倍** |
| TikTok上のオーガニックUGC | 2025年時点で全コンテンツの**40%** |
| UGC広告の効果 | ブランド制作動画より**22%**効果的 |
| UGCコンテンツのROI | 非UGCに比べ**+55%**向上 |
| 消費者の信頼 | **84%**が同世代のレコメンデーションを最も信頼 |
| 購買意思決定への影響 | **79%**のユーザーがUGCが強く影響すると回答 |

### ネイティブコンテンツのアプローチ

- 自然な照明、スマホ撮影感、カジュアルな語り口
- 最小限のブランディング
- ストーリーテリングとスピーチフック（冒頭の引きつけ）を重視
- プロフェッショナルな制作よりも「生っぽさ」が高パフォーマンス
- TikTokのアルゴリズムは「未加工で生々しい動画クリップ」を優遇する傾向

### エンターテインメントファーストの原則

- 冒頭1-3秒でフック（引きつけ）を設置
- ストーリーテリング構造（問題提起 → 解決策 → CTA）
- トレンドサウンド・音楽の活用
- 視聴者が「自分ごと」として感じられるリアリティ
- 過度なブランドロゴ・テロップの排除

### 縦型フルスクリーン体験の設計

| 設計原則 | 詳細 |
|---------|------|
| **9:16縦型** | フルスクリーンで没入感を最大化する唯一の正解 |
| **音声ON前提** | ユーザーの大半が音声ONで視聴。BGM・ナレーションが重要 |
| **短尺集中** | 9-15秒を推奨。最初の2秒が最重要 |
| **テキストオーバーレイ** | セーフゾーン内に配置（UI要素と重ならないよう注意） |

### クリエイティブ仕様

| 項目 | 推奨値 |
|------|--------|
| **アスペクト比** | 9:16（推奨）、1:1、16:9 |
| **動画長** | 9〜15秒（推奨）、最大60秒 |
| **解像度** | 720x1280以上（9:16の場合）、推奨1080x1920px |
| **ファイル形式** | MP4, MOV, MPEG, AVI |
| **ファイルサイズ** | 最大500MB |

## 2. フック設計（最初の3秒）

### フックの重要性

- **最初の3秒**でスクロールを止めるかどうかが決まる
- **最初の6秒間**で広告想起インパクトの**90%**が決まる
- **6秒視聴率40%以上**のクリエイティブは一般的にコンバージョンパフォーマンスが高い

### 8つのフックパターン

| パターン | 例 |
|---------|-----|
| **問いかけ型** | 「まだ〇〇で悩んでる?」「知ってた? 実は...」 |
| **呼びかけ型** | 「〇〇な人、集合!」「〇〇で困ってる人、聞いて...」 |
| **衝撃・驚き型** | 予想外のビジュアル、急展開、意外な事実の提示 |
| **ビフォーアフター型** | 劇的な変化を冒頭で見せる（結果 → 過程の順） |
| **緊張・サスペンス型** | 「これ、やばいことになった...」結果を隠して興味を引く |
| **トレンド活用型** | 流行中のサウンド・エフェクトを冒頭に配置 |
| **動き・視覚刺激型** | 素早い動き、鮮やかな色、大胆なテキスト表示 |
| **否定・逆説型** | 「〇〇は絶対やめて」「これ、間違ってます」 |

### フックの制作チェックリスト

- [ ] 最初の1秒で視覚的な変化がある
- [ ] 2〜3秒以内にバリュープロポジションが伝わる
- [ ] テキストオーバーレイで視覚的に補強している
- [ ] サウンドオンでもオフでも内容が理解できる
- [ ] フォーマットがオーガニック投稿に近い見た目になっている

## 3. 動画構成

### 基本フレームワーク: Hook → Body → CTA

| セクション | 時間配分 | 内容 |
|-----------|---------|------|
| **Hook（フック）** | 最初の3〜5秒 | 注意を引き、スクロールを止める |
| **Body（ボディ）** | 中間10〜20秒 | 価値を伝え、興味を構築し、解決策を提示 |
| **CTA（行動喚起）** | 最後の3〜5秒 | 具体的な行動を促す |

### 動画の最適な長さ

| 項目 | 推奨値 |
|------|--------|
| In-Feed広告の推奨長さ | 15〜30秒（TikTok公式: 21〜34秒が最適レンジ） |
| 最低限の長さ | 9〜15秒（オーガニックTikTokに馴染みやすい） |
| 技術的な最大長 | 60秒（In-Feed広告） |
| 最初の重要な時間 | 冒頭2〜3秒（スクロール停止の判断） |
| 広告想起への影響 | 最初の6秒間で広告想起インパクトの90%が決まる |

**ポイント:**
- 短いほどエンゲージメントは高い傾向だが、商品説明が必要な場合は15〜30秒が最適
- 60秒フルに使うことは推奨されない。冗長な動画は離脱率が高い
- 完全視聴率（VTR）がアルゴリズム評価に直結するため、尺を伸ばしすぎないことが重要

### 5つの構成パターン

**パターン1: 問題提起 → 解決**
```
[Hook] 「〇〇で困ってない?」
[Body] 問題の共感 → 解決策としての商品紹介 → 使用シーン
[CTA] 「リンクから今すぐチェック」
```

**パターン2: ビフォーアフター**
```
[Hook] 劇的なアフターを先に見せる
[Body] ビフォーの状態 → 使用過程 → アフターの詳細
[CTA] 「あなたも変わりませんか?」
```

**パターン3: チュートリアル・ハウツー**
```
[Hook] 「〇〇する方法、教えます」
[Body] ステップ1 → ステップ2 → ステップ3（商品を自然に組み込む）
[CTA] 「詳しくはプロフィールのリンクから」
```

**パターン4: 開封・レビュー型**
```
[Hook] 「これ、めっちゃ気になってたやつ届いた」
[Body] 開封 → 第一印象 → 実際に使ってみた → 正直な感想
[CTA] 「気になった人はこちらから」
```

**パターン5: ストーリーテリング（ショートドラマ）**
```
[Hook] 日常の1シーン（共感できる場面）
[Body] ストーリー展開 → 商品が自然に登場 → 解決
[CTA] さりげないブランド訴求
```

### 2025年の注目トレンド: ダイレクトショートドラマ

- UGC風広告の「見飽き」を受けて登場した新トレンド
- プロ品質の映像とダイレクトな広告訴求を組み合わせたフォーマット
- ストーリー性を持たせつつ、商品訴求を明確に行う

## 4. 音声設計

### 音声の重要性

TikTokはデフォルトでサウンドONのプラットフォーム。音声なしの広告は不完全に感じられ、スキップされやすい。

- **68%**のユーザーが音楽付きTikTokをより記憶に残ると回答

### サウンド活用の手法

| 手法 | 効果 | 注意点 |
|------|------|--------|
| **トレンドサウンド** | アルゴリズム優遇、親和性が高い | ライセンス確認が必要、トレンドの寿命が短い |
| **オリジナル音声/ナレーション** | ブランド独自性、メッセージの明確化 | 自然体で話す、台本読み感を出さない |
| **BGM + ナレーション** | 情報量とエンタメ性の両立 | 音量バランスに注意 |
| **ASMR・効果音** | 商品体験の臨場感 | 食品・美容系で特に効果的 |

### Commercial Music Library（CML）

- TikTok Creative Centerで現在トレンドのサウンドを確認可能
- 商用利用可能なサウンドライブラリを活用

### サウンドオフ対策

- テキストオーバーレイで内容が伝わるようにする
- 字幕・キャプションをリアルタイム表示

### サウンドのチェックリスト

- [ ] 動画にサウンド（音楽またはナレーション）が含まれている
- [ ] トレンドサウンドを使用する場合、商用利用可能か確認した
- [ ] ナレーションは自然な話し方になっている（棒読みNG）
- [ ] BGMとナレーションの音量バランスが適切
- [ ] サウンドオフでもテキストオーバーレイで内容が伝わる

## 5. テキストオーバーレイ

### 基本ルール

- セーフゾーン内にテキストを配置する
- フォントサイズは画面の1/3以下を占める程度
- 1画面に表示するテキスト量は最小限に
- キーメッセージは太字・大きめのフォントで強調
- 背景との明暗差を確保し、読みやすさを担保

### セーフゾーン（避けるべきエリア）

- **上部約150px**: ステータスバー等のUI要素
- **下部約170px**: キャプション・CTAボタン等のUI要素
- **右側**: インタラクションボタン（いいね、コメント等）にかぶらない
- **推奨配置**: 中央〜やや上に主要テキストを配置

### 効果的な使い方

| 用途 | 詳細 |
|------|------|
| **フックのテキスト** | 問いかけや衝撃的な一文を大きく表示 |
| **字幕・キャプション** | ナレーション内容をリアルタイム表示（サウンドオフ対策） |
| **ポイントリスト** | 商品の特徴やベネフィットを箇条書き表示 |
| **価格・割引情報** | CTAの近くにセール情報を配置 |
| **感情的な強調** | ハイライトで重要ポイントを視覚的に強調 |

## 6. CTA設計

### CTAの効果データ

| 指標 | 効果 |
|------|------|
| 広告想起 | **+45%**向上 |
| 好感度 | **+19%**向上 |
| コンバージョン率 | **+30%**向上 |
| CTAプロンプト使用率 | トップパフォーマンスのインテントキャンペーンの**80%**が使用 |

### CTA配置のパターン

| タイミング | 手法 | 適するケース |
|-----------|------|-------------|
| 動画の最後（3〜5秒） | テキスト + ボイスオーバー + CTAボタン | 商品購入、サイト誘導 |
| 動画の中盤 | さりげないテキスト表示 | 長尺動画で中間CTAを入れたい場合 |
| 動画全体を通じて | 画面下部にCTAバナーを常時表示 | 認知目的の広告 |

### 効果的なCTAフレーズ

- 「今すぐチェック」「詳しくはこちら」
- 「期間限定〇〇%OFF」「残りわずか」
- 「リンクから購入」「プロフィールから」
- 「無料で試す」「今なら送料無料」

### 動的CTA（Dynamic & Recommended CTA）

- TikTok Ads Managerが文脈に合わせてCTAを自動最適化
- 固定CTAセット + AIが生成するLPベースのCTA + 広告主が選択した候補（10-20個）から最適なものを自動選定
- エンゲージメント率・クリック率の向上に寄与

### CTAのチェックリスト

- [ ] 動画内にCTA（テキストまたは音声）が含まれている
- [ ] TikTokのCTAボタン機能を設定している
- [ ] CTAが動画の内容・トーンと一致している
- [ ] 緊急性や限定感を適切に活用している
- [ ] 具体的なアクション（購入、登録、ダウンロードなど）が明確

## 7. UGC風クリエイティブ

### 制作ガイドライン

| 要素 | UGC風にするポイント |
|------|-------------------|
| **撮影** | スマホで撮影、自然光、手持ち感 |
| **編集** | 最小限のエフェクト、TikTok内蔵機能を活用 |
| **出演者** | リアルな人物（社員、顧客、マイクロクリエイター） |
| **トーン** | 友達に話すような自然体、台本感を出さない |
| **環境** | 自宅、職場、日常のシーン |
| **音声** | 生声のナレーション、自然な反応 |

### 制作フロー

1. ターゲットに近い属性のクリエイター/社員を選定
2. 大まかな構成（Hook-Body-CTA）のみ伝え、細かいセリフは自由に
3. スマホで縦型撮影（9:16）
4. TikTokの編集機能またはCapCutで最低限の編集
5. トレンドサウンドやテキストオーバーレイを追加
6. 「広告っぽさ」がないか最終チェック

### 注意点

- 他者のUGCを広告に使用する場合、必ず権利・同意を取得（特にSpark Ads利用時）
- フォロワー10K未満のマイクロクリエイターが高いエンゲージメントを獲得しやすい
- 複数のクリエイターを起用し、異なるパーソナリティ・メッセージスタイルでテスト

## 8. クリエイター活用

### TikTok Creator Marketplace（TCM）

TikTok公式のブランド-クリエイターマッチングプラットフォーム。AIマッチングアルゴリズムにより、キャンペーン目標とオーディエンスフィットに基づいてクリエイターを推薦。

### クリエイター選定のポイント

| 基準 | 詳細 |
|------|------|
| **フォロワー数より質** | エンゲージメント品質、オーディエンスの真正性を重視 |
| **コンテンツスタイル** | ブランドのトーンとの整合性を確認 |
| **オーディエンス属性** | ターゲット層とフォロワー層の一致 |
| **過去の実績** | コラボ経験、広告パフォーマンスデータを確認 |

### インフルエンサータイプ別エンゲージメント

| タイプ | フォロワー数 | 平均エンゲージメント率 |
|--------|-----------|-------------------|
| マイクロインフルエンサー | 10K〜100K | 約**8.2%** |
| マクロインフルエンサー | 100K以上 | 約**5.3%** |

### クリエイター制作コンテンツの効果

- CTR **70%向上**
- エンゲージメント率 **159%向上**
- スポンサード投稿やレビュー形式が全体の**74%**を占める

### コラボレーションのベストプラクティス

- クリエイターにクリエイティブの自由度を与える
- 長期的なパートナーシップ（複数キャンペーン）がオーディエンスの信頼構築に効果的
- コメント、シェアなど意味のあるインタラクションを重視
- TikTokアナリティクスでキャンペーンをリアルタイムモニタリング

## 9. クリエイティブ疲労と運用

### TikTokにおけるクリエイティブ疲弊の特徴

- Facebookに比べて**4倍速く**クリエイティブ疲弊が発生
- Metaで2週間持つクリエイティブがTikTokでは**3日程度**で効果減退（高予算時）
- 数日〜数時間で疲弊が起こることも
- 2025年のアルゴリズム変更: ビジュアル類似性スコアリングで重複クリエイティブへのペナルティ強化

### 疲弊の検知シグナル

| 指標 | 疲弊の兆候 |
|------|----------|
| **フリクエンシー** | > 2.5 に到達したらリミックス・更新のタイミング |
| **CTR** | 持続的な低下 |
| **CPA** | 持続的な上昇 |
| **VTR** | 視聴完了率の低下 |
| **CVR** | コンバージョン率の低下 |

> リターゲティングオーディエンス（カート離脱者等）の場合はフリクエンシー**5〜8**まで許容

### リフレッシュサイクル

| レベル | 手法 | 頻度 |
|--------|------|------|
| **マイクロリフレッシュ** | 冒頭3秒を新しいB-Roll/テキストオーバーレイに差し替え | 3〜5日ごと |
| **マイクロリフレッシュ** | 映像速度を1.2倍に変更（視覚ペースと音声ピッチが変化） | 随時 |
| **中規模リフレッシュ** | フックを差し替え、ボディは流用 | 7〜10日ごと |
| **フルリフレッシュ** | 新規クリエイティブを3〜5本追加 | 7日ごと（疲弊検知時） |
| **完全刷新** | 新しいコンセプト・クリエイターで制作 | 2〜4週間ごと |

### ローテーション頻度の目安

| 項目 | 推奨頻度 |
|------|---------|
| 新規クリエイティブの投入 | 7日ごとに3〜5本 |
| 広告セットのローテーション | 10〜14日ごと |
| マイクロリフレッシュ | 3〜5日ごと |
| Meta広告との比較 | Metaは2〜4週間ごと、TikTokは週次更新が基本 |

### 運用のポイント

- 複数の広告バリエーションを持つ広告主はCVRが**+17%**向上
- 同時に複数のクリエイティブをテストし、勝者を特定
- パフォーマンス低下したクリエイティブは即座に停止・差し替え
- 成功パターンのバリエーションを展開（フック違い、CTA違い、クリエイター違い）

## 10. TikTok Symphony / CapCut

### TikTok Symphony（AI動画生成）

TikTokのジェネレーティブAIツールスイート。TikTok for Businessアカウントがあれば無料で利用可能。

#### 2025年の主要ツール

| ツール | 機能 |
|--------|------|
| **Image to Video** | 静止画（商品写真、ムードボード等）からTikTok風動画を自動生成 |
| **Text to Video** | テキストプロンプトから複数のキャンペーンコンセプトを生成 |
| **Showcase Products** | デジタルアバターが商品を紹介する動画を自動生成 |
| **AI Dubbing** | 多言語吹き替え |
| **Generate & Remix** | 既存クリエイティブのリミックス・バリエーション生成 |
| **Digital Avatars** | AIデジタルアバターによる商品プレゼンテーション |

#### 外部連携

- **Adobe Express連携**: Image to VideoがAdobe Express内のプラグインとして利用可能
- **WPP Open連携**: AI Dubbing、Generate & Remix、Digital Avatarsが利用可能

#### 活用シーン

- 大量のクリエイティブバリエーションが必要な場合
- 静止画しかない商品の動画広告を素早く作成
- 多言語展開のための吹き替え
- クリエイティブリフレッシュの高速化

### Symphony Automation（Smart+統合）

| 機能 | 説明 |
|------|------|

### 法規制コンプライアンス

#### 景品表示法（景表法）

- **優良誤認表示の禁止**: 品質や性能を実際よりも著しく優れていると表示することの禁止
- **有利誤認表示の禁止**: 価格や取引条件を実際よりも著しく有利と表示することの禁止
- **NG表現例**: 「日本No.1」（根拠なし）、不当な二重価格表示、「今だけ」（常時同条件）
- **2024年10月改正**: 課徴金制度の強化。再違反時は課徴金が売上額の4.5%に引き上げ

#### 薬機法（医薬品医療機器等法）

- ヘルスケア・医薬品は「制限業種」に分類
- **禁止表現**: 承認外の効果効能の訴求、ビフォーアフター形式での効果保証
- **二重規制リスク**: 薬機法 + 景表法の両方に違反する可能性あり、課徴金が二重に科される可能性

#### ステルスマーケティング規制

- 2023年10月施行のステマ規制（景表法改正）
- 「PR」「広告」「提供」等の表示が必須
- インフルエンサー起用時・Spark Ads利用時も同様に適用

### 広告審査のポイント

**よくある審査落ちの原因:**

| 原因カテゴリ | 具体例 |
|-------------|--------|
| 誇張・虚偽表現 | 「3日で痩せる」「病気が治る」「日本No.1」（根拠なし） |
| 薬機法違反 | 承認外の効果効能の訴求、ビフォーアフターでの効果保証 |
| クリエイティブ品質 | 低解像度、不明瞭な画像、音声なし、音質不良 |
| LP不一致 | 広告の商品とLPの内容が異なる、価格不一致 |
| 差別的コンテンツ | 年齢・性別・民族・宗教に関する差別的表現 |

**2025年の審査ポリシー更新:**
- AI生成コンテンツの明確なラベリングが必須
- 商業コンテンツに対する新ガイドライン（スポンサーシップの明記必須）
- 誤情報につながる可能性のあるAIコンテンツの禁止
