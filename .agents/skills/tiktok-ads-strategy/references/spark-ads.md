# Spark Ads（スパークアズ）

## 概要

既存のオーガニックTikTok投稿を広告クリエイティブとして使用するネイティブ広告フォーマット。自社アカウントの投稿、またはクリエイターが許可した投稿をブーストできる。広告経由で得たいいね・コメント・シェア・フォローはすべてオーガニック投稿に帰属し、広告停止後も残り続ける。

- **購入方式**: オークション型（運用型）。リアルタイム入札で配信。予算・ターゲティングを自由に調整可能。日予算2万〜10万円程度からスモールスタート可能
- **課金方式**: CPC, oCPM, CPM, CPV
- **費用目安**: CPC平均約$1.00、CPM $6〜7程度（2025年時点）
- **対応するキャンペーン目的**: リーチ、トラフィック、動画視聴、コミュニティインタラクション、アプリプロモーション、リード獲得、ウェブコンバージョン
- **主な差別化ポイント**: 通常のIn-Feed広告と比較して**完了率+30%、エンゲージメント率+142%、CVR+69%**。広告経由のエンゲージメントがオーガニック投稿に蓄積される。広告感が薄く信頼性が高い
- **運用上限**: 1アカウントあたり最大10,000件のSpark Adsを運用可能

## 仕様

### 動画仕様

| 項目 | 仕様 |
|------|------|
| アスペクト比 | 9:16（推奨）、1:1、16:9（オリジナル投稿に準拠） |
| 推奨解像度 | 1080 x 1920 px（9:16）。最低540 x 960 px以上 |
| 動画の長さ | 制限なし（オリジナル投稿に依存） |
| ファイル形式 | TikTokアプリ内で投稿されたものはそのまま使用 |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| キャプション | オーガニック動画のキャプションから抽出（**変更不可**） |
| 表示行数 | 最大4行（絵文字含む） |
| CTAボタン | Ads Managerで追加可能（プリセットから選択） |
| ランディングページURL | 設定可能 |

### Spark Ads特有のルール

| 項目 | ルール |
|------|-------|
| キャプション | 元投稿から変更不可 |
| 認証期間 | 7日 / 30日（デフォルト）/ 60日 / 365日から選択 |
| 一括認証 | 最大20本の動画を同時に認証可能 |
| エンゲージメント | 広告経由のいいね・コメント・シェアは投稿に蓄積 |
| 投稿元 | 自社アカウント（直接リンク）or 他者アカウント（認証コード） |
| 投稿の編集 | 広告掲載中はオーガニック投稿の編集を避けること |
| プロフィールリンク | クリエイターのプロフィールに遷移可能 |

## クリエイティブ制作ガイド

### 根本原則: オーセンティシティ

- Spark Adsは**既にオーガニックで好パフォーマンスの投稿**をブーストする形式
- 広告感が薄く、ユーザーにとって自然な体験を提供する
- いいね・コメント・シェアは**広告停止後もオーガニック投稿に残る**（最大の利点）

### ブーストすべき投稿の選定基準

| 基準 | 説明 |
|------|------|
| **オーガニックパフォーマンス** | 既に良い視聴回数・エンゲージメントを獲得している投稿 |
| **完了率** | 動画の視聴完了率が高い投稿 |
| **コメント品質** | ポジティブなコメントが多い投稿（ソーシャルプルーフとして機能） |
| **商品・サービスとの関連性** | 訴求したい商品やサービスが自然に紹介されている |
| **フックの強さ** | 冒頭3秒で視聴者を引き付ける要素がある |

### クリエイター起用のポイント

- **マイクロ・ナノインフルエンサー**（フォロワー1万-10万人）が最も効果的なケースが多い
- 「有名人」より「身近な人」の方がTikTokユーザーの信頼を得やすい
- クリエイターが**過去に使ったことのあるトレンド**を活用するよう依頼する
- 初めてのダンス動画やチャレンジ動画は不自然に見えるリスク
- クリエイターの通常のコンテンツスタイルを尊重する

### 3部構成の推奨ストーリー構造

```
1. フック（Hook）: 0-3秒
   → 視聴者の注意を引く。驚き、質問、感情的なトリガー

2. ボディ（Body）: 3-15秒
   → 価値提案を伝える。商品紹介、使用シーン、問題解決

3. クローズ（Close）: 最後の3-5秒
   → CTA。「リンクから詳しく」「プロフィールをチェック」
```

### 映像スタイル

- **DIY・ポリッシュしすぎない**スタイルがベスト
- スマホ撮影のUGC感が重要
- **縦型9:16**で撮影
- 解像度720p以上
- UIセーフゾーン内にコンテンツを配置

### クリエイティブの多様化

- 1広告グループあたり**3-5本の異なるクリエイティブ**を設定
- 1キャンペーンあたり**3-5広告グループ**で多様化
- **フックを起点にクリエイティブ全体を設計**する
- 異なるクリエイター、異なるアングル、異なるメッセージで複数バリエーション

### Do's（推奨事項）

- オーガニックで好パフォーマンスの投稿を選んでブーストする
- クリエイターの通常スタイルを尊重する
- DIY感・UGC感のあるスタイルを維持する
- 商品やサービスが自然に紹介されている投稿を選ぶ
- 複数クリエイターの投稿で多様化する

### Don'ts（禁止・非推奨事項）

- パフォーマンスが低い投稿を無理にブーストする
- クリエイターに不自然なコンテンツを強要する
- 広告掲載中にオーガニック投稿を編集する
- 認証期間の管理を怠る（期限切れで配信停止のリスク）
- キャプションが広告として不適切な投稿を選ぶ（後から変更不可）

## 広告設定ガイド

### 認証コードの設定方法（クリエイター側）

```
手順:
1. クリエイターがTikTokアプリで対象動画を開く
2. 「...」メニュー → 「広告設定」をタップ
3. 「コンテンツ開示と広告」→ 「広告認証」をオンに
4. 認証期間を選択（7日 / 30日 / 60日 / 365日）
5. 「コードを生成」をタップ
6. 生成されたコードをブランド側に共有
```

### ブランド側の設定方法

```
手順:
1. TikTok Ads Manager → ツール → クリエイティブライブラリ → Spark Ad投稿
2. 「認証申請」をクリック
3. クリエイターのコードを貼り付け → 「検索」
4. 動画プレビュー表示 → 「確認」で認証完了
5. 最大20本の動画コードを一括認証可能
```

### 自社投稿の直接リンク

- 自社TikTokビジネスアカウントの投稿は、認証コード不要で直接Spark Ads化可能
- Ads Manager内でビジネスアカウントをリンクするだけでOK

### キャンペーン設定

| 設定項目 | 推奨 |
|---------|------|
| **目的** | コンバージョン / トラフィック / アプリインストール / リーチ |
| **入札戦略** | Cost Cap（CPAが明確な場合）/ Maximum Delivery（最大配信目標） |
| **予算** | 日予算 = ターゲットCPA x 10以上 |
| **ターゲティング** | 広めに開始 → データ蓄積後に絞り込み |
| **最適化目標** | コンバージョン（推奨）/ クリック / リーチ |

### Spark Adsの特有設定

- **キャプションは元投稿のものがそのまま使用される**（Ads Managerからの編集不可）
- CTAボタンは追加可能（Shop Now、Learn More等）
- ランディングページURLは設定可能
- 投稿の認証期間が切れると広告配信が停止するため、**期間管理に注意**

### ターゲティングの推奨

| ターゲティング手法 | 推奨事項 |
|-----------------|---------|
| **類似オーディエンス** | エンゲージメントの高い既存顧客データをソースに |
| **インタレスト** | クリエイターのフォロワー層に近い興味関心を選択 |
| **ブロードターゲティング** | Smart+キャンペーンでAI最適化に任せる |
| **カスタムオーディエンス** | 最低1,000ユーザーが必要。最もCPA/CPLが低い |

### 学習フェーズの管理

- 学習フェーズは**約25結果または7日間**で安定化が始まる
- 学習フェーズ中は**ターゲティング・入札・予算の変更を避ける**（最低2日間は維持）
- 学習フェーズ中の予算調整は**40%以下の増額**に抑える
- 安定後のスケーリングは**2-3日ごとに15-20%ずつ**増額

## コピーライティングガイド

### 最重要ポイント: キャプションは編集不可

- **Spark Adsのキャプションは元のオーガニック投稿のテキストがそのまま使用される**
- Ads Managerからキャプションを変更することは不可能
- そのため、**ブースト前にオーガニック投稿のキャプションが広告として適切か確認**が必須

### クリエイターへのキャプションディレクション

| ポイント | ガイドライン |
|---------|------------|
| **トーン** | 会話調・カジュアル。営業トークNG |
| **長さ** | 最も重要な情報を**最初の100-150文字以内**に（残りは折りたたみ） |
| **ハッシュタグ** | ブランドタグ + 関連タグ2-3個 |
| **絵文字** | 適度に使用。感情やポイントを素早く伝える |
| **CTA要素** | テキスト内に行動喚起を含める（「詳しくはリンクから」「プロフィールをチェック」） |
| **ストーリーテリング** | ハードセルではなく、体験談・レビュー・発見のストーリー |

### 動画内テキストオーバーレイ

- 元のオーガニック投稿のテキストオーバーレイもそのまま使用される
- クリエイターにブリーフ段階で以下を依頼:
  - **冒頭にフックテキスト**を配置
  - **商品名やブランド名**をテキストで表示
  - **CTA的なテキスト**を動画後半に配置
  - セーフゾーン内にテキストを収める

### CTAボタン（追加設定）

- Spark AdsでもCTAボタン（Shop Now、Learn More等）は**Ads Manager側で追加可能**
- CTAボタンのテキストはプリセットから選択
- ランディングページURLも設定可能
- CTAディスプレイカードを使用すると**リコール+45%、好感度+19%**

### テキスト戦略の違い（Spark Ads vs 通常In-Feed）

| 項目 | Spark Ads | 通常In-Feed |
|------|----------|------------|
| キャプション編集 | 不可（元投稿のまま） | 自由に設定可能 |
| テキストオーバーレイ | 元投稿のまま | 自由に編集可能 |
| CTAボタン | Ads Managerで追加可能 | Ads Managerで設定 |
| トーン | クリエイターの自然なトーン | ブランドが設定 |
| 信頼性 | 高い（オーガニック感） | やや低い（広告感） |
