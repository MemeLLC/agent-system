# TopView（トップビュー）

## 概要

TikTokアプリ起動時にFor Youフィードの最上部にフルスクリーンで表示される動画広告。最初の3秒間はズームインのテイクオーバー表示（UI要素なし）、その後インフィード動画体験に移行する2段階構成。最もインパクトのあるプレミアム広告フォーマット。

- **購入方式**: 予約型（リザベーション）のみ。TikTok営業担当経由で事前予約が必要。オークション入札は不可。インプレッションは完全保証型
- **課金方式**: CPM（予約型）
- **費用目安**: $50,000〜$150,000/日（$65,000/日〜が一般的）、CPM $10〜20
- **対応するキャンペーン目的**: 認知（Reach）
- **主な差別化ポイント**: アプリ起動時のフルスクリーン表示で最大インパクト。ブランド想起が他の広告タイプの1.5倍。サウンドオンがデフォルト。CTR 12-16%の高パフォーマンス
- **$65,000日予算で約740万インプレッション**を獲得可能

## 仕様

### 動画仕様

| 項目 | 仕様 |
|------|------|
| アスペクト比 | 9:16（縦型のみ） |
| 推奨解像度 | 1080 x 1920 px以上 |
| 最小解像度 | 720 x 1280 px（540 x 960 pxまで対応） |
| 動画の長さ | 5〜60秒（推奨: 9〜15秒） |
| ファイル形式 | .mp4, .mov, .mpeg, .avi |
| ファイルサイズ | 最大500 MB |
| ビットレート | 2,500 kbps以上（推奨） |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| キャプション | 1〜100文字（In-Feedと同一） |
| キャプション内リンク | 非対応 |
| CTAボタン色 | 動画コンテンツから自動マッチング（カスタム不可） |
| CTAテキスト | プリセット選択式 |
| CTA言語 | ユーザーのアプリ言語に自動翻訳 |

### 2段階のセーフゾーン

| ステージ | セーフゾーン |
|---------|------------|
| **テイクオーバー（0-3秒）** | UI要素なし。画面全体を使用可能 |
| **In-Feed遷移（3秒以降）** | 通常のIn-Feedセーフゾーンが適用（1080x1920pxの場合、上部・下部各120pxがUI被りゾーン） |

### 表示の仕組み

- 最初の3秒: フルスクリーンのズームインテイクオーバー（画面全体にフィット）
- 3秒以降: インフィードコンテンツとしてTikTok UIと共に表示
- リプレイ時はインフィード表示サイズに適応

### 購入・配信ルール

| 項目 | ルール |
|------|-------|
| 購入方式 | 予約型（Reserved）のみ |
| インプレッション | 完全保証 |
| 事前承認 | 必須。提出後2日以内にレビュー |
| クリエイティブ排他性 | 同日に他の枠で同じクリエイティブ使用不可 |
| 100% SOV | 前日・翌日も同クリエイティブ使用不可 |

## クリエイティブ制作ガイド

### 2段階構成を理解する

```
ステージ1: テイクオーバー（0-3秒）
├── フルスクリーン、UI要素なし
├── ズームインエフェクト
├── キャプション・ボタン表示なし
└── 最もインパクトのある瞬間

ステージ2: In-Feed遷移（3秒以降）
├── 通常のIn-Feed形式に遷移
├── キャプション、CTA、エンゲージメントボタン表示
├── UIセーフゾーンの制約適用
└── スクロール可能になる
```

### ステージ1（テイクオーバー）のクリエイティブ

- **最初の3秒が全て**。キャプションもボタンもない純粋なフルスクリーン
- **映像で圧倒する**: ミニスーパーボウルCMのようなインパクトを目指す
- ブランドの世界観を凝縮した映像美で勝負
- 動きのある映像が効果的（静止画的な出だしはスキップされやすい）
- **サウンドオンがデフォルト**のため、冒頭から音声で引き込む

### ステージ2（In-Feed遷移後）のクリエイティブ

- In-Feedセーフゾーンのルールが適用される
- キャプション・CTAボタンが表示されるため、**UI被りを考慮した配置**
- メッセージの深掘り、商品詳細、CTAへの誘導を行う

### 映像制作のポイント

- **9:16の縦型のみ**対応（横型・正方形は不可）
- 推奨解像度: 1080x1920px以上
- ビットレート: **2,500kbps以上**（テイクオーバー時のクリスプな品質確保）
- プロフェッショナルな映像品質が求められる（In-Feedと異なりUGC感は不要）
- **ブランドの顔となる映像**として制作する

### 音声・サウンド設計

- **サウンドオンがデフォルト**の数少ないフォーマット
- 冒頭1秒目から**魅力的なオーディオ**を設計
- ブランドのサウンドロゴやジングルの活用が効果的
- ナレーション、BGM、効果音のバランスを緻密に設計
- 音声がない状態でも理解できるよう、テキストオーバーレイも併用

### 推奨動画尺

- **9-15秒**が最も効果的
- 短すぎるとメッセージが伝わらない、長すぎるとスキップされる
- 最大60秒まで対応しているが、15秒以内が推奨

### Do's（推奨事項）

- 最初の3秒に圧倒的なビジュアルインパクトを配置する
- サウンドオンを前提とした音声設計をする
- プロフェッショナルな映像品質で制作する
- 両ステージのセーフゾーンに対応したデザインにする
- 9-15秒の推奨尺に収める

### Don'ts（禁止・非推奨事項）

- 横型・正方形の映像を使用する
- UGCスタイルの低品質映像を使用する（TopViewではプレミアム感が重要）
- 静止画的な出だしにする（スキップされやすい）
- 同日に他の広告枠で同じクリエイティブを使用する
- ゲーム広告でスクリーンレコーディングのみの動画を使用する
- 不快感を与えるシーンや暴力シーンのみの動画を使用する

## 広告設定ガイド

### 購入方式

- **予約型（Reserved）のみ**。オークション入札は不可
- TikTokの営業チームとの直接交渉・事前予約が必要
- インプレッションは**完全保証型**

### 費用

| 項目 | 目安 |
|------|------|
| 日額費用 | $50,000〜$150,000 |
| CPM | $10〜$20 |
| 参考: $65,000日予算 | 約740万インプレッション |

### 最適な使用シーン

- **大型商品ローンチ**: 新商品・新サービスの発表時
- **ブランドキャンペーン**: テントポールイベント（季節商戦、周年記念等）
- **映画・エンタメプロモーション**: 公開日の告知
- **大規模セール告知**: ブラックフライデー、年末商戦等
- ブランド想起を最大化したい**認知度向上キャンペーン**

### キャンペーン計画のポイント

- **事前予約が必須**。人気の日程は早めに確保
- 広告アセットは**事前承認制**。提出後2日以内にTikTokがレビュー
- **同じクリエイティブは同日に他の広告枠で使用不可**
- 100% SOV TopViewの場合、前日・翌日も同じクリエイティブは使用不可

### 他フォーマットとの組み合わせ

| 組み合わせ | 効果 |
|-----------|------|
| **TopView + In-Feed Ads** | 認知→リターゲティングのフルファネル |
| **TopView + Branded Hashtag Challenge** | 認知→エンゲージメント→UGC生成 |
| **TopView + Spark Ads** | 認知→オーガニックコンテンツによる信頼構築 |

### KPI設定

- TopViewの主要KPIは**認知度・ブランド想起・リーチ**
- TikTokユーザーは**TopViewからのブランド想起が他の広告タイプの1.5倍**
- 直接的なCVRよりもブランドリフト調査との組み合わせで効果測定

## コピーライティングガイド

### テイクオーバー時（0-3秒）のテキスト

- **キャプション・CTAは表示されない**。映像とサウンドのみ
- この3秒間で伝えるメッセージはテキストオーバーレイで対応
- **ブランド名/商品名のテキストオーバーレイを冒頭に配置**
- シンプルで大胆なメッセージ（例: 「新登場」「ついに日本上陸」）
- テイクオーバー時: **大きく、太く、短いテキスト**

### In-Feed遷移後のキャプション

- **TopViewのキャプションにはリンクを含めることができない**
- キャプション文字数制限は通常のIn-Feed Adsと同様（1-100文字）
- 短く、ブランドメッセージを明確に
- 商品名・キャンペーン名を含める

### CTAボタン

- CTAボタンの背景色は動画コンテンツから**自動マッチング**（カスタム不可）
- CTAテキストの完全カスタマイズは不可。プリセットから選択
- CTAテキストはユーザーのアプリ言語に合わせて**自動翻訳**される
- 翻訳が利用できない言語の場合は英語で表示

### テキストオーバーレイの推奨

| ステージ | ガイドライン |
|---------|------------|
| テイクオーバー時（0-3秒） | 大きく、太く、短いテキスト。ブランド名・商品名を明確に |
| In-Feed遷移後（3秒以降） | 通常のIn-Feed Adsと同様のルール（5-10語/秒） |

- ブランドロゴはセーフゾーン内に配置
- 音声オフでも理解できるよう、キーメッセージはテキストで補完
- **センター左寄りに配置**推奨

### ランディングページ

- **WebView URL**にリンク可能（アプリ内ブラウザで開く）
- httpsプロトコル必須（httpは非推奨、リダイレクト失敗のリスク）
- モバイルデバイス（Android/iOS両方）に最適化されたLPを用意
- LP読み込み速度を一定に保つ
- **広告のビジュアル要素をLPのファーストビューにも反映**させ、遷移のシームレスさを確保
