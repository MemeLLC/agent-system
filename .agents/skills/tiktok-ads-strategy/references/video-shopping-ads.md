# Video Shopping Ads / VSA（動画ショッピング広告）

## 概要

動画形式のショッピング特化型広告。商品ページへの直接遷移が可能で、EC事業者に人気の高いフォーマット。2023年4月にCollection AdsとDynamic Showcase Adsの機能を統合した後継フォーマットとして誕生。TikTok Shop内で購入が完結するため、シグナルロスがなく正確な計測が可能。

- **購入方式**: オークション型（運用型）
- **課金方式**: oCPM
- **対応キャンペーン目的**: 販売（Product Sales）
- **主な差別化ポイント**: TikTok Shop連携、アプリ内購入完結、シグナルロスなし
- **パフォーマンス実績**: セール情報を含む広告はROASが48%向上

> **重要: GMV Max移行スケジュール**
> - 2025年6月1日: 新規TikTok Shop出店者は従来型広告の作成不可
> - 2025年6月25日: 従来型広告機能の完全廃止
> - 2025年7月15日: 全出店者がGMV Maxに移行完了
> - 2025年8月5日: グローバル施行。VSA/PSA/LSAの新規作成・編集・複製が不可に
> - 既存のVSAは配信完了まで継続するが、編集・複製は不可
> - 移行方法: Ads Manager内の「Upgrade now」通知からワンクリック移行。過去のROI実績をターゲットROIとして引き継ぎ
> - 対象地域: インドネシア、マレーシア、フィリピン、タイ、ベトナム、シンガポール、米国

## 仕様

### 動画仕様

| 項目 | 仕様 |
|------|------|
| アスペクト比 | 9:16（推奨） |
| 推奨解像度 | 1080 x 1920 px |
| ファイル形式 | .mp4、.mov、.mpeg、.3gp、.avi |
| 最大ファイルサイズ | 500MB |
| 推奨動画の長さ | 10〜25秒（最適）、最大60秒 |
| 音声 | 必須（音声ON環境がデフォルト） |

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| 広告キャプション | 英語: 最大100文字 / 日本語等: 最大50文字 |
| 商品名表示 | 最適表示は約34文字以内 |
| 商品タイトル（カタログ） | 最大150文字 |
| 商品説明（カタログ） | 最大20,000文字 |

### 商品画像仕様（カタログ）

| 項目 | 仕様 |
|------|------|
| アスペクト比 | 1:1 または 3:4 |
| 最小解像度 | 500 x 500 px（推奨720px以上） |
| 画像リンク | 最大10画像/商品 |
| 動画リンク | 対応（カタログフィード内） |

### カタログ（商品フィード）仕様

| 項目 | 仕様 |
|------|------|
| 必須フィールド | 9項目（商品ID、タイトル、説明、URL、画像URL、価格、在庫状況など） |
| オプションフィールド | 27項目 |
| フィード形式 | CSV、XML |
| 更新頻度 | 毎日更新を推奨 |
| 商品タイトル | 最大150文字 |
| 商品説明 | 最大20,000文字 |

### 商品設定

| 項目 | 仕様 |
|------|------|
| 商品リンク | Product Anchor Linkがデフォルト（高いCVR・CTR実績） |
| 商品選択 | 1〜20商品を手動選択、または「全商品」を自動選択 |
| 商品情報同期 | TikTok Shopから自動同期 |

### 配信ルール

| 項目 | ルール |
|------|--------|
| 配信面 | TikTokのみ（Pangle等の外部プレースメント不可） |
| 年齢制限 | 18歳以上 |
| 商品数 | 最大20商品（手動選択）または全商品（自動選択） |
| シグナルロス | なし（TikTok Shop内で完結するため正確な計測が可能） |

## クリエイティブ制作ガイド

### 基本原則

- **オーセンティシティ重視**: TikTokユーザーは磨き上げたプロ映像よりもリアルなコンテンツを好む。教育・エンターテインメント・インスピレーションを意識
- **Spark Adsの活用**: 既にオーガニックでコンバージョンを生んでいる動画をSpark Adsとして広告利用するのが最も効果的
- **フックに90%の注力**: 最初の数秒で視聴者の注意を引くことに注力し、残り10%で残りの広告を最適化

### クリエイティブの量と更新

| 項目 | 推奨 |
|------|------|
| 1広告グループあたりの動画数 | 3〜5本（最低） |
| クリエイティブ更新頻度 | 週1回、またはCTR/エンゲージメント低下時 |
| GMV Max移行時の推奨動画数 | 50〜70本/回、または毎日5〜10本の新規動画 |

### 動画の長さ

- 推奨: 10〜25秒（最も高いパフォーマンスを発揮）
- 短尺: 5〜15秒でも効果的
- 最大60秒（推奨範囲外だが設定可能）

### 効果的なクリエイティブパターン

| パターン | 説明 |
|---------|------|
| 商品デモ | 実際の使用シーンを見せる |
| UGC風レビュー | ユーザーが実際に使っている様子 |
| アンボックス | 開封動画で商品の魅力を伝える |
| ビフォー/アフター | 使用前後の変化を見せる |
| 問題解決型 | 課題を提示し、商品で解決 |

### クリエイティブチェックリスト

- 最初の3秒で視聴者の注意を引くフックがあるか
- 音声/音楽が含まれているか（TikTokは音声ONがデフォルト）
- 商品とクリエイティブの関連性が高いか
- 明確なCTAが含まれているか
- 縦型9:16フルスクリーンで制作しているか
- セーフゾーン（下部20%、右側10%）に重要な情報を配置していないか

### やるべきこと

- 3〜5本以上の動画クリエイティブを用意する
- Spark Adsで既存の高パフォーマンスオーガニック投稿を活用する
- 週1回のクリエイティブ更新サイクルを維持する
- オーセンティックでリアルなコンテンツを制作する

### やってはいけないこと

- プロ映像に過度にこだわらない（リアルさ重視）
- セーフゾーンに重要情報を配置しない
- 音声なしの動画を使用しない
- クリエイティブ疲れを放置しない

## 広告設定ガイド

### キャンペーン目的

- **Product Sales（商品販売）** 目的で、TikTok Shopを販売先として設定

### 最適化目標

| 最適化目標 | 説明 | 推奨ケース |
|-----------|------|-----------|
| 総収益（Gross Revenue） | 広告経由の総売上を最大化 | スケール重視 |
| 購入（Purchase） | 購入件数を最大化 | コンバージョン重視 |
| チェックアウト開始（Checkout Initiation） | チェックアウト開始数を最大化 | ファネル中間最適化 |
| クリック（Click） | クリック数を最大化 | 認知・トラフィック重視 |

### 入札戦略

| 戦略 | 説明 | 推奨 |
|------|------|------|
| 最高総収益（Highest Gross Revenue） | 予算内で最大の総収益を目指す | スケール時のメイン戦略 |
| コストキャップ（Cost Cap） | 目標CPAを設定し、コスト管理 | 効率管理時 |

- 2つの入札戦略を組み合わせて実験することを推奨
- 最高総収益の広告グループは、過去7日間のROASをローリングベースで観察しながら毎週予算をスケール

### 予算のスケーリング

- 急激な予算増は避ける
- 週単位で段階的にスケール
- 毎回スケール後、7日間のROASを確認してから次のスケールを判断
- パフォーマンスが安定するまで日予算の大幅変更を避ける

### シングルリンク vs マルチリンク

- シングル商品リンクとマルチ商品リンクの両方をテストし、自社に最適な方法を見つける
- 結果はセラーごとに異なるため、一律の正解はない

### 予算要件

- 最低日予算: キャンペーン $50/日、広告グループ $20/日

### GMV Maxへの移行準備

GMV Maxの主な特徴:
- TikTokフィード、検索、TikTok Shopタブ、Pangleの全配信面を自動活用
- 動画・商品カードの両形式で自動配信
- クリエイティブの自動生成・停止、最適なアセットの選択、オーディエンス設定の自動最適化
- パフォーマンスがROI目標の90%を下回った場合、一部の支出が自動クレジットバック
- GMV Maxでは50〜70本/回、または毎日5〜10本の新規動画の投入を推奨
- 初期テストでGMVが30%向上（TikTok内部データ）

## コピーライティングガイド

### コピーライティングのコツ

- **商品の価値を端的に**: 限られた文字数で最も訴求力のあるベネフィットを伝える
- **価格・割引情報**: セール情報を含む広告はROASが48%向上するデータあり
- **緊急性の演出**: 「期間限定」「残りわずか」「本日最終日」
- **社会的証明**: 「〇〇万個突破」「レビュー4.8」

### CTA（Call to Action）

- Product Anchor Linkがデフォルトで、商品ページへの直接遷移
- TikTok Shop内での購入完了が可能（アプリ離脱なし）
- 「今すぐ購入」「商品を見る」などの明確なCTAボタン

### 動画内テキストオーバーレイ

- 商品名・価格・割引率を動画内に表示
- テキストは大きく読みやすいフォントで
- セーフゾーンを必ず遵守

### 商品カタログのコピー最適化

- **商品タイトル**: 簡潔で検索性の高いタイトル。最適表示は約34文字以内
- **商品説明**: 特徴・素材・サイズ・使用シーンなど詳細情報を記載（最大20,000文字）
- **価格情報**: 正確な価格・セール価格を常に最新の状態に維持
