# Brand Takeover（ブランドテイクオーバー）

## 概要

ユーザーがTikTokアプリを開いた瞬間に全画面で表示される3〜5秒の短尺広告。動画または静止画に対応。1日1カテゴリにつき1広告主のみが利用できるため、最大限の独占的露出を実現する。

- **購入方式**: 予約型（リザベーション）のみ。TikTok営業担当経由で事前予約が必要。オークション入札は不可。1カテゴリ1日1広告主の独占枠（カテゴリエクスクルーシビティ）
- **課金方式**: CPM（予約型）/ 日額固定
- **費用目安**: $50,000〜$100,000/日、CPM $20〜50
- **対応するキャンペーン目的**: 認知（Reach）
- **主な差別化ポイント**: カテゴリ独占で競合を完全排除。動画と静止画の両方に対応。広告全体がクリッカブル。CTR 7-10%の高パフォーマンス。サウンドオンがデフォルト
- **キャプション（テキスト説明文）は追加不可** - 全てのメッセージは映像/画像内のテキストオーバーレイで伝える

## クリエイティブ制作ガイド

### 3-5秒で全てを伝える設計

- **3秒は驚くほど短い**。メッセージは1つに絞る
- 「新商品発売」「今すぐダウンロード」「〇〇キャンペーン開催中」など、**ワンメッセージ**
- 複雑な説明は不要。シンプルで大胆なビジュアルと短いコピー
- 例: 「新しいiPhoneが登場」+ プロダクトショット、程度のシンプルさ

### フォーマット別の制作ポイント

#### 動画（3-5秒）の場合

- 冒頭1秒で**ブランド/商品を明確に表示**
- 動きのある映像で注目を集める
- サウンドオンがデフォルト。魅力的な音声設計
- 圧縮に耐えるクリスプな画質（ビットレート2,500kbps以上）
- ブランドカラーを効果的に使用

#### 静止画（3秒表示）の場合

- **太字のプロダクトショット + クリーンなロゴ + パンチのあるタグライン**
- 複雑なイメージは圧縮で潰れる。シンプルなデザインを徹底
- 高解像度画像を使用
- テキスト量は最小限に

#### GIF（3-5秒）の場合

- 動画ほどの表現力はないが、静止画より動的なインパクト
- ループアニメーションで注目を維持
- ファイルサイズに注意

### デザインの鉄則

- **シンプルさが命**: 1つのビジュアル、1つのメッセージ、1つのCTA
- ブランドのアイデンティティを明確に表現
- TikTokのUI要素を模倣するデザインは禁止
- テイクオーバー時はUI被りなし。画面全体を活用
- 高品質な映像・画像素材を使用（圧縮に耐える品質）

### Do's（推奨事項）

- メッセージを1つに絞る
- 3-5秒で完結する内容にする
- ブランド/商品を明確に認識できるようにする
- シンプルで大胆なデザインにする
- サウンド設計をする（動画の場合）
- 高品質な映像・画像素材を使用する

### Don'ts（禁止・非推奨事項）

- 複数のメッセージを詰め込む
- 複雑な説明を入れる
- 低品質・低解像度の素材を使用する
- TikTokのUIを模倣するデザインにする
- 不快感を与えるシーンや暴力シーンを使用する
- 事前承認なしで配信しようとする

## 広告設定ガイド

### 購入方式

- **予約型（Reserved）のみ**。オークション入札は不可
- TikTok営業チームとの直接交渉・事前予約が必要
- **1カテゴリ1日1広告主の独占枠**（カテゴリエクスクルーシビティ）

### 費用

| 項目 | 目安 |
|------|------|
| 日額費用 | $50,000〜$100,000 |
| CPM | $20〜$50 |
| 独占性 | 1カテゴリ1日1広告主 |

### 最適な使用シーン

- **大型商品ローンチ**: 新商品発表の初日
- **映画・ゲームのリリース日**: 公開日の大量リーチ
- **大規模セール告知**: セール開始日の最大認知
- **ブランドイベント**: 周年記念、テントポールモーメント
- **短くシンプルなメッセージ**で十分なキャンペーン

### 他フォーマットとの組み合わせ戦略

| 組み合わせ | フロー | 効果 |
|-----------|--------|------|
| **Brand Takeover + Hashtag Challenge** | テイクオーバーで認知→チャレンジで参加 | 認知→エンゲージメント→UGC生成 |
| **Brand Takeover + In-Feed Ads** | テイクオーバーで認知→In-Feedでリターゲティング | フルファネルカバレッジ |
| **Brand Takeover + TopView** | 同日に両方で最大インパクト | 圧倒的な認知度獲得（超高予算向け） |

### 遷移先の設定

- **外部ランディングページ**にリンク可能
- **TikTok内の遷移先**: ブランドプロフィール、Hashtag Challengeページ等
- 広告全体がクリッカブル（画面のどこをタップしても遷移）
- LP/遷移先はモバイル最適化必須

### KPI設定

- 主要KPI: **リーチ、インプレッション、ブランド想起、広告認知度**
- カテゴリ独占のため、**そのカテゴリ内での競合排除効果**も重要
- ブランドリフト調査との組み合わせで効果測定推奨

## コピーライティングガイド

### 最重要ポイント: テキスト追加不可

- **Brand Takeover広告にはキャプション（テキスト説明文）を追加できない**
- そのため、**全てのメッセージは映像/画像内のテキストオーバーレイで伝える**必要がある

### テキストオーバーレイの設計

| ポイント | ガイドライン |
|---------|------------|
| **文字数** | 最小限。10語以内が理想 |
| **フォントサイズ** | 大きく、太く、読みやすく |
| **配置** | 画面中央。視線が自然に集まる位置 |
| **コントラスト** | 背景映像とのコントラストを確保 |
| **メッセージ** | ワンメッセージ。1つの行動喚起 |
| **ブランド名** | 必ず含める。ロゴも併用 |

### 効果的なメッセージパターン

| パターン | 例 |
|---------|---|
| **新発売告知** | 「新登場 〇〇」「ついに発売」 |
| **セール告知** | 「本日限定 50%OFF」「ブラックフライデー開催中」 |
| **アプリ誘導** | 「今すぐダウンロード」「無料で始めよう」 |
| **イベント告知** | 「〇月〇日 開催」「チャレンジに参加しよう」 |
| **ブランドメッセージ** | ブランドスローガン + ビジュアル |

### CTA設計

- 広告全体がクリッカブルなため、**暗示的なCTAが効果的**
- 「タップして詳しく」「スワイプして見る」のようなテキスト
- CTAボタンの追加は可能（プリセットから選択）
- **ビジュアルCTA**: 矢印やアニメーションで「タップ」を促す

### 3秒以内で読み切れるテキスト設計

- テキストは**3秒以内で読み切れる文字数**に制限する
- 日本語の場合、**15-20文字以内**が目安
- ブランド名 + ワンメッセージ + CTA的要素を最大限シンプルに配置
- フォントサイズは十分に大きく、背景とのコントラストを確保する

### LP/遷移先のコピー

- Brand Takeover→LPの遷移はシームレスにする
- 広告内のメッセージとLPのファーストビューのメッセージを一致させる
- LPでの詳細説明・CVR最適化はLP側で行う

| CTR | 7〜10% |

### Brand Takeover vs TopView 比較

| 項目 | Brand Takeover | TopView |
|------|---------------|---------|
| 表示タイミング | アプリ起動直後 | アプリ起動直後 |
| 動画尺 | 3-5秒 | 5-60秒（推奨9-15秒） |
| 静止画対応 | あり（JPG, PNG） | なし |
| キャプション | なし | あり（1-100文字） |
| アスペクト比 | 9:16, 1:1, 16:9 | 9:16のみ |
| カテゴリ独占 | あり（1カテゴリ1社/日） | なし |
| 費用（日額） | $50K-$100K | $50K-$150K |
| CPM | $20-$50 | $10-$20 |
| CTR | 7-10% | 12-16% |
| 最適用途 | 短いインパクト重視の認知 | ストーリーテリング込みの認知 |

### Brand Takeoverの選択基準

- **Brand Takeoverを選ぶべき場合**: メッセージがシンプルで3-5秒で完結する、カテゴリ独占による競合排除が重要、静止画クリエイティブを使いたい
- **TopViewを選ぶべき場合**: ストーリーテリングが必要（9-15秒）、CTRの最大化が重要（TopViewの方がCTR高い）、キャプションでテキスト情報を補足したい
