## 5. 楽天独自の仕組み・制度

### SALEサーチ申請

| 種類 | 必要割引率 |
|------|-----------|
| 半額サーチ | 50%以上の割引 |
| 通常サーチ | 10%以上の割引 |

- 開始4週間前に販売実績が必要
- 通常価格での実績は2週間以上
- 申請締切はSALE開始の1〜2週間前
- 掲載で通常の**20倍以上**のインプレッション

### 最強翌日配送（2024年7月導入）

**店舗基準**: 納期遵守率96%以上 / 6日以内お届け80%以上 / 月間出荷100件以上

**商品基準**: 午前注文→365日翌日届け / 午後注文→365日翌々日届け

**メリット**: 転換率**約9%上昇**（楽天調査）、SEO好影響、検索結果にラベル・お届け予定日表示

### RSL（楽天スーパーロジスティクス）

| 項目 | 内容 |
|------|------|
| 保管料 | 7.5円/月/PCS × 体積(cm3)÷1,000 × 保管日数÷当月日数 |
| 出荷作業料（小型） | 65円/ピース〜 |
| 配送料（ポスト投函） | 14〜24円 |
| 配送料（宅配便） | 24〜62円（サイズ別） |
| メリット | 最強翌日配送対応、転換率最大2.5倍、365日出荷 |

### 検索アルゴリズム主要要因

| 順位 | 要因 | 重要度 |
|------|------|--------|
| 1 | 売上実績 | 非常に高い |
| 2 | キーワードの関連性 | 非常に高い |
| 3 | レビュー（件数・評価） | 高い |
| 4 | 転換率（CVR） | 高い |
| 5 | クリック率（CTR） | 高い |
| 6 | 配送品質 | 中〜高 |
