## 1. 商品登録の必須項目と推奨項目

### 必須項目

| 項目 | 制限 |
|------|------|
| 商品管理番号 | 半角英数字、ハイフン、アンダースコア |
| 商品名 | 全角127文字以内 |
| 販売価格 | 税込価格 |
| 商品画像（1枚目） | ガイドライン準拠必須 |
| 在庫数 | 数値 |
| ジャンルID | 楽天カテゴリツリーから選択 |
| 商品属性情報 | カテゴリ別の必須属性あり |
| 配送方法・送料 | 事前設定が必要 |

### 検索対象項目（デバイス別）

| 項目 | PC検索 | スマホ/アプリ検索 |
|------|--------|------------------|
| 商品名 | 対象 | 対象 |
| キャッチコピー（PC用） | 対象 | **対象外** |
| キャッチコピー（モバイル用） | **対象外** | 対象 |
| PC用販売説明文 | 対象 | **対象外** |
| PC用商品説明文 | 対象 | **対象外** |
| スマホ用商品説明文 | **対象外** | 対象 |
| 商品属性 | 対象 | 対象 |

> **重要**: 重要キーワードはPC用・モバイル用の両方に入れること。

## 2. 商品名の最適化

### 基本ルール

| 項目 | 詳細 |
|------|------|
| 文字数上限 | **全角127文字** |
| 推奨文字数 | **30〜60文字** |
| スマホ表示 | 検索結果で**約30文字**まで |
| 最重要部分 | **冒頭19文字**（クリック率に直結） |

### 構成テンプレート

```
[ブランド名] [メインキーワード] [特徴/スペック] [サイズ/容量] [ターゲット/用途] [補足KW]
```

**具体例:**
```
OK: 【送料無料】国産 有機抹茶パウダー 100g 無添加 オーガニック 粉末 京都産 製菓用
NG: ★★超お買い得★★限定!!抹茶抹茶抹茶パウダー粉末茶
```

### ベストプラクティス

- **左寄せ**: 最重要キーワードを先頭に配置
- **冒頭19文字**: ブランド名+メインキーワードを必ず含める
- **固有情報**: ブランド名、型番、数量、サイズ、カラーを漏れなく
- **過度な装飾禁止**: 【】の乱用、無関係KW詰め込みは自粛要請（2024年秋改定）
- **重複回避**: 同じキーワードを繰り返さない

## 3. キャッチコピーの書き方

### 文字数制限

| デバイス | 上限 | 実際の表示 |
|----------|-----|----------|
| PC用 | 全角87文字 | 全文表示 |
| モバイル用 | 全角87文字 | 約30文字 |

### 構成テンプレート

**PC用**: `[訴求ポイント] [キーワード補完] [限定感/特典] [ターゲット]`

```
【ポイント10倍】楽天ランキング1位獲得！累計10万個突破 30代40代 敏感肌 保湿クリーム
```

**モバイル用**: 最初の30文字に最重要情報を凝縮

```
【送料無料】ランキング1位 累計10万個突破の保湿クリーム
```

> **重要**: 重要キーワードはPC用・モバイル用の**両方**に入れる。

## 4. 商品説明文の最適化

### 種類と検索対象

| 種類 | HTML | 検索対象 |
|------|------|---------|
| PC用商品説明文 | 可 | PC検索のみ |
| スマホ用商品説明文 | 可 | スマホ検索のみ |
| PC用販売説明文 | 可 | PC検索のみ |

### PC用商品説明文の推奨構成

1. ベネフィット（なぜこの商品が必要か）
2. 特徴・スペック
3. 使用方法・活用シーン
4. 素材・成分・原材料
5. サイズ表・仕様一覧
6. お客様の声
7. FAQ
8. 注意事項・返品ポリシー
9. 関連商品リンク

### スマホ用の最適化ポイント

- **簡潔さ**: 箇条書きやアイコン活用
- **画像中心**: テキストより商品説明画像
- **ファーストビュー**: 最重要情報を最初に
- **タップしやすいリンク**: ボタン64px以上
- **読み込み速度**: 画像最適化

## 5. 商品画像のガイドライン

### 基本仕様

| 項目 | 規格 |
|------|------|
| ファイル形式 | JPEG、PNG、GIF（アニメ不可） |
| 推奨サイズ | 1,000 x 1,000px以上（ズーム対応） |
| ファイル容量 | 1枚5MB以下（推奨200KB） |
| 登録可能枚数 | 最大20枚 |
| アスペクト比 | 正方形（1:1）推奨 |

### サムネイル（1枚目）の4原則

| 原則 | 詳細 |
|------|------|
| テキスト占有率20%以内 | ロゴ・マーク・イラストもカウント対象 |
| 白背景または写真背景 | #FFFFFF。合成背景もOK（2024年6月改定） |
| 枠線の禁止 | 画像周囲に枠線配置不可 |
| 商品以外の装飾禁止 | 過度なテキスト装飾は禁止 |

### テキスト占有率の計算

画像を10x10の100マスに分割し、テキスト要素が占めるマスが20マス以内。商品自体に印字されたテキストやパッケージ上のテキストは**カウント対象外**。

### 2枚目以降の推奨構成

| 画像番号 | 推奨内容 |
|----------|---------|
| 2枚目 | ディテール・質感のアップ |
| 3枚目 | 使用シーン・利用イメージ |
| 4枚目 | サイズ感比較 |
| 5枚目 | カラーバリエーション |
| 6枚目 | スペック・インフォグラフィック |
| 7枚目 | お客様の声 |
| 8枚目 | 素材・成分詳細 |
| 9枚目 | 付属品・セット内容 |
| 10枚目〜 | FAQ、注意事項、配送情報 |

## 6. SKU移行後の商品管理

### 従来型との違い

| 項目 | 従来 | SKU移行後 |
|------|------|----------|
| バリエーション | 項目選択肢別在庫 | SKU単位管理 |
| 価格設定 | 1ページ1価格 | **SKUごとに異なる価格** |
| レビュー | 別ページなら分散 | **同一ページに統合** |
| 売上実績 | 別ページなら分散 | **同一ページに統合**（SEO効果向上） |

### SKU移行で必要な対応

- **商品属性情報の登録**: 旧タグIDが「商品属性」に変更。必須属性は必ず登録
- **バリエーション統合**: 別ページの色・サイズ違いを1ページにまとめる
- **SKU画像の設定**: 各SKUに対応画像を紐付け
- **SKU別在庫・価格管理**: SKU単位での管理に切り替え

### 現状（2024年時点）

- 必須属性の登録完了店舗: 約34%
- 「最強配送」ラベル獲得条件にSKU対応が含まれる

## 7. カテゴリ・タグID設定

### ディレクトリID（ジャンルID）

- 1商品につき1つ
- **可能な限り深い階層（末端カテゴリ）を選択**

```
NG: トップス
OK: トップス > Tシャツ・カットソー > 半袖
```

### 商品属性情報

- 必須属性: カテゴリごとに楽天指定
- 推奨属性: 設定すると絞り込み検索での露出が増加
- **必須属性は100%登録**が検索表示に有利

## 8. 楽天SEOのアルゴリズム要因

### 4つの評価軸（2024年10月発表）

| 評価軸 | 重要度 |
|--------|--------|
| 商品情報の品質（KW最適化・属性充実度・画像品質） | 非常に高い |
| 売上実績（直近の売上金額・件数） | 非常に高い |
| ユーザー行動データ（CTR・CVR・カートイン率） | 高い |
| 顧客評価（レビュー件数・評価） | 高い |

### 2024年1月導入 セマンティックサーチ

- ディープラーニングによるユーザー検索意図の解析
- キーワード単純一致 → 意味的関連性の評価へ
- **キーワード詰め込みよりも商品との関連性が重要に**

### アルゴリズムの特性

- **ハイブリッド型**: 静的設定情報 + 動的ユーザー行動データ
- **デバイス別最適化**: PC/スマホで検索対象項目が異なる
- **最強配送ラベル**: 2024年7月開始。SEOに好影響

## 9. コピーライティング手法

### ベネフィット訴求

| スペック訴求（NG） | ベネフィット訴求（OK） |
|-------------------|----------------------|
| ビタミンC 1000mg配合 | 毎朝の1粒で内側から透明感をサポート |
| 防水素材使用 | 急な雨でも安心。通勤が快適に |
| 容量500ml | たっぷり使える大容量。約2ヶ月分 |

### 楽天特有のコピーTips

| Tip | 詳細 |
|-----|------|
| 送料無料の訴求 | 楽天ユーザーは送料無料を重視。商品名やキャッチコピーに明記 |
| ポイント訴求 | 「ポイント10倍」など効果的 |
| あす楽対応 | 即日配送対応ならキャッチコピーで訴求 |
| レビュー件数 | 多い場合はキャッチコピーに記載 |
| ランキング | 入賞実績があれば積極的に訴求 |

### 社会的証明

- レビュー活用: 「評価4.5以上」
- ランキング: 「楽天ランキング1位」
- 実績: 「累計10万個突破」
- メディア: 「雑誌〇〇で紹介」
- 専門家: 「管理栄養士監修」

### 限定感・緊急性

- 数量限定: 「残りわずか」「限定100個」
- 期間限定: 「本日23:59まで」
- イベント連動: 「お買い物マラソン限定クーポン」
- 先着特典: 「先着200名様にサンプルプレゼント」

## 10. 商品ページのクリエイティブ設計

### 商品説明画像の推奨フロー

| 順番 | 内容 | 目的 |
|------|------|------|
| 1 | キャッチ画像 | 最大ベネフィットを一言で |
| 2 | 悩み共感 | ターゲットの悩みに共感 |
| 3 | 解決策の提示 | 商品がどう解決するか |

## 12. レビュー獲得戦略

### レビュー数と効果

| レビュー数 | 効果 |
|-----------|------|
| 0件 | 購入ハードルが高い |
| 1〜10件 | 最低限の安心感 |
| 10〜50件 | 信頼性向上、SEO好影響 |
| 50〜100件 | CVR大幅改善 |
| 100件以上 | 強力な社会的証明 |

### 主要施策

1. **フォローメール**: 到着後3〜7日が最適。1〜2回まで
2. **レビュー特典**: 100〜500円クーポン。高評価指定は厳禁
3. **同梱カード**: QRコード付きレビュー投稿ガイド
4. **レビュー返信**: 高・低評価ともに返信
5. **品質徹底**: 丁寧な梱包、迅速な配送

### 目標設定

| フェーズ | 目標 | 施策 |
|----------|------|------|
| 立ち上げ期（〜3ヶ月） | 10件 | フォローメール + 同梱カード + 初回クーポン |
| 成長期（3〜6ヶ月） | 50件 | 上記 + ポイント特典 + SNS活用 |
| 安定期（6ヶ月〜） | 100件以上 | 継続 + 定期キャンペーン |
