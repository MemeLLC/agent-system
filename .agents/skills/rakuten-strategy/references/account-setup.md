## 3. 出店審査の流れと必要書類

### 審査は2段階制

```
Step 1: 出店申込（Web） → Step 2: 書類審査（2週間〜1ヶ月）
→ Step 3: RMSアカウント発行 → Step 4: 店舗構築
→ Step 5: オープン審査 → Step 6: 開店
```

### 出店資格

- **法人**: 法人格を持つ企業
- **個人事業主**: 税務署に開業届を提出済み
- **個人（副業・未届）**: 出店不可

### 必要書類

**法人**: 出店申込書、登記簿謄本（3ヶ月以内）、代表者本人確認書類、営業許可（該当時）、商材写真

**個人事業主**: 出店申込書、住民票（3ヶ月以内）、印鑑証明書（3ヶ月以内）、営業許可（該当時）、商材写真

### 商材別に必要な営業許可

| 商材 | 必要な許可 |
|------|----------|
| 中古品・リユース品 | 古物商許可証 |
| 酒類 | 通信販売酒類小売業免許 |
| 食品 | 食品衛生法に基づく営業許可 |
| 医薬品 | 薬局開設許可 / 医薬品販売業許可 |
| 化粧品 | 化粧品製造販売業許可（自社製造の場合） |
| コンタクトレンズ | 高度管理医療機器等販売業許可 |

### 審査に落ちやすいケース

- 出品禁止商品に該当
- 書類不備・有効期限切れ
- 事業実態が不明確
- 電話確認に応答しない
- 過去のトラブル履歴

## 4. RMSの基本構成

RMS（Rakuten Merchant Server）は楽天市場の全店舗が使用する管理システム。

### 4つの主要機能

| モジュール | 主な機能 |
|-----------|---------|
| **R-Storefront** | 商品登録・編集、店舗トップページ編集、カテゴリ管理、クーポン発行 |
| **R-Backoffice** | 受注管理、在庫管理、配送管理、CSV一括処理 |
| **R-Mail** | メルマガ配信、セグメント配信、フォローメール |
| **R-Karte** | アクセス分析、売上分析、転換率分析、検索キーワード分析 |

### 2024年のRMSアップデート

- 2024年3月: RMS AIアシスタント β版（商品説明文自動生成、画像加工支援）
- R-Karteに時間単位データ提供と新分析軸追加
- SKUプロジェクト対応の機能改善

## 6. 楽天ポイント制度

### 負担区分

| ポイント種別 | 負担元 |
|-------------|--------|
| 通常ポイント（1%） | **店舗負担** |
| SPUポイントアップ分 | **楽天負担** |
| イベントポイント | **楽天負担**（一部例外） |
| 店舗独自ポイントアップ | **店舗負担** |

### SPU（スーパーポイントアッププログラム）

最大17倍のポイント還元。SPUのポイントアップ分は楽天が負担するため、店舗に追加コストなし。

### 店舗独自のポイント施策

- 商品個別ポイント変倍: 2倍〜20倍
- 店舗全体ポイント変倍
- イベント連動ポイント施策

## 7. 出店から開店までのスケジュール

### 全体スケジュール（約1〜2ヶ月）

```
Week 1-2: 出店申込・書類準備
Week 2-5: 書類審査（2週間〜1ヶ月）
Week 5-6: RMSアカウント発行・初期設定（特商法表記・決済・配送）
Week 6-8: 店舗構築（デザイン・商品登録・テスト注文）
Week 8:   オープン審査 → 開店
```

### 開店後に優先すべきアクション

| 優先度 | アクション |
|--------|-----------|
| 高 | SKUプロジェクト対応（商品属性・タグ設定） |
| 高 | 商品画像ガイドライン対応（白背景、テキスト20%以内） |
| 高 | 楽天SEO対策（商品名・キーワード最適化） |
| 中 | フォローメール設定（レビュー獲得） |
| 中 | クーポン・ポイント施策の開始 |
| 低 | メルマガ配信・R-Karte分析体制構築 |

### コスト試算例（がんばれ！プラン、月商100万円）

| 費目 | 金額（税別/月） |
|------|----------------|
| 月額出店料 | 25,000円 |
| システム利用料（約5.5%） | 55,000円 |
| 楽天ペイ利用料（約3.5%） | 35,000円 |
| ポイント原資（1.0%） | 10,000円 |
| アフィリエイト（推定） | 3,900円 |
| 安全性・利便性（0.1%） | 1,000円 |
| **変動費合計** | **約104,900円（売上比約10.5%）** |
