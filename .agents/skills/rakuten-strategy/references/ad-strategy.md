## 1. フルファネル広告設計

### ファネル構造と広告メニュー対応

| ファネル | 広告メニュー | KPI | 目標値 |
|---------|------------|-----|--------|
| 認知 | 純広告, TDA（属性セグメント）, ROOM | インプレッション・リーチ | ターゲットの30%以上 |
| 興味・関心 | TDA（閲覧/購買履歴）, RPP（ビッグKW）, CA | CTR | 0.5〜1.5% |
| 比較・検討 | RPP（ミドル/ロングテールKW）, CA, CPA | ページ訪問数 | 前月比+20% |
| 購入 | RPP（商品名/型番KW）, CA（値引き訴求）, イベント | CVR, ROAS | CVR 3〜10%, ROAS 400%+ |
| リピート | TDA（自店舗購入者）, R-Mail, CA | リピート率, LTV | リピート率20%+ |

### 設計原則

1. **初期はRPPに集中**: 「比較・検討」「購入」段階をカバーし売上基盤を作る
2. **売上安定後に上位ファネル追加**: TDA・純広告で認知拡大
3. **リピート施策は早期から**: R-Mail・CAでリピーター育成を並行
4. **ファネル間の数値を連動管理**: 認知投資→下位ファネルの改善を検証

## 3. RPP広告の最適化手法

### キーワードの3層構造

| 層 | 特徴 | CPC目安 | 例（アパレル） |
|---|------|--------|--------------|
| ビッグKW | 検索ボリューム大、競合多 | 100〜500円 | 「ワンピース」 |
| ミドルKW | 適度なボリューム | 40〜150円 | 「ワンピース 花柄 春」 |
| ロングテールKW | ニッチ、転換率高 | 40〜80円 | 「ワンピース 花柄 ロング丈 30代」 |

### キーワード選定の方法

1. 楽天RMSキーワード候補機能（最大50候補）
2. 流入キーワード分析（RMSアクセス分析）
3. 楽天サジェスト
4. Googleキーワードプランナー
5. 競合分析（上位表示商品のタイトル参考）

### CPC調整の基本原則

- ROASが高い → CPC引き上げ（機会損失防止）
- ROASが低い → CPC引き下げ or 除外設定
- **基準ROAS: 最低400%**

### CPC調整タイミング

| タイミング | アクション |
|-----------|----------|
| 通常期 | 週1回、10〜20%ずつ微調整 |
| イベント前（1週間前） | 主力商品CPC 20〜50%引き上げ |
| イベント期間中 | 転換率監視しながらさらに引き上げ |
| イベント終了後 | 通常水準に戻す |

### 自動最適化（2025年11月以降）

- 「固定CPC」→「上限CPC」に変更
- 楽天AIが上限CPC以内で最適CPCを自動調整
- AIの最適化にはデータ蓄積期間が必要（最低2〜4週間）

### 除外設定

**除外商品**: 転換率低い / 利益率低い / 在庫少ない / 低単価商品

**除外キーワード**: 完全一致（ピンポイント）/ フレーズ一致（広範囲）

## 4. イベント連動の広告戦略

### スーパーSALE（年4回: 3/6/9/12月）

| 時期 | アクション |
|------|----------|
| 2ヶ月前 | 純広告枠予約検討、イベント用商品選定・価格設定 |
| 1ヶ月前 | TDAバナー制作・入稿（審査6営業日） |
| 2週間前 | RPPキーワード・CPC見直し、クーポン準備 |
| 1週間前 | RPP CPC 20〜50%引き上げ、メルマガ事前告知 |
| 期間中 | 全広告最大強化。日次でROAS・在庫監視 |
| 終了後 | 通常水準に戻す。レポート分析 |

**固有施策**: 半額サーチ対応（50%以上割引）、超目玉枠申請、買い回り対策（1,000円以上商品）

### お買い物マラソン（毎月）

- RPP CPC 10〜30%引き上げ
- CA予算増額
- 買い回り対策、メルマガ告知、ポイント倍率設定

### 5と0のつく日（毎月6回）

- RPP CPC 10〜20%引き上げ
- 「5と0のつく日」告知バナー設置
- メルマガ前日〜当日配信

### マラソン × 5と0のつく日 重複日

最もアクセスが集中する「ゴールデンタイム」。RPP CPC・予算を最大化し、クーポン・ポイントを重ねがけ。

### イベント予算配分

| 月のタイプ | 予算目安 |
|-----------|---------|
| スーパーSALE月 | 通常月の1.5〜2.0倍 |
| マラソン月 | 通常月の1.2〜1.5倍 |
| イベントなし月 | 通常予算 |

## 5. 楽天SEOと広告の相乗効果

### 相乗効果メカニズム

```
RPP広告で露出増加 → クリック・売上増加 → 売上実績・レビュー蓄積
→ 自然検索順位上昇 → オーガニック流入増加 → 広告依存度低下
→ 新商品・新KWに予算を再配分
```

### 実践戦略

1. **タイトル最適化**: RPPレポートで成果の良いKWを商品タイトルに反映
2. **売上実績蓄積**: 新商品はRPPで初期売上確保→自然検索順位向上
3. **レビュー獲得**: 広告経由購入者にフォローメール
4. **検索面の独占**: RPP広告枠 + 自然検索上位に同時掲載で露出最大化

## 6. ROAS改善のPDCAサイクル

### 週次チェック項目

| 項目 | 確認内容 | アクション基準 |
|------|---------|-------------|
| 全体ROAS | 400%以上か | 下回れば商品・KW単位で原因特定 |
| 商品別ROAS | 各商品の状況 | 基準値大幅下回りは除外 |
| CTR | クリック率 | 0.5%未満はサムネイル・タイトル改善 |
| CVR | 転換率 | 2%未満は商品ページ改善 |
| CPC消化額 | 予算消化ペース | 月半ばで60%以上消化なら調整 |

### ROAS低い場合の対処

| 原因 | 対処 |
|------|------|
| CTR低い | サムネイル・タイトル改善 |
| CVR低い | 商品ページ改善（画像追加・説明充実・レビュー促進） |
| CPC高すぎ | CPC引き下げ、ロングテールKWシフト |
| 商品不適切 | 転換率/利益率低い商品を除外 |
| KWミスマッチ | 除外KW追加、適切なKWに変更 |

## 7. 新商品ローンチ時の広告戦略

### 4フェーズ

**フェーズ1: 準備（2〜4週間前）**: 商品ページ最適化、事前告知、TDAバナー制作

**フェーズ2: ローンチ（1〜4週間）**: RPP（ロングテールKW集中）、CPA ON、新商品クーポン（10〜20%OFF）、TDA（自店舗来訪者）、R-Mail
- ビッグKWは避ける（売上実績なく不利）
- 目標はROASより売上件数（レビュー蓄積優先）

**フェーズ3: 成長（1〜3ヶ月）**: ミドルKW展開、CPC最適化、レビュー促進、CA追加

**フェーズ4: 安定（3ヶ月〜）**: SEO×広告最適化、ビッグKW挑戦、リピート施策強化

### ローンチ月の予算目安

通常月の1.5〜2倍

## 9. A/Bテスト

### テスト可能な要素

| 対象 | テスト内容 | 期待効果 |
|------|----------|---------|
| サムネイル画像 | バリエーション比較 | CTR改善 |
| 商品タイトル | KW配置・訴求変更 | CTR・検索順位 |
| クーポン値引き率 | 5%/10%/15%/20%比較 | CVR・利益バランス |
| RPP CPC | CPC金額比較 | ROAS最適化 |
| TDAバナー | クリエイティブ比較 | CTR改善 |
| ポイント倍率 | 3倍/5倍/10倍比較 | CVR・利益バランス |

### 実施ルール

- 1回のテストで変更は1要素のみ
- 最低1〜2週間（1週間ごとにA/B切替）
- イベント期間は避ける
- 最低100クリック以上のデータ量で判断
- 結果を記録しナレッジ蓄積

## 10. 広告レポートと主要KPI

### KPI一覧

| KPI | 算出方法 | 目標目安 |
|-----|---------|---------|
| ROAS | 広告経由売上 ÷ 広告費 × 100 | 400%以上 |
| CTR | クリック数 ÷ インプレッション × 100 | 0.5〜2.0% |
| CVR | 注文数 ÷ クリック数 × 100 | 3〜10% |
| CPA | 広告費 ÷ 新規顧客数 | 商品単価の20%以下 |

### 売上分解公式

```
売上 = アクセス数 × 転換率（CVR） × 客単価
```

| 要素 | 改善施策 |
|------|---------|
| アクセス数 | RPP CPC引き上げ・予算増、SEO対策、イベント活用 |
| CVR | 商品ページ改善、クーポン、ポイント、レビュー充実 |
| 客単価 | セット販売、まとめ買い訴求、アップセル |

### 広告種類別の評価指標

| 広告 | 最重要KPI | 補助KPI |
|------|---------|--------|
| RPP | ROAS（720h） | CTR, CVR, CPC |
| CA | ROAS | クーポン獲得率・利用率 |
| TDA | CTR, 来店数 | vimps, CVR |
| CPA | 売上金額 | CPA経由売上比率 |
| 純広告 | インプレッション, CTR | 掲載期間中の売上変動 |
| R-Mail | 開封率, メルマガ経由売上 | CTR, 転換率 |

### レポート分析サイクル

| 頻度 | 分析内容 |
|------|---------|
| 日次 | 異常値チェック（急なROAS低下・予算過消費） |
| 週次 | 商品別・KW別パフォーマンス → CPC調整・除外更新 |
| 月次 | 全広告メニュー横断比較 → 翌月予算・施策見直し |
| 四半期 | ファネル全体効果分析 → 中期戦略修正 |
