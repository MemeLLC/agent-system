## 1. RPP広告（検索連動型広告）

### 概要

RPP（Rakuten Promotion Platform）は楽天市場内の検索連動型広告。検索結果の上位に広告枠として商品を表示する主力広告メニュー。

### 課金・表示仕様

| 項目 | 内容 |
|------|------|
| 課金方式 | CPC（クリック課金） |
| 表示位置 | PC: 検索結果上位5枠 / スマホ・アプリ: 上位7枠 |
| 掲載ロジック | 入札CPC × 品質スコア（売上実績・CTR等） |
| 最低月額予算 | 5,000円 |
| デフォルト配信 | 全商品が対象（除外設定が必要） |

### CPC設定

| 設定種別 | 最低CPC | 上限CPC | 特徴 |
|---------|--------|--------|------|
| キャンペーンCPC | 20円 | 10,000円 | 全商品に適用されるベースCPC（自動最適化移行後10→20円に引き上げ） |
| 商品別CPC | 20円 | 10,000円 | 特定商品に個別設定。キャンペーンCPCより優先（同様に20円〜） |
| キーワード別CPC | 40円 | 10,000円 | 特定キーワードに個別設定。商品CPCより優先 |

**CPC優先順位**: キーワードCPC > 商品CPC > キャンペーンCPC

### ターゲティング設定

| 設定 | 内容 |
|------|------|
| キーワードターゲティング | 検索キーワードに入札（最大10キーワード/商品） |
| 商品別ターゲティング | 商品単位でCPC個別設定 |
| ランク別入札（2024年1月追加） | 会員ランクに応じてCPC自動調整（高ランク: +25%以内 / 低ランク: -80%以内） |
| 除外キーワード | 完全一致・フレーズ一致で不要KWを除外 |
| 除外商品 | 配信対象から外す商品を登録 |

### 自動最適化機能（2025年7月リリース・11月完全移行）

2025年7月14日にAI活用の自動最適化機能がリリースされ、2025年11月1日から全店舗に完全適用。新規キャンペーンは「自動最適化タイプ」のみ作成可能。

- 掲載面・配信日時・ユーザー属性に応じてCPCを自動最適化
- 「固定クリック単価」が「上限クリック単価」に名称変更
- 請求CPCは設定した上限CPC以内で楽天が最適値に自動調整
- **手動配信は2025年10月31日で終了**
- 適用対象: キャンペーンCPCと商品CPCに適用。**キーワードCPCには非対応**（手動調整が必要）
- 最低CPCが10円→20円に引き上げ

### 自動最適化後のROAS悪化問題と対策

2025年8月以降、多くの店舗でROAS悪化が報告されている。

**悪化の原因:**
- AIがCTR（クリック率）を重視する傾向があり、CVRが高くてもCTRが低い掲載面の露出が減少
- クリックされやすい掲載面に配信が集中し、購買意欲の低いユーザーへの表示が増加
- 広告消化額が増加するが売上アップにつながらないケースが多発
- 学習期間中はROASが乱高下する不安定さがある

**対策:**
- ROAS 300%以上の商品に集中投資し、低パフォーマンス商品は除外
- 利益率から逆算した上限CPC設計を徹底
- CTR改善（タイトル・サムネイル最適化）とCVR向上（商品ページ改善・レビュー獲得）の両立が必須

### RPP-EXP（エクスパンション広告）

2024年5月リリース。楽天グループ外メディア（Google検索結果/ショッピングタブ/画像タブ）に配信。

- 2025年から**「ROAS重視キャンペーン」が標準設定**（従来のクリック重視型を廃止）
- RMSの商品情報と連動するため、別途広告原稿の入稿は不要
- 楽天市場外のユーザーにリーチ可能

### ROAS目安

| 指標 | 目安 |
|------|------|
| 平均ROAS | 300%前後（多くの店舗の実態） |
| 推奨最低ROAS | 400%以上（手数料考慮） |
| 優良店舗のROAS | 500〜1,000%以上 |

### 運用ポイント

1. **除外商品を先に設定**: 転換率・利益率の低い商品を除外し、売れ筋に予算集中
2. **キーワード選定**: 楽天KW候補機能（最大50候補）+ 流入KW分析 + Googleキーワードプランナー併用
3. **段階的CPC調整**: キャンペーンCPCでテスト → 成果良い商品に商品CPC → 成果良いKWにキーワードCPC
4. **週次レポート確認**: ROAS基準下回る商品は除外、上回る商品はCPC引き上げ
5. **イベント時強化**: スーパーSALE・マラソン期間はCVR1.5〜7倍のため、CPC・予算を引き上げ

### 向いている目的

**売上直結** / **集客**

## 2. クーポンアドバンス広告

### 概要

楽天市場独自のクーポン付き運用型広告。ユーザーの閲覧・購買履歴に基づきAIが自動マッチングしてクーポンを配信する。

### 課金・費用構造

| 費用種別 | 内容 | 目安 |
|---------|------|------|
| クーポン獲得費用 | 入札CPC × クリック数 | CPC 25〜1,000円（設定範囲） |
| クーポン割引費用 | クーポン利用時の値引き額 | 商品価格の4%〜が推奨 |
| 最低月額予算 | -- | 5,000円 |

- 一般的なCPC目安: 30〜60円程度
- 2024年1月に目安CPC表示機能追加

### 配信面

- 楽天市場トップページ
- ジャンルトップページ
- 楽天市場アプリのプッシュ通知
- 楽天外部の提携サイト
- メール配信

### プラン制（2025年12月15日 大規模アップデート）

従来の自動/手動設定から、3つの新プラン制に変更:

| プラン | 対象商品 | 値引き率 | 特徴 |
|--------|---------|---------|------|
| おまかせプラン | AIが自動選定 | AIが自動最適化 | 商品別・ユーザー別に最適クーポン発行。割引率上限の設定可能 |
| 値引率設定プラン | 楽天が自動選定 | 店舗が設定 | 対象商品はAI任せ、割引率はコントロール |
| こだわり設定プラン | 店舗が手動選定 | 店舗が手動設定 | 対象商品・値引率を細かく設定 |

**新機能:**
- キャンペーン単位での最大値引率設定が可能に（意図しない過度な割引を防止）
- 推奨値引率の事前確認機能（CSV）
- より詳細なオーディエンスセグメント（例: 「30代女性かつ過去30日以内にヘアケア商品購入」）
- 類似ユーザーターゲティング機能の強化

**注意:** 既存キャンペーンの設定が自動移行されるため、対象商品やクーポン内容が想定と異なるケースが報告されている。必ず確認・修正が必要。

- **クーポン有効期限**: 「翌月曜日の14:59」で固定（変更不可）

### ターゲティング

- 楽天市場ログインユーザーのみが対象
- 閲覧・購買履歴に基づく自動ターゲティング
- 商品登録による対象商品の指定
- 類似ユーザーターゲティング（売れ筋商品の購入者と類似行動をとる層）

### 運用ポイント

1. **プラン選択**: まずは「おまかせプラン」で開始し、データを蓄積してから「こだわり設定プラン」に移行
2. **利益率の高い商品を優先登録**: クーポン割引分を含めても利益が出る商品を選定
3. **イベント時の活用**: スーパーSALE・マラソン期間は購買意欲が高く効果的
4. **RPPとの併用**: RPPで検索面カバー、クーポンアドバンスでパーソナライズ配信
5. **転換率の高い商品を登録**: クーポン効果で通常より高いCVRが期待できる
6. **移行後の設定確認**: 2025年12月の自動移行後、必ず設定内容を確認・修正

### 向いている目的

**売上直結** / **集客** / **リピート促進**

## 3. ターゲティングディスプレイ広告（TDA）

### 概要

楽天市場内の広告枠に、セグメント指定でバナー広告を表示する運用型ディスプレイ広告。認知拡大から購買促進まで幅広い目的に対応。

### 課金・配信仕様

| 項目 | 内容 |
|------|------|
| 課金方式 | vimps（ビューアブルインプレッション）課金 |
| 基本単価 | 0.75円/vimp |
| 入札強化 | 0.05円刻みで引き上げ可能 |
| 最低月額予算 | 50,000円 |
| 審査 | 約6営業日（2024年10月よりAI審査導入） |
| 入稿期限 | 配信開始の10営業日前まで |

### セグメント種類

| カテゴリ | セグメント | 詳細 |
|---------|----------|------|
| 会員属性 | 性別 | 男性/女性 |
| 会員属性 | 年齢 | 年代別に指定可能 |
| 会員属性 | 居住地域 | 都道府県単位 |
| 会員属性 | 会員ランク | ダイヤモンド/プラチナ/ゴールド等 |
| 行動履歴 | 閲覧履歴 | カテゴリ単位（特定ジャンル閲覧者） |
| 行動履歴 | 購入履歴 | カテゴリ単位（特定ジャンル購入者） |
| 行動履歴 | 3ヶ月以内3回以上購買 | ヘビーバイヤー |
| イベント | イベント関心度 | イベント参加傾向・買い回り数 |
| 自店舗 | 来訪履歴 | 過去2年間の訪問者 |
| 自店舗 | 購入履歴 | 過去2年間の購入者 |
| ユーザー傾向 | 優良会員 | 高頻度・高単価の購買傾向 |

### クリエイティブ仕様（4サイズ必須）

| バナーサイズ | 用途 |
|------------|------|
| 1,280px × 200px | ワイドバナー |
| 880px × 320px | レクタングルバナー |
| 400px × 800px | スカイスクレイパー |
| 480px × 360px | スクエアバナー |

**制作ルール:**
- テキスト比率: バナー面積の1/3以内
- フォントサイズ: 28〜80px
- 余白: 文字外側に16px以上
- 背景色: 最大2色まで（モデル背景・グラデーション・自然背景はカウント外）

### 配信面

- 楽天市場トップページ
- お気に入り / 閲覧履歴 / 購買履歴ページ
- 商品レビューページ
- スーパーSALE / お買い物マラソンページ

### 2025年の新機能

#### 枠指定型機能（2025年6月19日リリース）
- 指定した掲載箇所にのみ配信できる機能
- 1キャンペーンにつき指定できる枠は1つまで
- 入札単価: 1.8円〜10円/Vimp
- 人気枠に絞った配信で、従来より配信箇所の制御が細かく可能

#### 検索キーワードターゲティング（2025年10月1日リリース）
- 特定キーワードで検索したユーザーに対し、**検索結果最上部にバナー広告を表示**
- 従来のセグメント（人）ベースから、**検索キーワード（意図）ベースのアプローチ**が追加
- 課金方式: Vimp型（広告の50%以上が1秒以上表示された場合のみ課金）
- 入札単価: 3.00円〜10.00円/Vimp
- 楽天指定キーワードまたは任意指定キーワードから選択可能
- バナー入稿後は審査あり、**キャンセル不可・修正不可**

#### TDA-EXP（エクスパンション）
- 2023年11月リリース。Meta社メディア（Facebook/Instagram）への配信に対応
- 楽天グループ外のユーザーへリーチ可能

### 向いている目的

**認知拡大** / **集客** / **リターゲティング** / **リピート促進**

## 4. 楽天CPA広告（効果保証型広告）

### 概要

成果報酬型広告。広告経由で購入された場合のみ費用が発生し、表示・クリックでは費用がかからない。

### 課金仕様

| 項目 | 内容 |
|------|------|
| 課金方式 | CPA（成果報酬型）。売上の20% |
| 成果定義 | 広告クリックから720時間（30日間）以内の全商品購入に課金 |
| ROAS | 固定500% |
| 初期費用・月額固定費 | なし |
| 予算上限設定 | 不可（売れた分だけ課金） |

### 掲載面

- 楽天市場トップページ
- ジャンルページ / 各種特集ページ
- 閲覧履歴に基づくレコメンド面
- 楽天グループ横断面（楽天トラベル、楽天ブックス等）
- 配信は楽天アルゴリズムが自動最適化（店舗側で指定不可）

### 向いている商材

- **低〜中単価商品**: 高単価は20%報酬が大きくなるため注意
- **転換率の高い商品**: 30日間の全購入が対象のため「ついで買い」ジャンルでは意図しない課金リスク
- **新規出店店舗**: 初期費用ゼロで売上実績を作る使い方が有効
- **利益率の高い商品**: 20%差し引いても利益確保できる商品

### 注意点

- 広告クリック後720時間以内の「全商品」が課金対象（クリック商品以外も含む）
- 予算上限不可で想定以上の広告費リスク
- 配信面・表示タイミングの制御不可
- 除外商品の登録上限: 50,000件
- **利益率20%以下の商品は必ず除外設定すべき**（広告費が20%のため利益が出ない）

### 向いている目的

**売上直結**（特に広告初心者・新規出店）

## 5. 楽天市場広告（純広告）

### 概要

楽天市場内の指定枠にバナー広告を掲載する予約型広告。掲載枠購入で確実に掲載され、大量インプレッションを獲得できる。

### 枠の種類と費用

| 枠の種類 | 掲載場所 | 費用目安 |
|---------|---------|---------|
| トップページ枠 | 楽天市場トップページ | 80,000〜1,200,000円 |
| ジャンルページ枠 | 各ジャンルのトップページ | 40,000〜200,000円 |
| 特集ページ枠 | 季節・テーマ別特集 | 40,000〜300,000円 |
| イベント枠 | マラソン・スーパーSALE等 | 50,000〜数百万円 |

- スーパーSALE期間の掲載費はマラソン期間の約2倍

### 特別枠

**大口予算枠:**
- スーパーSALE・マラソン等の大型イベント期間にトップ・特設ページの目立つ位置に掲載
- 参加条件: 一定期間内に200万円以上の広告枠購入
- 最も露出の高い枠が利用可能

**超目玉枠:**
- 大型イベントの協賛商品枠。**広告掲載費は無料**
- 条件: 通常価格から50%以上の割引を実施
- 楽天イベント盛り上げ目的の枠

### シーズナル枠

| 時期 | 主な特集 |
|------|---------|
| 1月 | 福袋・初売り |
| 2月 | バレンタイン |
| 3月 | ホワイトデー・新生活準備 |
| 4〜5月 | 母の日・GW |
| 6月 | 父の日・梅雨対策 |
| 7〜8月 | 夏物・お中元 |
| 9月 | 敬老の日 |
| 10月 | ハロウィン |
| 11月 | ブラックフライデー |
| 12月 | クリスマス・お歳暮・年末 |

### 予約・掲載

- 掲載期間は枠ごとに決定（1週間〜1ヶ月が一般的）
- RMS広告メニューまたはECCコンサルタント経由で申込
- 人気枠は2〜3ヶ月前の予約推奨

### 向いている目的

**認知拡大** / **ブランディング** / **イベント時の大量集客**

## 6. その他の広告・販促手段

### 6-1. 楽天アフィリエイト

| 項目 | 内容 |
|------|------|
| 課金方式 | 成果報酬型 |
| 報酬率 | 通常2〜4%（カテゴリ別） |
| 店舗負担 | 基本料率 + 追加報酬設定可能 |
| Cookie有効期間 | 24時間以内のクリック + 89日間の再訪問 |
| 特徴 | 全店舗自動参加。外部からの集客が可能 |

### 6-2. 楽天ROOM

| 項目 | 内容 |
|------|------|
| 課金方式 | 成果報酬型（アフィリエイト準拠） |
| 基本報酬率 | 購入金額の2〜4% |
| ROOMランクボーナス | Bランク+1% / Aランク+2% / Sランク+3% |
| 店舗追加負担 | なし |
| 活用法 | 公式アカウント開設、インフルエンサー活用、コレクション機能 |

### 6-3. メルマガ（R-Mail）

| 項目 | 内容 |
|------|------|
| 費用 | 1通1円（税別）/ 週1回無料配信あり |
| 種類 | テキスト / HTML / レスポンシブメール |
| 効果測定 | 開封率・送客数・CVR・メルマガ経由売上・リンク別クリック数 |
| 特徴 | 楽天市場全体の売上の20%以上がメルマガ経由とも言われる |

### 6-4. 楽天ランキング対策

ランキングページ掲載による大量アクセス獲得施策（広告ではなく販売実績ベース）。

**ランキング種類:** リアルタイム / デイリー / 週間 / ジャンル別

**上位獲得の施策:**
- イベント期間に合わせて価格を下げ、短期間に売上集中
- RPP・クーポンアドバンスで集客強化
- クーポン配布・ポイント倍率アップでCVR引き上げ
- SNS・メルマガで特定時間帯に注文を集中

### 6-5. イベント活用

| イベント | 頻度 | 特徴 |
|---------|------|------|
| スーパーSALE | 年4回（3/6/9/12月） | 最大級。ポイント最大47倍、半額サーチ、買い回り |
| お買い物マラソン | ほぼ毎月 | 買い回りでポイントアップ |
| 5と0のつく日 | 毎月6回 | 楽天カードでポイント4倍 |
| 楽天イーグルス感謝祭 | 年1回（10〜11月） | プロ野球シーズン終了連動 |
| ブラックフライデー | 年1回（11月） | 海外由来の大型セール |
